Threaded::count

(PECL pthreads >= 2.0.0)

Threaded::count — Manipulation

説明

public integer Threaded::count ( void )

Returns the number of properties for this object

パラメータ

この関数にはパラメータはありません。

返り値

例

例1 Counting the properties of an object

<?php
$safe = new Threaded();

while (count($safe) < 10) {
    $safe[] = count($safe);
}

var_dump(count($safe));
?>

上の例の出力は以下となります。

int(10)
