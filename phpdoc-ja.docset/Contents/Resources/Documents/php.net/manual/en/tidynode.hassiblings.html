tidyNode::hasSiblings

(PHP 5 >= 5.0.1)

tidyNode::hasSiblings — このノードが兄弟ノードを持つかどうかを調べる

説明

bool tidyNode::hasSiblings ( void )

ノードが兄弟ノードを持つかどうかを調べます。

返り値

兄弟ノードを持つ場合に TRUE、それ以外の場合に FALSE を返します。

例

例1 tidyNode::hasSiblings() の例

<?php

$html = <<< HTML
<html><head>
<?php echo '<title>title</title>'; ?>
<# 
  /* JSTE code */
  alert('Hello World'); 
#>
</head>
<body>

<?php
  // PHP code
  echo 'hello world!';
?>

<%
  /* ASP code */
  response.write("Hello World!")
%>

<!-- Comments -->
Hello World
</body></html>
Outside HTML
HTML;


$tidy = tidy_parse_string($html);
$num = 0;

// the html tag
var_dump($tidy->html()->hasSiblings());

// the head tag
var_dump($tidy->html()->child[0]->hasSiblings());

?>

上の例の出力は以下となります。

bool(false)
bool(true)

注意

注意:

この関数は、PHP 4/Tidy 1 では tidy_node::has_siblings() という名前でした。
