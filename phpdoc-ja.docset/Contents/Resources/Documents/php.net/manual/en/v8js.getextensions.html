V8Js::getExtensions

(PECL v8js >= 0.1.0)

V8Js::getExtensions — 登録された拡張の配列を返す

説明

public static array V8Js::getExtensions ( void )

この関数は、V8Js::registerExtension() で登録された Javascript 拡張の配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

登録された拡張の配列を返します。何も登録されていない場合は空の配列を返します。
