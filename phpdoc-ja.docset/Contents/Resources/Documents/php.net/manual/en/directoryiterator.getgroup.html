DirectoryIterator::getGroup

(PHP 5)

DirectoryIterator::getGroup — 現在の DirectoryIterator アイテムのグループを取得する

説明

public int DirectoryIterator::getGroup ( void )

ファイルのグループ ID を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の DirectoryIterator アイテムのグループ ID を数値形式で返します。

例

例1 DirectoryIterator::getGroup() の例

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
$groupid  = $iterator->getGroup();
echo 'Directory belongs to group id ' . $groupid . "\n";
print_r(posix_getgrgid($groupid));
?>

上の例の出力は、 たとえば以下のようになります。

Directory belongs to group id 42
Array
(
    [name]    => toons
    [passwd]  => x
    [members] => Array
        (
            [0] => tom
            [1] => jerry
        )
    [gid]     => 42
)
