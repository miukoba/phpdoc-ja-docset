px_numrecords

(PECL paradox >= 1.0.0)

px_numrecords — データベース内のレコードの数を返す

説明

int px_numrecords ( resource $pxdoc )

データベースファイル内のレコードの数を取得します。

パラメータ

pxdoc

px_new() が返す paradox データベースのリソース ID。

返り値

データベースファイル内のレコードの数を返します。この関数の返す値は、 px_get_info() が返す配列における要素 numrecords の値と等しくなります。
