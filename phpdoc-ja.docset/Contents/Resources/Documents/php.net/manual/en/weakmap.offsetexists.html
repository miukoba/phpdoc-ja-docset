WeakMap::offsetExists

(PECL weakref >= 0.2.0)

WeakMap::offsetExists — オブジェクトがマップ内にあるかどうかを調べる

説明

public bool WeakMap::offsetExists ( object $object )

渡されたオブジェクトがマップ内で参照されているかどうかを調べます。

パラメータ

object

調べたいオブジェクト。

返り値

オブジェクトがマップ内に含まれる場合に TRUE、それ以外の場合に FALSE を返します。
