ReflectionMethod::isAbstract

(PHP 5)

ReflectionMethod::isAbstract — 抽象メソッドであるかどうかを調べる

説明

public bool ReflectionMethod::isAbstract ( void )

抽象メソッドであるかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

抽象メソッドである場合に TRUE、それ以外の場合に FALSE を返します。
