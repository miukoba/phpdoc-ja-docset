SplFixedArray::offsetGet

(PHP 5 >= 5.3.0)

SplFixedArray::offsetGet — 指定した添字の値を返す

説明

public mixed SplFixedArray::offsetGet ( int $index )

index で指定した添字の値を返します。

パラメータ

index

値を取得したい添字。

返り値

index で指定した添字の値を返します。

エラー / 例外

index が配列で定義されているサイズの範囲外の場合や index を数値として解釈できない場合に RuntimeException をスローします。
