RarEntry::getCrc

(PECL rar >= 0.1)

RarEntry::getCrc — エントリの CRC を取得する

説明

public string RarEntry::getCrc ( void )

アーカイブエントリの CRC を、十六進文字列表現で返します。

パラメータ

この関数にはパラメータはありません。

返り値

アーカイブエントリの CRC、あるいはエラー時に FALSE を返します。

変更履歴

バージョン 説明
2.0.0 複数ボリュームのアーカイブでも正しい値を返すようになりました。
