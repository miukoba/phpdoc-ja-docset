HttpRequest::getOptions

(PECL pecl_http >= 0.10.0)

HttpRequest::getOptions — オプションを取得する

説明

public array HttpRequest::getOptions ( void )

現在設定されているオプションを取得します。

返り値

現在設定されているオプションを含む連想配列を返します。
