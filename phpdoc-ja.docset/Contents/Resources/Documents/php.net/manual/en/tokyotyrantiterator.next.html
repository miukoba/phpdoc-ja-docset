TokyoTyrantIterator::next

(PECL tokyo_tyrant >= 0.2.0)

TokyoTyrantIterator::next — 次のキーに移動する

説明

public mixed TokyoTyrantIterator::next ( void )

反復処理中に次のキーに移動し、その値を返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に次の値、失敗した場合に false を返します。
