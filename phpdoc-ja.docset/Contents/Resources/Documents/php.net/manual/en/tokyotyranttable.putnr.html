TokyoTyrantTable::putNr

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrantTable::putNr — 値を格納する

説明

public void TokyoTyrantTable::putNr ( mixed $keys [, string $value ] )

このメソッドは、テーブルデータベースではサポートしていません。 このメソッドを TokyoTyrantTable でコールするとエラーと見なされ、 TokyoTyrantException をスローします。

パラメータ

keys

文字列のキー、あるいは key-value ペアの配列。

value

文字列の値。

返り値

このメソッドは、テーブルデータベースではサポートしていません。 このメソッドを TokyoTyrantTable でコールするとエラーと見なされ、 TokyoTyrantException をスローします。
