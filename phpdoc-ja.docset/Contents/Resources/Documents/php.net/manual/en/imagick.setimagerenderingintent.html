Imagick::setImageRenderingIntent

(PECL imagick 2.0.0)

Imagick::setImageRenderingIntent — 画像のレンダリング方向を設定する

説明

bool Imagick::setImageRenderingIntent ( int $rendering_intent )

画像のレンダリング方向を設定します。

パラメータ

rendering_intent

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
