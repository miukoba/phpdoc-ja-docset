LimitIterator::rewind

(PHP 5 >= 5.1.0)

LimitIterator::rewind — イテレータを指定した開始オフセットに巻き戻す

説明

public void LimitIterator::rewind ( void )

イテレータを、LimitIterator::__construct() で指定した開始オフセットに巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
