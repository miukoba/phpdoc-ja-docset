Gmagick::modulateimage

(PECL gmagick >= Unknown)

Gmagick::modulateimage — 明度、彩度、色相を制御する

説明

public void Gmagick::modulateimage ( float $brightness , float $saturation , float $hue )

画像の明度、彩度、色相を制御します。 色相は、現在位置からの絶対回転角のパーセンテージで表します。 たとえば 50 の場合は反時計回りに 90 度の回転を表します。 また 150 の場合は時計回りの 90 度の回転、0 および 200 はともに 180 度の回転を表します。

パラメータ

brightness

画像の明度 (-100 から +100)。

saturation

画像の彩度 (-100 から +100)。

hue

画像の色相 (-100 から +100)。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
