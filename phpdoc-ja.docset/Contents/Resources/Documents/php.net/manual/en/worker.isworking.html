Worker::isWorking

(PECL pthreads >= 2.0.0)

Worker::isWorking — 状態を検出する

説明

public boolean Worker::isWorking ( void )

ワーカーが Stackable を実行中かどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

状態を表す boolean 値を返します。

例

例1 ワーカーの状態を検出する

<?php
$my = new Worker();
$my->start();
/* ... */
if ($my->isWorking()) {
    /* ... ワーカーは、他のオブジェクトを実行するのに忙しい状態 */
}
?>
