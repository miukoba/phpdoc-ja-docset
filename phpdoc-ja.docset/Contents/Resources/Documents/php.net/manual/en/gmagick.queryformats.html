Gmagick::queryformats

(PECL gmagick >= Unknown)

Gmagick::queryformats — Gmagick がサポートするフォーマットを返す

説明

public array Gmagick::queryformats ([ string $pattern = "*" ] )

Gmagick がサポートするフォーマットを返します。

パラメータ

pattern

パターンを含む文字列へのポインタを指定します。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
