fann_set_quickprop_mu

(PECL fann >= 1.0.0)

fann_set_quickprop_mu — Sets the quickprop mu factor

説明

bool fann_set_quickprop_mu ( resource $ann , float $quickprop_mu )

Sets the quickprop mu factor.

パラメータ

ann

ニューラルネットワークリソース。

quickprop_mu

The mu factor.

返り値

成功した場合に TRUE、それ以外の場合に FALSE を返します。
