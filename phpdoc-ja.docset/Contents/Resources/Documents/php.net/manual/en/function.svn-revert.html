svn_revert

(PECL svn >= 0.3.0)

svn_revert — 作業コピーへの変更を取り消す

説明

bool svn_revert ( string $path [, bool $recursive = false ] )

作業コピー内のパスでのローカルの変更内容を取り消します。

パラメータ

path

作業コピーへのパス。

recursive

再帰的に処理するかどうかを表すオプションのパラメータ。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
