PDF_fit_pdi_page

(PECL pdflib >= 2.0.0)

PDF_fit_pdi_page — インポートした PDF ページを配置する

説明

bool PDF_fit_pdi_page ( resource $pdfdoc , int $page , float $x , float $y , string $optlist )

さまざまなオプションに基づいて、インポートした PDF ページを配置します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
