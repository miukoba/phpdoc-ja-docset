GmagickDraw::getfont

(PECL gmagick >= Unknown)

GmagickDraw::getfont — フォントを返す

説明

public string GmagickDraw::getfont ( void )

テキストのアノテーションで使うフォントを表す文字列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に文字列を返します。フォントが設定されていない場合は false を返します。
