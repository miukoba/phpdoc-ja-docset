OAuth::setNonce

(PECL OAuth >= 0.99.1)

OAuth::setNonce — 続くリクエスト用のノンスを設定する

説明

public mixed OAuth::setNonce ( string $nonce )

それ以降のすべてのリクエスト用のノンスを設定します。

パラメータ

nonce

oauth_nonce の値。

返り値

成功した場合に TRUE を返します。 nonce が無効な場合に FALSE を返します。

変更履歴

バージョン 説明
1.0.0 以前は、失敗したときに FALSE ではなく NULL を返していました。
