DOMElement::hasAttribute

(PHP 5)

DOMElement::hasAttribute — 属性が存在するかどうかを調べる

説明

public bool DOMElement::hasAttribute ( string $name )

要素のメンバとして name という名前の属性が 存在するかどうかを示します。

パラメータ

name

要素名。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
