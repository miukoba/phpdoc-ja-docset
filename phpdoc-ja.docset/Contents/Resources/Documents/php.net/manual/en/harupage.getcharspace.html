HaruPage::getCharSpace

(PECL haru >= 0.0.1)

HaruPage::getCharSpace — 現在の文字間隔を取得する

説明

float HaruPage::getCharSpace ( void )

現在の文字間隔を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の文字間隔を返します。

エラー / 例外

エラー時に HaruException をスローします。
