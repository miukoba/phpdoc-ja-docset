swf_setfont

(PHP 4)

swf_setfont — 現在のフォントを変更する

説明

void swf_setfont ( int $fontid )

swf_setfont() は、現在のフォントをパラメータ fontid で指定した値に設定します。

パラメータ

fontid

フォント ID。

返り値

値を返しません。
