Gmagick::setimageunits

(PECL gmagick >= Unknown)

Gmagick::setimageunits — 画像の解像度の単位を設定する

説明

public Gmagick Gmagick::setimageunits ( int $resolution )

画像の解像度の単位を設定します。

パラメータ

resolution

画像の解像度の単位。Undefinedresolution、PixelsPerInchResolution あるいは PixelsPerCentimeterResolution。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
