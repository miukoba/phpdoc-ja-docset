streamWrapper::__destruct

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::__destruct — 既存のストリームラッパーを破棄する

説明

streamWrapper::__destruct ( void )

ストリームラッパーを閉じるときに、 streamWrapper::stream_flush() の直前に呼び出されます。

パラメータ

この関数にはパラメータはありません。
