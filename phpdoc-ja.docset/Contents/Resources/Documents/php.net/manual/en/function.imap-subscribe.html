imap_subscribe

(PHP 4, PHP 5)

imap_subscribe — メールボックスを購読する

説明

bool imap_subscribe ( resource $imap_stream , string $mailbox )

新規メールボックスを購読します。

パラメータ

imap_stream

imap_open() が返す IMAP ストリーム。

mailbox

メールボックス名。詳細は imap_open() を参照ください。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
