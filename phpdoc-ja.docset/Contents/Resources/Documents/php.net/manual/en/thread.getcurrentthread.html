Thread::getCurrentThread

(PECL pthreads >= 2.0.0)

Thread::getCurrentThread — Identification

説明

public static Thread Thread::getCurrentThread ( void )

Return a reference to the currently executing Thread

パラメータ

この関数にはパラメータはありません。

返り値

An object representing the currently executing Thread

例

例1 Return the currently executing Thread

<?php
class My extends Thread {
    public function run() {
        var_dump(Thread::getCurrentThread());
    }
}
$my = new My();
$my->start();
?>

上の例の出力は以下となります。

object(My)#2 (0) {
}
