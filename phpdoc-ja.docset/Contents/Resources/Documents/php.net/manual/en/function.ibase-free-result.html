ibase_free_result

(PHP 5)

ibase_free_result — 結果セットを解放する

説明

bool ibase_free_result ( resource $result_identifier )

結果セットを解放します。

パラメータ

result_identifier

ibase_query() あるいは ibase_execute() で作成した結果セット。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
