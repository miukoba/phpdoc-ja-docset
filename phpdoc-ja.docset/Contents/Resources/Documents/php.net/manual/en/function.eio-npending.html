eio_npending

(PECL eio >= 0.0.1dev)

eio_npending — 終了したけれども処理されなかったリクエストの数を返す

説明

int eio_npending ( void )

eio_npending() は、終了したけれども処理されなかったリクエストの数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

eio_npending() は、終了したけれども処理されなかったリクエストの数を返します。
