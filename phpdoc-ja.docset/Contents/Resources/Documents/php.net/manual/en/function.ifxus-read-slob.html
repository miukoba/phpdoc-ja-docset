ifxus_read_slob

(PHP 4, PHP <=5.2.0)

ifxus_read_slob — SLOB オブジェクトから n バイト読みこむ

説明

string ifxus_read_slob ( int $bid , int $nbytes )

SLOB オブジェクトから nbytes バイトを読みこみます。

パラメータ

bid

既存の SLOB の ID。

nbytes

読み込むバイト数。

返り値

SLOB の内容を表す文字列、あるいはエラー時に FALSE を返します。
