The MongoWriteConcernException class

(PECL mongo >= 1.5.0)

導入

MongoWriteConcernException is thrown when a write fails. See 書き込み確認 for how to set failure thresholds.

Prior to MongoDB 2.6.0 » GLE would determin when a write failed.
