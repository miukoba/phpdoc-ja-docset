radius_close

(PECL radius >= 1.1.0)

radius_close — すべてのリソースを開放する

説明

bool radius_close ( resource $radius_handle )

各リクエストの終了時に PHP がすべてのリソースを開放するので、 この関数をコールする必要はありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
