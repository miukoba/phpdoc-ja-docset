WeakMap::current

(PECL weakref >= 0.2.0)

WeakMap::current — 反復処理中の現在の値を返す

説明

public mixed WeakMap::current ( void )

マップ内の反復処理中に、現在の値を返します。

パラメータ

この関数にはパラメータはありません。

返り値

反復処理中の現在の値を返します。
