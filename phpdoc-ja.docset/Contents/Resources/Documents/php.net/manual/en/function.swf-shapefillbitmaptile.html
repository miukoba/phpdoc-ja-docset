swf_shapefillbitmaptile — 現在の塗りつぶしモードをタイル状のビットマップに設定する

説明

void swf_shapefillbitmaptile ( int $bitmapid )

タイル状タイルでの塗りつぶしを設定します。空いた部分は、 ビットマップで埋められます。

パラメータ

bitmapid

ビットマップ ID。

返り値

値を返しません。
