WeakMap::offsetSet

(PECL weakref >= 0.2.0)

WeakMap::offsetSet — 新しいキー/値のペアでマップを更新する

説明

public void WeakMap::offsetSet ( object $object , mixed $value )

新しいキー/値のペアでマップを更新します。そのキーが既にマップに存在する場合は、 古い値を新しい値で置き換えます。

パラメータ

object

キー/値のペアでキーになるオブジェクト。

value

キー/値のペアで値として扱う任意のデータ。

返り値

値を返しません。
