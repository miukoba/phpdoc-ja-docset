V8Js::getPendingException

(PECL v8js >= 0.1.0)

V8Js::getPendingException — キャッチされなかった Javascript 例外の中で未処理のものを返す

説明

public V8JsException V8Js::getPendingException ( void )

V8Js::executeString() をコールしたときに発生した Javascript の例外の中で未処理のものを V8JsException で返します。

パラメータ

この関数にはパラメータはありません。

返り値

V8JsException あるいは NULL を返します。
