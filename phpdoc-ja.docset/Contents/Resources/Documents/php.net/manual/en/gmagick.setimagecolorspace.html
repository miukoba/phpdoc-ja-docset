Gmagick::setimagecolorspace

(PECL gmagick >= Unknown)

Gmagick::setimagecolorspace — 画像の色空間を設定する

説明

public Gmagick Gmagick::setimagecolorspace ( int $colorspace )

画像の色空間を設定します。

パラメータ

colorspace

画像の色空間。UndefinedColorspace、RGBColorspace、GRAYColorspace、TransparentColorspace、 OHTAColorspace、XYZColorspace、YCbCrColorspace、YCCColorspace、YIQColorspace、 YPbPrColorspace、YPbPrColorspace、YUVColorspace、CMYKColorspace、sRGBColorspace、HSLColorspace あるいは HWBColorspace。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
