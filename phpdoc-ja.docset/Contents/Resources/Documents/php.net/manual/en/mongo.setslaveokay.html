Mongo::setSlaveOkay

(PECL mongo >=1.1.0)

Mongo::setSlaveOkay — この接続の slaveOkay の設定を変更する

説明

public bool Mongo::setSlaveOkay ([ bool $ok = true ] )

読み込みのセカンダリへの分散についての情報は、このマニュアルの 問い合わせの節 を参照ください。

パラメータ

ok

可能な限りの問い合わせをレプリカセットのセカンダリメンバに送信したいときに、この MongoClient インスタンスを使います。

返り値

このインスタンスの、変更前の slaveOkay の値を返します。

変更履歴

バージョン 説明
1.2.11 E_DEPRECATED を発行するようになりました。
