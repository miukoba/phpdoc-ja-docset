ifx_free_blob

(PHP 4, PHP <=5.2.0)

ifx_free_blob — BLOB オブジェクトを削除する

説明

bool ifx_free_blob ( int $bid )

指定された BLOB オブジェクト ID の BLOB オブジェクトを削除します。

パラメータ

bid

BLOB オブジェクト ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
