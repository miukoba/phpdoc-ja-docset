nsapi_response_headers

(PHP 4 >= 4.3.3, PHP 5)

nsapi_response_headers — すべての HTTP レスポンスヘッダを取得する

説明

array nsapi_response_headers ( void )

Gets all the NSAPI response headers.

返り値

すべての NSAPI レスポンスヘッダの連想配列を返します。
