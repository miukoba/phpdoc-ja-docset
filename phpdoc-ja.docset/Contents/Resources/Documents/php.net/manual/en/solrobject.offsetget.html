SolrObject::offsetGet

(PECL solr >= 0.9.2)

SolrObject::offsetGet — プロパティを取得するために使用される

説明

public mixed SolrObject::offsetGet ( string $property_name )

プロパティの値を取得するために使用されます。これは、オブジェクトが配列として扱われる際に使用されます。

パラメータ

property_name

プロパティ名

返り値

プロパティの値を返します。
