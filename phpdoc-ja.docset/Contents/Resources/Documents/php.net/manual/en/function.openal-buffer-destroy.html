openal_buffer_destroy

(PECL openal >= 0.1.0)

openal_buffer_destroy — OpenAL バッファを削除する

説明

bool openal_buffer_destroy ( resource $buffer )

パラメータ

buffer

Open AL (バッファ) リソース (事前に openal_buffer_create() で作成したもの)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
