ZipArchive::unchangeName

(PHP 5 >= 5.2.0, PECL zip >= 1.5.0)

ZipArchive::unchangeName — 指定した名前のエントリに対するすべての変更を取り消す

説明

bool ZipArchive::unchangeName ( string $name )

エントリに対するすべての変更を取り消します。

パラメータ

name

エントリの名前。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
