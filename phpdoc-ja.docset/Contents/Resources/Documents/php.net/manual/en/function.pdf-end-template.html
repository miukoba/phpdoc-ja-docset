PDF_end_template

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_end_template — テンプレートを終了する

説明

bool PDF_end_template ( resource $p )

テンプレートの定義を終了します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
