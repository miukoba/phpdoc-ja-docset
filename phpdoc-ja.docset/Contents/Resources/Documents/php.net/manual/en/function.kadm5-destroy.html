kadm5_destroy

(PECL kadm5 >= 0.2.3)

kadm5_destroy — 管理サーバーへの接続を閉じ、関連するすべてのリソースを開放する

説明

bool kadm5_destroy ( resource $handle )

管理サーバーへの接続を閉じ、関連するすべてのリソースを開放します。

パラメータ

handle

KADM5 ハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
