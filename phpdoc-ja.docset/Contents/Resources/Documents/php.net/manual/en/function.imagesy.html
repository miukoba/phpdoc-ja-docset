imagesy

(PHP 4, PHP 5)

imagesy — 画像の高さを取得する

説明

int imagesy ( resource $image )

指定した画像リソース image の高さを返します。

パラメータ

image

imagecreatetruecolor() のような画像作成関数が返す画像リソース。

返り値

image の高さ、あるいはエラー時に FALSE を返します。

例

例1 imagesy() の使用法

<?php

// 300*200 の画像を作成します
$img = imagecreatetruecolor(300, 200);

echo imagesy($img); // 200

?>
