JsonSerializable インターフェイス

(PHP 5 >= 5.4.0)

導入

JsonSerializable を実装したオブジェクトは、 json_encode() を呼んで処理されたときの自身の JSON 表現をカスタマイズできます。

インターフェイス概要

JsonSerializable {
/* メソッド */
abstract public mixed jsonSerialize ( void )
}
