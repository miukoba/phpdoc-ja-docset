Gender\Gender::similarNames

(PECL gender >= 0.9.0)

Gender\Gender::similarNames — 似た名前を取得する

説明

public array Gender\Gender::similarNames ( string $name [, integer $country ] )

名前と国を指定して、似た名前を取得します。

パラメータ

name

調べる名前。

country

Gender クラスの定数で指定する国 ID。省略した場合は ANY_COUNTRY を使います。

返り値

似ている名前の配列を返します。
