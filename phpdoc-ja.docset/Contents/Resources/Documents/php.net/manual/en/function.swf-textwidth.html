swf_textwidth

(PHP 4)

swf_textwidth — 文字列の幅を得る

説明

float swf_textwidth ( string $str )

現在のフォントおよびフォントサイズを用いて、 文字列の幅をピクセル値で返します。

パラメータ

str

文字列。

返り値

文字列の幅を表す float 値を返します。
