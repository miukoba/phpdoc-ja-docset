TokyoTyrant::putCat

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrant::putCat — レコードを連結する

説明

public TokyoTyrant TokyoTyrant::putCat ( mixed $keys [, string $value ] )

ある値を既存のキーに追記します。keys が配列の場合は複数の値となります。keys が文字列の場合は二番目のパラメータは必須です。レコードが存在しない場合は新しいレコードを作成します。

パラメータ

keys

文字列のキー、あるいは key-value ペアの配列。

value

文字列のキーを使用した場合の値。

返り値

現在のオブジェクトへの参照を返します。失敗した場合は TokyoTyrantException をスローします。

例

例1 TokyoTyrant::putCat() の例

<?php
/* データベースのデフォルトポートに接続します */
$tt = new TokyoTyrant("localhost");

/* 新しいキーを作成します */
$tt->put("key", "value");

/* 単一の key-value ペアを連結します */
$tt->putCat("key", " has more data");

/* キーを表示します */
echo $tt->get("key");
?>

上の例の出力は以下となります。

value has more data
