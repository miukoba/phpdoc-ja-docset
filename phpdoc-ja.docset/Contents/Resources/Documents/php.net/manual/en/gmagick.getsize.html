Gmagick::getsize

(PECL gmagick >= Unknown)

Gmagick::getsize — Gmagick オブジェクトのサイズを取得する

説明

public array Gmagick::getsize ( void )

Gmagick オブジェクトのサイズを返します。 "columns" および "rows" というキーを持つ連想配列となります。

パラメータ

この関数にはパラメータはありません。

返り値

Gmagick オブジェクトのサイズを返します。 "columns" および "rows" というキーを持つ連想配列となります。

エラー / 例外

エラー時に GmagickException をスローします。
