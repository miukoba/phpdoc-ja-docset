swf_fontsize

(PHP 4)

swf_fontsize — フォントの大きさを変更する

説明

void swf_fontsize ( float $size )

フォントの大きさを、 パラメータ size で指定した値に変更します。

パラメータ

size

フォントの大きさを表す整数値。

返り値

値を返しません。
