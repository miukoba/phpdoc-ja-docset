pcntl_get_last_error

(PHP 5 >= 5.3.4)

pcntl_get_last_error — 直近の pcntl 関数が失敗したときのエラー番号を取得する

説明

int pcntl_get_last_error ( void )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

エラーコードを返します。
