pi

(PHP 4, PHP 5)

pi — 円周率の値を得る

説明

float pi ( void )

円周率の近似値を返します。返される float 値の小数点以下の 桁数は、php.ini の precision ディレクティブに依存します。デフォルトは 14 です。 また、定数 M_PI を使用することで pi() とまったく同じ結果を取得することも可能です。

返り値

円周率の値を不動小数点数で返します。

例

例1 pi() の例

<?php
echo pi(); // 3.1415926535898
echo M_PI; // 3.1415926535898
?>
