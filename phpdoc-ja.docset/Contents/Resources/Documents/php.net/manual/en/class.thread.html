Thread クラス

(PECL pthreads >= 2.0.0)

導入

このオブジェクトの start メソッドが呼ばれると、run メソッドのコードが個別のスレッドで非同期で実行されます。

run メソッドの実行後は Thread はすぐに終了し、作成元のスレッドに適切な時期に join します。

警告

Thread をいつ join させるのかをエンジンに決めさせていると、予期せぬ振る舞いを引き起こすことがあります。 可能な限り、プログラマーが明示的に指定するようにしましょう。
