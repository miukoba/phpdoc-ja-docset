socket_import_stream

(PHP 5 >= 5.4.0)

socket_import_stream — ストリームをインポートする

説明

resource socket_import_stream ( resource $stream )

ストリームをインポートし、ソケットをソケットリソースにカプセル化します。

パラメータ

stream

インポートするストリームリソース。

返り値

FALSE を返します。失敗した場合は NULL を返します。

例

例1 socket_import_stream() の例

<?php
$stream = stream_socket_server("udp://0.0.0.0:58380", $errno, $errstr, STREAM_SERVER_BIND); 
$sock   = socket_import_stream($stream);
?>
