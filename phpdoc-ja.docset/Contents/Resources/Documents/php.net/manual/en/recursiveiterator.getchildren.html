RecursiveIterator::getChildren

(PHP 5 >= 5.1.0)

RecursiveIterator::getChildren — 現在のエントリのイテレータを返す

説明

public RecursiveIterator RecursiveIterator::getChildren ( void )

現在のエントリのイテレータを返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のエントリのイテレータを返します。
