SolrIllegalArgumentException::getInternalInfo

(PECL solr >= 0.9.2)

SolrIllegalArgumentException::getInternalInfo — 例外がスローされた際の内部情報を返す

説明

public array SolrIllegalArgumentException::getInternalInfo ( void )

例外がスローされた際の内部情報を返します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

エラーがスローされた際の内部情報を含む配列を返します。エクステンション開発者によるデバッグのためだけに使われます。
