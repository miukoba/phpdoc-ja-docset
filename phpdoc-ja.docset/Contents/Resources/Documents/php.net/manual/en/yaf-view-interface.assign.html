Yaf_View_Interface::assign

(Yaf >=1.0.0)

Yaf_View_Interface::assign — 値をビューエンジンに代入する

説明

abstract public bool Yaf_View_Interface::assign ( string $name [, string $value ] )

値をビューエンジンに代入します。これで、テンプレートから名前を指定してその値にアクセスできるようになります。

警告
