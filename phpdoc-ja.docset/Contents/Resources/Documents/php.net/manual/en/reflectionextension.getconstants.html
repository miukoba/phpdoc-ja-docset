ReflectionExtension::getConstants

(PHP 5)

ReflectionExtension::getConstants — 定数を取得する

説明

public array ReflectionExtension::getConstants ( void )

拡張モジュールで定義されている定数を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

定数名をキーとする連想配列を返します。

例

例1 ReflectionExtension::getConstants() の例

<?php
$ext = new ReflectionExtension('DOM');

print_r($ext->getConstants());
?>
