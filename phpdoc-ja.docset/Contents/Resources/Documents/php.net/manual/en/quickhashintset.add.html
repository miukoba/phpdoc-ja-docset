QuickHashIntSet::add

(PECL quickhash >= Unknown)

QuickHashIntSet::add — 新たなエントリをセットに追加する

説明

public bool QuickHashIntSet::add ( int $key )

このメソッドは、新たなエントリをセットに追加して追加に成功したかどうかを返します。 セットの作成時に QuickHashIntSet::CHECK_FOR_DUPES を渡さない限り、 エントリの追加はデフォルトで常に成功します。

パラメータ

key

追加するエントリのキー。

返り値

エントリが追加された場合に TRUE、追加されなかった場合に FALSE を返します。

例

例1 QuickHashIntSet::add() の例

<?php
echo "重複チェックなしの場合\n";
$set = new QuickHashIntSet( 1024 );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );

echo "\n重複チェックありの場合\n";
$set = new QuickHashIntSet( 1024, QuickHashIntSet::CHECK_FOR_DUPES );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );
?>

上の例の出力は、 たとえば以下のようになります。

重複チェックなしの場合
bool(false)
bool(true)
bool(true)
bool(true)

重複チェックありの場合
bool(false)
bool(true)
bool(true)
bool(false)
