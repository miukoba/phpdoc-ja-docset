SoapFault::__toString

(PHP 5 >= 5.0.1)

SoapFault::__toString — SoapFault の文字列表現を取得する

説明

public string SoapFault::__toString ( void )

SoapFault の文字列表現を返します。

パラメータ

この関数にはパラメータはありません。

返り値

SoapFault を文字列であらわした内容を返します。
