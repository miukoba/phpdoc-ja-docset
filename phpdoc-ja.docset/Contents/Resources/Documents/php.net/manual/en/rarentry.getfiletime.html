RarEntry::getFileTime

(PECL rar >= 0.1)

RarEntry::getFileTime — エントリの最終更新時刻を取得する

説明

public string RarEntry::getFileTime ( void )

エントリの最終更新時刻を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

エントリの最終更新時刻を YYYY-MM-DD HH:II:SS 形式で表した文字列、あるいはエラー時に FALSE を返します。
