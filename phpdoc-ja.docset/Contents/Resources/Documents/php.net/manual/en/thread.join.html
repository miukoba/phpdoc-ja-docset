Thread::join

(PECL pthreads >= 2.0.0)

Thread::join — 同期処理

説明

public boolean Thread::join ( void )

呼び出し元のコンテキストが、このスレッドの実行終了を待つようにします。

パラメータ

この関数にはパラメータはありません。

返り値

成功したかどうかを表す boolean 値を返します。

例

例1 スレッドとのジョイン

<?php
class My extends Thread {
    public function run() {
        /* ... */
    }
}
$my = new My();
$my->start();
/* ... */
var_dump($my->join());
/* ... */
?>

上の例の出力は以下となります。

bool(true)
