imap_qprint

(PHP 4, PHP 5)

imap_qprint — quoted-printable 文字列を 8 ビット文字列に変換する

説明

string imap_qprint ( string $string )

（» RFC2045, section 6.7 に基づき）quoted-printable 文字列を 8 ビット文字列に変換します。

パラメータ

string

quoted-printable 文字列。

返り値

8 ビット文字列を返します。

I just got rid of the loops using a regex.

<?php
function decode_qprint($str) {
    $str = preg_replace("/\=([A-F][A-F0-9])/","%$1",$str);
    $str = urldecode($str);
    $str = utf8_encode($str);
    return $str;
}
?>
