GmagickDraw::gettextdecoration

(PECL gmagick >= Unknown)

GmagickDraw::gettextdecoration — テキストのデコレーションを返す

説明

public int GmagickDraw::gettextdecoration ( void )

テキストのアノテーションに適用するデコレーションを返します。

パラメータ

この関数にはパラメータはありません。

返り値

DECORATION_ 定数のいずれかを返します。デコレーションが設定されていない場合は 0 を返します。
