xslt_getopt

(PHP 4 >= 4.3.0)

xslt_getopt — xsl プロセッサのオプションを取得する

説明

int xslt_getopt ( resource $processor )

xslt_getopt() は、与えられた processor のオプションを返します。

パラメータ

processor

xslt_create() で作成した XSLT プロセッサリンク ID。

返り値

オプションを返します。これは、定数 XSLT_SABOPT_XXX のビットマスクとなります。
