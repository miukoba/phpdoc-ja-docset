MongoDB::selectCollection

(PECL mongo >=0.9.0)

MongoDB::selectCollection — コレクションを取得する

説明

public MongoCollection MongoDB::selectCollection ( string $name )

パラメータ

name

コレクションの名前。

返り値

コレクションを返します。

エラー / 例外

コレクション名が不正な場合に Exception をスローします。
