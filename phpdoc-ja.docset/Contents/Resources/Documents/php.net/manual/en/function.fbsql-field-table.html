fbsql_field_table

(PHP 4 >= 4.0.6, PHP 5)

fbsql_field_table — 指定したフィールドがあるテーブルの名前を得る

説明

string fbsql_field_table ( resource $result [, int $field_offset ] )

指定したフィールドがあるテーブルの名前を返します。

パラメータ

result

fbsql_query() あるいは fbsql_db_query() が返す結果 ID。

field_offset

フィールドの数値オフセット。フィールドのインデックスは 0 からはじまります。

返り値

テーブル名を文字列で返します。
