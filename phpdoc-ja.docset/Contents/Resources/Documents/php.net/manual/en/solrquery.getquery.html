SolrQuery::getQuery

(PECL solr >= 0.9.2)

SolrQuery::getQuery — 主なクエリーを返す

説明

public string SolrQuery::getQuery ( void )

主な検索クエリーを返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合は文字列、設定されなかった場合には NULL を返します。
