Exception::__clone

(PHP 5 >= 5.1.0)

Exception::__clone — 例外をコピーする

説明

final private void Exception::__clone ( void )

例外のコピーを作成しようとします。結果としてこれは Fatal エラーとなります。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

エラー / 例外

例外はクローン不可能です。
