OAuth::enableDebug

(PECL OAuth >= 0.99.3)

OAuth::enableDebug — デバッグを有効にする

説明

public bool OAuth::enableDebug ( void )

デバッグに使用できる、冗長なリクエスト情報の出力を有効にします。 デバッグ情報は debugInfo メンバーに格納されます。 あるいは、debug メンバーに FALSE 以外の値を設定することによってもデバッグを有効にできます。

パラメータ

この関数にはパラメータはありません。

返り値

TRUE

変更履歴

バージョン 説明
0.99.8 debug および debugInfo メンバーが追加されました。
