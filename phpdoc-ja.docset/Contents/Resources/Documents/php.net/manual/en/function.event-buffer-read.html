event_buffer_read

(PECL libevent >= 0.0.1)

event_buffer_read — バッファつきイベントからデータを読み込む

説明

string event_buffer_read ( resource $bevent , int $data_size )

バッファつきイベントの入力バッファからデータを読み込みます。

パラメータ

bevent

有効なバッファつきイベントリソース。

data_size

データサイズ (バイト単位)。
