newt_suspend

(PECL newt >= 0.1)

newt_suspend — 端末を元の状態に戻すよう、newt に通知する

説明

void newt_suspend ( void )

端末を元の状態に戻すよう、newt に通知します。いったん実行されると、 (自分自身に SIGTSTP を送信する・子プログラムをフォークするなどの方法で) アプリケーションが自分自身でサスペンドすることが可能となります。

返り値

値を返しません。
