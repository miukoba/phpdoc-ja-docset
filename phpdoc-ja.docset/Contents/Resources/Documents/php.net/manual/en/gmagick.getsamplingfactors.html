Gmagick::getsamplingfactors

(PECL gmagick >= Unknown)

Gmagick::getsamplingfactors — 水平方向および垂直方向のサンプリング係数を取得する

説明

public array Gmagick::getsamplingfactors ( void )

水平方向および垂直方向のサンプリング係数を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

画像の水平方向および垂直方向のサンプリング係数を含む連想配列を返します。

エラー / 例外

エラー時に GmagickException をスローします。
