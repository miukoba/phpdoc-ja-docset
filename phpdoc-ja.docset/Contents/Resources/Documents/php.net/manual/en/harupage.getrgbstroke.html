HaruPage::getRGBStroke

(PECL haru >= 0.0.1)

HaruPage::getRGBStroke — 現在の描画色を取得する

説明

array HaruPage::getRGBStroke ( void )

現在の描画色を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の描画色を返します。

エラー / 例外

エラー時に HaruException をスローします。
