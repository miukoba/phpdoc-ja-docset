Yaf_Dispatcher::disableView

(Yaf >=1.0.0)

Yaf_Dispatcher::disableView — ビューのレンダリングを無効にする

説明

public bool Yaf_Dispatcher::disableView ( void )

ビューエンジンを無効にします。ユーザーが自分自身で出力しているようなアプリケーションで利用します。

注意:

アクションで FALSE を返せば、そのアクションでの自動レンダリングを回避できます。

パラメータ

この関数にはパラメータはありません。
