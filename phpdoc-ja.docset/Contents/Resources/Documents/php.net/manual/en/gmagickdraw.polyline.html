GmagickDraw::polyline

(PECL gmagick >= Unknown)

GmagickDraw::polyline — ポリラインを描く

説明

public GmagickDraw GmagickDraw::polyline ( array $coordinate_array )

現在の描画色、線幅、塗りつぶし色、塗りつぶしテクスチャを使って、指定した座標の配列によるポリラインを描きます。

パラメータ

coordinate_array

座標の配列。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
