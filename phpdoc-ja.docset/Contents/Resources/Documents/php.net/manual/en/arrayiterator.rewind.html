ArrayIterator::rewind

(PHP 5 >= 5.0.0)

ArrayIterator::rewind — 配列を最初に巻き戻す

説明

public void ArrayIterator::rewind ( void )

イテレータを最初に巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

例

例1 ArrayIterator::rewind() の例

<?php
$arrayobject = new ArrayObject();

$arrayobject[] = 'zero';
$arrayobject[] = 'one';
$arrayobject[] = 'two';

$iterator = $arrayobject->getIterator();

$iterator->next();
echo $iterator->key(); //1

$iterator->rewind(); //最初に巻き戻す
echo $iterator->key(); //0
?>
