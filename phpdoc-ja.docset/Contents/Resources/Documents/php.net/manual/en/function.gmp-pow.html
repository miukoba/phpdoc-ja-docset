gmp_pow

(PHP 4 >= 4.0.4, PHP 5)

gmp_pow — べき乗を計算する

説明

GMP gmp_pow ( GMP $base , int $exp )

base の exp 乗を計算します。

パラメータ

base

もととなる数。

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

exp

正の数で、base を何乗するかを指定します。

返り値

べき乗の結果を GMP 数で返します。 0^0 は 1 となります。

例

例1 gmp_pow() の例

<?php
$pow1 = gmp_pow("2", 31);
echo gmp_strval($pow1) . "\n";
$pow2 = gmp_pow("0", 0);
echo gmp_strval($pow2) . "\n";
$pow3 = gmp_pow("2", -1); // 負のべき乗を指定したため、警告を発生します
echo gmp_strval($pow3) . "\n";
?>

上の例の出力は以下となります。

2147483648
1
