Gmagick::getreleasedate

(PECL gmagick >= Unknown)

Gmagick::getreleasedate — GraphicsMagick のリリース日を返す

説明

public string Gmagick::getreleasedate ( void )

GraphicsMagick のリリース日を文字列で返します。

パラメータ

この関数にはパラメータはありません。

返り値

GraphicsMagick のリリース日を文字列で返します。

エラー / 例外

エラー時に GmagickException をスローします。
