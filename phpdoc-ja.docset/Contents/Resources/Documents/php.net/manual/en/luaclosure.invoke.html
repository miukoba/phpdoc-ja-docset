LuaClosure::__invoke

(PECL lua >=0.9.0)

LuaClosure::__invoke — luaclosure を起動

説明

public void LuaClosure::__invoke ( mixed $arg [, mixed $... ] )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

arg

...

返り値

例

例1 LuaClosure::__invoke()の例

<?php
$lua = new Lua();
$closure = $lua->eval(<<<CODE
    return (function ()
        print("hello world")
    end)
CODE
);

$lua->call($closure);
/* PHP 5.3 以降 */
$closure();
?>

上の例の出力は以下となります。

hello worldhello world
