event_del

(PECL libevent >= 0.0.1)

event_del — モニタするイベントから削除する

説明

bool event_del ( resource $event )

event をキャンセルします。

パラメータ

event

有効なイベントリソース。

返り値

event_del() は、成功した場合に TRUE、 エラー時に FALSE を返します。
