Imagick::displayImage

(PECL imagick 2.0.0)

Imagick::displayImage — 画像を表示する

説明

bool Imagick::displayImage ( string $servername )

このメソッドは、画像を X サーバーに表示します。

パラメータ

servername

X サーバーの名前。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
