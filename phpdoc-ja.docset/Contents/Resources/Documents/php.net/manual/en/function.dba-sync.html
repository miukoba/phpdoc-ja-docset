dba_sync

(PHP 4, PHP 5)

dba_sync — データベースを同期する

説明

bool dba_sync ( resource $handle )

dba_sync() は データベースを同期します。サポートされる場合には、 おそらくディスクへの物理的アクセスへのきっかけとなることでしょう。

パラメータ

handle

dba_open() あるいは dba_popen() によって返されたデータベースハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
