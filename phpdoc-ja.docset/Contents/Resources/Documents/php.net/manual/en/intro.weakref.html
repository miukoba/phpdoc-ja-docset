導入

一時的なオブジェクトへの、非侵入型のゲートウェイを提供します。 通常の (強い) 参照とは異なり、弱い参照はガベージコレクタがオブジェクトを解放するのを邪魔しません。 そのため、オブジェクトが破棄された後でもそのオブジェクトへの弱い参照が残っている可能性があります。 そんな場合は、弱い参照は自動的に無効になります。

例1 Weakref の使用例

<?php
class MyClass {
    public function __destruct() {
        echo "Destroying object!\n";
    }
}

$o1 = new MyClass;

$r1 = new Weakref($o1);

if ($r1->valid()) {
    echo "Object still exists!\n";
    var_dump($r1->get());
} else {
    echo "Object is dead!\n";
}

unset($o1);

if ($r1->valid()) {
    echo "Object still exists!\n";
    var_dump($r1->get());
} else {
    echo "Object is dead!\n";
}
?>

上の例の出力は以下となります。

Object still exists!
object(MyClass)#1 (0) {
}
Destroying object!
Object is dead!
