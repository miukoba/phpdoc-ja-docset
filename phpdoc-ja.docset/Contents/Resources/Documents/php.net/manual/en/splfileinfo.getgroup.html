SplFileInfo::getGroup

(PHP 5 >= 5.1.2)

SplFileInfo::getGroup — ファイルのグループを取得する

説明

public int SplFileInfo::getGroup ( void )

ファイルのグループを取得します。グループ ID を数値形式で返します。

パラメータ

この関数にはパラメータはありません。

返り値

グループ ID を数値形式で返します。

エラー / 例外

エラー時に RuntimeException をスローします。

例

例1 SplFileInfo::getGroup() の例

<?php
$info = new SplFileInfo(__FILE__);
print_r(posix_getgrgid($info->getGroup()));
?>
