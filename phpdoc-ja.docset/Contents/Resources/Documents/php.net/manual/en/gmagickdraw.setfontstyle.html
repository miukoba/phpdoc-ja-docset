GmagickDraw::setfontstyle

(PECL gmagick >= Unknown)

GmagickDraw::setfontstyle — テキストのアノテーションで使うフォントのスタイルを設定する

説明

public GmagickDraw GmagickDraw::setfontstyle ( int $style )

テキストのアノテーションで使うフォントのスタイルを設定します。 AnyStyle はワイルドカードで、「気にしない」という意味になります。

パラメータ

style

フォントスタイル (NormalStyle, ItalicStyle, ObliqueStyle, AnyStyle)。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
