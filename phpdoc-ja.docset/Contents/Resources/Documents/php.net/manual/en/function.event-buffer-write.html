event_buffer_write

(PECL libevent >= 0.0.1)

event_buffer_write — バッファつきイベントにデータを書き込む

説明

bool event_buffer_write ( resource $bevent , string $data [, int $data_size = -1 ] )

指定したバッファつきイベントにデータを書き込みます。 データは出力バッファに追記され、書き込み可能になったときに記述子に書き込まれます。

パラメータ

bevent

有効なバッファつきイベントリソース。

data

書き込むデータ。

data_size

オプションでサイズを指定します。event_buffer_write() は、 デフォルトでは data をすべて書き込みます。

返り値

event_buffer_write() は、成功した場合に TRUE、エラー時に FALSE を返します。
