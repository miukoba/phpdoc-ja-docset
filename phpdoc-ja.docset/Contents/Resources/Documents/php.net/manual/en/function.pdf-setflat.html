PDF_setflat

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setflat — 平面度を設定する

説明

bool PDF_setflat ( resource $pdfdoc , float $flatness )

平面度を設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
