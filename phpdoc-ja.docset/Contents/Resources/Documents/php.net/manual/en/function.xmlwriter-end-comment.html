XMLWriter::endComment

xmlwriter_end_comment

(PHP 5 >= 5.1.2, PECL xmlwriter >= 1.0.0)

XMLWriter::endComment -- xmlwriter_end_comment — コメントの終了部を作成する

説明

オブジェクト指向型

bool XMLWriter::endComment ( void )

手続き型

bool xmlwriter_end_comment ( resource $xmlwriter )

現在のコメントを終了します。

パラメータ

xmlwriter

手続き型のコールでのみ使用します。 変更される XMLWriter resource です。 このリソースは、xmlwriter_open_uri() あるいは xmlwriter_open_memory() のコールによって取得したものです。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
