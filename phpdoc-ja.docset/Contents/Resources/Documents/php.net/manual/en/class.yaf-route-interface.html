Yaf_Route_Interface クラス

(Yaf >=1.0.0)

導入

Yaf_Route_Interface を使うと、開発者がカスタムルートを定義できるようになります。

クラス概要

Yaf_Route_Interface {
/* メソッド */
abstract public string assemble ( array $info [, array $query ] )
abstract public bool route ( Yaf_Request_Abstract $request )
}
