MongoCollection::validate

(PECL mongo >=0.9.0)

MongoCollection::validate — コレクションを検証する

説明

public array MongoCollection::validate ([ bool $scan_data = FALSE ] )

パラメータ

scan_data

インデックスの検証のみを行い、コレクションは検証しない。

返り値

このオブジェクトをデータベースで評価した結果を返します。
