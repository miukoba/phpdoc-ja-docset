ocifetchinto

(PHP 4, PHP 5, PECL OCI8 >= 1.0.0)

ocifetchinto — 廃止された関数。oci_fetch_array() や oci_fetch_object()、 oci_fetch_assoc()、 oci_fetch_row() の変種

説明

oci_fetch_array() や oci_fetch_object()、 oci_fetch_assoc()、 oci_fetch_row() の変種です。

警告

このエイリアスは PHP 5.4.0 で 非推奨 となりました。 このエイリアスを使用しないことを強く推奨します。
