mysqli_stmt::free_result

mysqli_stmt_free_result

(PHP 5)

mysqli_stmt::free_result -- mysqli_stmt_free_result — 指定したステートメントハンドルの結果を格納したメモリを開放する

説明

オブジェクト指向型

void mysqli_stmt::free_result ( void )

手続き型

void mysqli_stmt_free_result ( mysqli_stmt $stmt )

ステートメントに関連する結果のメモリを開放します。このメモリは mysqli_stmt_store_result() によって割り当てられたものです。

パラメータ

stmt

手続き型のみ: mysqli_stmt_init() が返すステートメント ID。

返り値

値を返しません。
