ArrayIterator::natsort

(PHP 5 >= 5.2.0)

ArrayIterator::natsort — 自然順で配列をソートする

説明

public void ArrayIterator::natsort ( void )

"自然順" アルゴリズムを使って、値でエントリをソートします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
