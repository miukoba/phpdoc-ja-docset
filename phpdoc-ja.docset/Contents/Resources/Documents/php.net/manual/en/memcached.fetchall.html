Memcached::fetchAll

(PECL memcached >= 0.1.0)

Memcached::fetchAll — 残りのすべての結果を取得する

説明

public array Memcached::fetchAll ( void )

Memcached::fetchAll() は、 直近のリクエストで残っているすべての結果を返します。

パラメータ

この関数にはパラメータはありません。

返り値

結果、失敗した場合に FALSE を返します。 必要に応じて Memcached::getResultCode() を使用しましょう。

例

例1 Memcached::getDelayed() の例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->set('int', 99);
$m->set('string', 'a simple string');
$m->set('array', array(11, 12));

$m->getDelayed(array('int', 'array'), true);
var_dump($m->fetchAll());
?>

上の例の出力は以下となります。

array(2) {
  [0]=>
  array(3) {
    ["key"]=>
    string(3) "int"
    ["value"]=>
    int(99)
    ["cas"]=>
    float(2363)
  }
  [1]=>
  array(3) {
    ["key"]=>
    string(5) "array"
    ["value"]=>
    array(2) {
      [0]=>
      int(11)
      [1]=>
      int(12)
    }
    ["cas"]=>
    float(2365)
  }
}
