http_request_method_register

(PECL pecl_http >= 0.10.0)

http_request_method_register — リクエストメソッドを登録する

説明

int http_request_method_register ( string $method )

独自のリクエストメソッドを登録します。

パラメータ

method

登録するリクエストメソッドの名前。

返り値

成功した場合にリクエストメソッドの ID、 失敗した場合に FALSE を返します。
