streamWrapper::stream_flush

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_flush — 出力をフラッシュする

説明

public bool streamWrapper::stream_flush ( void )

このメソッドは、fflush() に対応してコールされます。

データをストリームにキャッシュした状態でまだストレージに格納していないのなら、 ここで格納しなければなりません。

パラメータ

この関数にはパラメータはありません。

返り値

キャッシュデータが正常に格納できた (あるいは格納するデータがなかった) 場合に TRUE、データを格納できなかった場合に FALSE を返します。

注意

注意:

実装されていない場合の返り値は FALSE です。
