ps_clip

(PECL ps >= 1.1.0)

ps_clip — 現在のパスを描画範囲として指定する

説明

bool ps_clip ( resource $psdoc )

現在のパスを、描画範囲の境界として定義します。 この領域の外に描画された内容は、見えなくなります。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
