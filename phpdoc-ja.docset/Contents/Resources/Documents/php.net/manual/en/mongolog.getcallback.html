MongoLog::getCallback

(PECL mongo >= 1.3.0)

MongoLog::getCallback — 以前に設定されたコールバック関数の名前を取得する

説明

public static callable MongoLog::getCallback ( void )

コールバック関数を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

コールバック関数を返します。まだ設定されていない場合は FALSE を返します。

注意

警告

この関数は、PHP 5.3.0 以降でしか使えません。
