MongoInt64 クラス

(PECL mongo >=1.0.9)

導入

このクラスを使うと、64 ビット整数値を 32 ビットシステム上のデータベースに保存することができます。

クラス概要

MongoInt64 {
/* フィールド */
public string $value ;
/* メソッド */
public __construct ( string $value )
public string __toString ( void )
}

フィールド

value
64 ビット数値を文字列で表した値。たとえば 123 の value は "123" となります。

目次

  • MongoInt64::__construct — 新しい 64 ビット整数値を作成する
  • MongoInt64::__toString — この 64 ビット整数値の文字列表現を返す
