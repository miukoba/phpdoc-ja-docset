ArrayIterator::uksort

(PHP 5 >= 5.2.0)

ArrayIterator::uksort — ユーザー定義のソート

説明

public void ArrayIterator::uksort ( string $cmp_function )

ユーザー定義の関数を使って、キーでエントリをソートします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

cmp_function

ソートに使う比較関数。

返り値

値を返しません。
