ReflectionClass::getExtensionName

(PHP 5)

ReflectionClass::getExtensionName — クラスを定義する拡張モジュールの名前を取得する

説明

public string ReflectionClass::getExtensionName ( void )

クラスを定義する拡張モジュールの名前を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

クラスを定義する拡張モジュールの名前を返します。 ユーザー定義のクラスの場合は FALSE を返します。

例

例1 ReflectionClass::getExtensionName() の基本的な使用例

<?php
$class = new ReflectionClass('ReflectionClass');
$extension = $class->getExtensionName();
var_dump($extension);
?>

上の例の出力は以下となります。

string(10) "Reflection"
