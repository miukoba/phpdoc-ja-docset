SplHeap::rewind

(PHP 5 >= 5.3.0)

SplHeap::rewind — イテレータを先頭に巻き戻す (何もしない)

説明

public void SplHeap::rewind ( void )

これは、イテレータを先頭に巻き戻します。ヒープの場合は何もしません。 というのもイテレータは仮想的なものであり、 実際はヒープの先頭から動くことがないからです。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
