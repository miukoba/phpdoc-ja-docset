Exception::getMessage

(PHP 5 >= 5.1.0)

Exception::getMessage — 例外メッセージを取得する

説明

final public string Exception::getMessage ( void )

例外メッセージを返します。

パラメータ

この関数にはパラメータはありません。

返り値

例外メッセージ文字列を返します。

例

例1 Exception::getMessage() の例

<?php
try {
    throw new Exception("Some error message");
} catch(Exception $e) {
    echo $e->getMessage();
}
?>

上の例の出力は、 たとえば以下のようになります。

Some error message
