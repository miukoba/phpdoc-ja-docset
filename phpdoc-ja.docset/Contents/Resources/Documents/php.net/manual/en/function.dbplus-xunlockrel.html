dbplus_xunlockrel

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_xunlockrel — リレーションの排他的ロックを解放する

説明

int dbplus_xunlockrel ( resource $relation )

以前に dbplus_xlockrel() で取得した排他的ロックを開放します。

パラメータ

relation

dbplus_open() でオープンしたリレーション。

返り値
