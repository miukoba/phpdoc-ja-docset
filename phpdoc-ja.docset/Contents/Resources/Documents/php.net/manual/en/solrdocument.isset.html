SolrDocument::__isset

(PECL solr >= 0.9.2)

SolrDocument::__isset — フィールドが存在するかチェック

説明

public bool SolrDocument::__isset ( string $fieldName )

フィールドが存在するかチェックします。

パラメータ

fieldName

フィールド名

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
