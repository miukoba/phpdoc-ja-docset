Reflector インターフェイス

(PHP 5)

導入

Reflector は、 すべてのエクスポート可能なリフレクションクラスが実装しているインターフェイスです。

クラス概要

Reflector {
/* メソッド */
public static string export ( void )
public string __toString ( void )
}
