QuickHashStringIntHash::get — キーを指定してハッシュから値を取得する

説明

public mixed QuickHashStringIntHash::get ( string $key )

キーを指定してハッシュから値を取得します。

パラメータ

key

エントリのキー。

返り値

キーが存在する場合はその値、キーが存在しない場合は NULL を返します。

例

例1 QuickHashStringIntHash::get() の例

<?php
$hash = new QuickHashStringIntHash( 8 );
var_dump( $hash->get( "one" ) );

var_dump( $hash->add( "two", 2 ) );
var_dump( $hash->get( "two" ) );
?>

上の例の出力は、 たとえば以下のようになります。

bool(false)
bool(true)
int(2)
