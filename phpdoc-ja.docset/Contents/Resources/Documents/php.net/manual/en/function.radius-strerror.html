radius_strerror

(PECL radius >= 1.1.0)

radius_strerror — エラーメッセージを返す

説明

string radius_strerror ( resource $radius_handle )

Radius 関数が失敗した場合にはエラーメッセージが記録されます。 この関数により、エラーメッセージを取得することが可能となります。

返り値

失敗した radius 関数からのエラーメッセージを文字列で返します。
