timezone_version_get

(PHP 5 >= 5.3.0)

timezone_version_get — timezonedb のバージョンを取得する

説明

string timezone_version_get ( void )

timezonedb の現在のバージョンを返します。

返り値

文字列を返します。

例

例1 timezonedb のバージョンの取得

<?php
echo timezone_version_get();
?>

上の例の出力は、 たとえば以下のようになります。

2009.7
