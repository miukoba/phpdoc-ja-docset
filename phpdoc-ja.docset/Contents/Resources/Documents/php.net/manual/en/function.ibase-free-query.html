ibase_free_query

(PHP 5)

ibase_free_query — プリペアドクエリにより確保されたメモリを解放する

説明

bool ibase_free_query ( resource $query )

プリペアドクエリを解放します。

パラメータ

query

ibase_prepare() で準備したクエリ。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
