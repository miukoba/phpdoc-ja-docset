IteratorIterator::current

(PHP 5 >= 5.1.0)

IteratorIterator::current — 現在の値を取得する

説明

public mixed IteratorIterator::current ( void )

現在の要素の値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の要素の値を返します。
