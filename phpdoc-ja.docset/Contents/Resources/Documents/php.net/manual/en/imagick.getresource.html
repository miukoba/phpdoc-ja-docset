Imagick::getResource

(PECL imagick 2.0.0)

Imagick::getResource — 指定したリソースのメモリ使用状況を返す

説明

int Imagick::getResource ( int $type )

指定したリソースのメモリ使用状況をメガバイト単位で返します。

パラメータ

type

リソース型定数 の一覧を参照ください。

返り値

指定したリソースのメモリ使用状況をメガバイト単位で返します。

エラー / 例外

エラー時に ImagickException をスローします。
