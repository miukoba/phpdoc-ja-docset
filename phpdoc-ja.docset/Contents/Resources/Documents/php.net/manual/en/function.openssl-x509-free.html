openssl_x509_free

(PHP 4 >= 4.0.6, PHP 5)

openssl_x509_free — 証明書リソースを開放する

説明

void openssl_x509_free ( resource $x509cert )

openssl_x509_free() は、メモリから指定した x509cert リソースに関連した証明書を開放します。

パラメータ

x509cert

返り値

値を返しません。
