Directory::read

(PHP 4, PHP 5)

Directory::read — ディレクトリハンドルからエントリを読み込む

説明

public string Directory::read ([ resource $dir_handle ] )

readdir() と同じですが、 dir_handle のデフォルトが $this になります。
