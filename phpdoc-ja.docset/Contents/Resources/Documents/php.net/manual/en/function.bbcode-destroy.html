bbcode_destroy

(PECL bbcode >= 0.9.0)

bbcode_destroy — BBCode_container リソースを閉じる

説明

bool bbcode_destroy ( resource $bbcode_container )

この関数は、bbcode_create() がオープンしたリソースを閉じます。

パラメータ

bbcode_container

bbcode_create() が返す BBCode_Container。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
