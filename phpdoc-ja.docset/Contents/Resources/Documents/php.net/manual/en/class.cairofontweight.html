CairoFontWeight クラス

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

導入

フォントのウェイトに基づいたフォントフェイスを指定します。

クラス概要

CairoFontWeight {
/* 定数 */
const integer NORMAL = 0 ;
const integer BOLD = 1 ;
}

定義済み定数

CairoFontWeight::NORMAL

ノーマル。

CairoFontWeight::BOLD

ボールド。
