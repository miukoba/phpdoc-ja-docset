Imagick::getImageWidth

(PECL imagick 2.0.0)

Imagick::getImageWidth — 画像の幅を返す

説明

int Imagick::getImageWidth ( void )

画像の幅を返します。

返り値

画像の幅を返します。

エラー / 例外

エラー時に ImagickException をスローします。
