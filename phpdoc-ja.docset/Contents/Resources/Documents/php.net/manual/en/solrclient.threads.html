SolrClient::threads

(PECL solr >= 0.9.2)

SolrClient::threads — スレッドの状態をチェック

説明

public void SolrClient::threads ( void )

スレッドの状態をチェック

パラメータ

この関数にはパラメータはありません。

返り値

SolrGenericResponse オブジェクトを返します。

エラー / 例外

クライアントの処理が失敗した場合や接続に問題が発生した場合に SolrClientException をスローします。

Solr サーバーがリクエストの処理に失敗した場合に SolrServerException をスローします。
