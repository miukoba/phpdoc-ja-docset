HttpResponse::getCache

(PECL pecl_http >= 0.10.0)

HttpResponse::getCache — キャッシュを取得する

説明

static bool HttpResponse::getCache ( void )

現在のキャッシュ設定を取得します。

返り値

キャッシュを行おうとしている場合に TRUE、 そうでない場合に FALSE を返します。
