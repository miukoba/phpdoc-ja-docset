SolrInputDocument::getField

(PECL solr >= 0.9.2)

SolrInputDocument::getField — 名前でフィールドを取得

説明

public SolrDocumentField SolrInputDocument::getField ( string $fieldName )

文書内のフィールドを取得します。

パラメータ

fieldName

フィールド名

返り値

成功時には SolrDocumentField オブジェクト、失敗時には FALSE を返します。
