swf_startsymbol

(PHP 4)

swf_startsymbol — シンボルを定義する

説明

void swf_startsymbol ( int $objid )

シンボルとしてオブジェクト ID を定義します。シンボルは、同時に実行可能な 小さな flash ムービーです。

パラメータ

objid

シンボルとして定義したいオブジェクトの ID。

返り値

値を返しません。
