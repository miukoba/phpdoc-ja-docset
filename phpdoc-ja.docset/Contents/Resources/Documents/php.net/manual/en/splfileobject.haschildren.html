SplFileObject::hasChildren

(PHP 5 >= 5.1.2)

SplFileObject::hasChildren — SplFileObject は子を持ちません

説明

public bool SplFileObject::hasChildren ( void )

SplFileObject は子を持たないのでこのメソッドは常に FALSE を返します。

パラメータ

この関数にはパラメータはありません。

返り値

Returns FALSE
