Imagick::affineTransformImage

(PECL imagick 2.0.0)

Imagick::affineTransformImage — 画像を変換する

説明

bool Imagick::affineTransformImage ( ImagickDraw $matrix )

アフィン行列の指定にしたがって画像を変換します。

パラメータ

matrix

アフィン行列。

返り値

成功した場合に TRUE を返します。
