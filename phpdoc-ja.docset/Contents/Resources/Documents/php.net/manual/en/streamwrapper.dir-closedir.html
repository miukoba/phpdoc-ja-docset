streamWrapper::dir_closedir

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::dir_closedir — ディレクトリハンドルを閉じる

説明

public bool streamWrapper::dir_closedir ( void )

このメソッドは closedir() に対応してコールされます。

ディレクトリストリームをオープンして使用している際にロックしたり割り当てたりしたリソースは、 すべて開放しなければなりません。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
