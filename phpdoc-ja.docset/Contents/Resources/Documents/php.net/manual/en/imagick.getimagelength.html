Imagick::getImageLength

(PECL imagick 2.0.0)

Imagick::getImageLength — 画像の長さをバイト数で取得する

説明

int Imagick::getImageLength ( void )

画像の長さをバイト数で返します。

返り値

現在の画像のサイズを整数値で返します。

例

例1 Imagick::getImageLength() の使用法

画像の長さをバイト数で取得します。

<?php
$image = new Imagick('test.jpg');
echo $image->getImageLength() . ' bytes';
?>
