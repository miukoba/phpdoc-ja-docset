導入

SNMP 拡張モジュールは非常にシンプルで使い勝手の良いツール群です。 Simple Network Managment Protocol を使ってリモートデバイスを管理することができます。

これは Net-SNMP (古いシステムでは UCD-SNMP) ライブラリのラッパーなので、 基本的な考え方はこのライブラリと同じです。また、Net-SNMP の設定ファイルや環境変数によって PHP の関数の挙動も変わります。

PHP 5.4 以降では、オブジェクト指向型の API も使えます。

Net-SNMP についての詳細な情報は » http://www.net-snmp.org/ にあります。
