rewinddir

(PHP 4, PHP 5)

rewinddir — ディレクトリハンドルを元に戻す

説明

void rewinddir ([ resource $dir_handle ] )

dir_handle で指定されたディレクトリの ストリームをディレクトリの先頭にリセットします。

パラメータ

dir_handle

opendir() が事前にオープンした ディレクトリハンドルリソース。 ディレクトリハンドルを指定しなかった場合は、 opendir() が最後にオープンしたものを使用します。
