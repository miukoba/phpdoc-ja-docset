dba_list

(PHP 4 >= 4.3.0, PHP 5)

dba_list — オープンされている全データベースファイルのリストを得る

説明

array dba_list ( void )

dba_list() は、オープンされている全ての データベースファイルの一覧を返します。

返り値

resourceid => filename 形式の連想配列を返します。
