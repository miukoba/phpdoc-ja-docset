eio_sync

(PECL eio >= 0.0.1dev)

eio_sync — バッファのキャッシュをディスクに書き込む

説明

resource eio_sync ([ int $pri = EIO_PRI_DEFAULT [, callable $callback = NULL [, mixed $data = NULL ]]] )

パラメータ

この関数にはパラメータはありません。

返り値

eio_sync() は、成功した場合にリクエストリソースを返します。エラー時に FALSE を返します。
