Gmagick::removeimage

(PECL gmagick >= Unknown)

Gmagick::removeimage — 画像リストから画像を削除する

説明

public void Gmagick::removeimage ( void )

画像リストから画像を削除します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
