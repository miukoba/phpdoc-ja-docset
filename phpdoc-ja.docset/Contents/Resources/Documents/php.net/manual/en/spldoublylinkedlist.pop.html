SplDoublyLinkedList::pop

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::pop — 双方向リンクリストの末尾からノードを取り出す

説明

public mixed SplDoublyLinkedList::pop ( void )

パラメータ

この関数にはパラメータはありません。

返り値

取り出したノードの値を返します。

エラー / 例外

データが空のときは RuntimeException をスローします。
