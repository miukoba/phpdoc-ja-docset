cairo_matrix_create_scale

(PECL cairo >= 0.1.0)

cairo_matrix_create_scale — のエイリアス CairoMatrix::initScale()

説明

この関数は次の関数のエイリアスです。 CairoMatrix::initScale().

この関数エイリアスは非推奨であり、下位互換性維持のために残されています。 今後、PHP から削除される可能性がありますので、この関数を使用しないことを推奨します。
