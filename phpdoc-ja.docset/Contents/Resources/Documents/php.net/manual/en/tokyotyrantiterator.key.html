TokyoTyrantIterator::key

(PECL tokyo_tyrant >= 0.2.0)

TokyoTyrantIterator::key — 現在のキーを返す

説明

public mixed TokyoTyrantIterator::key ( void )

現在のキーを返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に現在のキー、失敗した場合に false を返します。
