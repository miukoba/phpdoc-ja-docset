Gender\Gender::get

(PECL gender >= 0.6.0)

Gender\Gender::get — 性別を取得する

説明

public integer Gender\Gender::get ( string $name [, integer $country ] )

指定した名前について、特定の国における性別を取得します。

パラメータ

name

調べたい名前。

country

Gender クラスの定数で表される国 ID。

返り値

性別を返します。
