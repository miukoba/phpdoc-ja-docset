curl_strerror

(PHP 5 >= 5.5.0)

curl_strerror — エラーコードの説明を返す

説明

string curl_strerror ( int $errornum )

指定したエラーコードに対応するエラーメッセージを返します。

パラメータ

errornum

» cURL エラーコード 定数のいずれか。

返り値

エラーの説明を返します。無効なエラーコードを指定した場合は NULL を返します。

例

例1 curl_errno() の例

<?php
// URL をスペルミスした状態で curl ハンドルを作ります
$ch = curl_init("htp://example.com/");

// リクエストを送信します
curl_exec($ch);

// エラーをチェックし、エラーメッセージを表示します
if($errno = curl_errno($ch)) {
    $error_message = curl_strerror($errno);
    echo "cURL error ({$errno}):\n {$error_message}";
}

// ハンドルを閉じます
curl_close($ch);
?>

上の例の出力は以下となります。

cURL error (1):
 Unsupported protocol
