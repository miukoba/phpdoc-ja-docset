SplDoublyLinkedList::unserialize

(PHP 5 >= 5.4.0)

SplDoublyLinkedList::unserialize — ストレージをデシリアライズする

説明

public void SplDoublyLinkedList::unserialize ( string $serialized )

SplDoublyLinkedList::serialize() でシリアライズしたストレージを復元します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

serialized

シリアライズした文字列。

返り値

値を返しません。
