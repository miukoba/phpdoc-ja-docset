swf_setframe

(PHP 4)

swf_setframe — 指定したフレームに切替える

説明

void swf_setframe ( int $framenumber )

アクティブなフレームを framenumber で指定したフレームに変更します。

パラメータ

framenumber

設定するフレーム番号。

返り値

値を返しません。
