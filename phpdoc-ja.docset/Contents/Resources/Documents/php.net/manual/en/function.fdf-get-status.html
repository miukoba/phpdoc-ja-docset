fdf_get_status

(PHP 4, PHP 5)

fdf_get_status — /STATUS キーの値を得る

説明

string fdf_get_status ( resource $fdf_document )

/STATUS キーの値を取得します。

パラメータ

fdf_document

fdf_create()、 fdf_open() あるいは fdf_open_string() が返す FDF ドキュメントハンドル。

返り値

キーの値を文字列で返します。
