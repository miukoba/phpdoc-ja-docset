gupnp_service_action_return

(PECL gupnp >= 0.1.0)

gupnp_service_action_return — Return successfully

説明

bool gupnp_service_action_return ( resource $action )

Return successfully.

パラメータ

action

A service action identifier.

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
