maxdb_thread_safe

(PECL maxdb >= 7.6.06.04)

maxdb_thread_safe — スレッドセーフであるかどうかを返す

説明

bool maxdb_thread_safe ( void )

maxdb_thread_safe() は、 クライアントライブラリがスレッドセーフにコンパイルされているかどうかを示します。

返り値

クライアントライブラリがスレッドセーフである場合に TRUE、 それ以外の場合に FALSE を返します。
