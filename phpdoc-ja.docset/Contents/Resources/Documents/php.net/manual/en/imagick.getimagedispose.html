Imagick::getImageDispose

(PECL imagick 2.0.0)

Imagick::getImageDispose — 画像の配置方法を取得する

説明

int Imagick::getImageDispose ( void )

画像の配置方法を取得します。

返り値

エラー / 例外

エラー時に ImagickException をスローします。
