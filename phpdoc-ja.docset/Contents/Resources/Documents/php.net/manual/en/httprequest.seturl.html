HttpRequest::setUrl

(PECL pecl_http >= 0.10.0)

HttpRequest::setUrl — URL を設定する

説明

public bool HttpRequest::setUrl ( string $url )

リクエスト URL を設定します。

パラメータ

url

リクエスト URL。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
