Gender\Gender::connect

(PECL gender >= 0.6.0)

Gender\Gender::connect — 外部の名前ディレクトリに接続する

説明

public boolean Gender\Gender::connect ( string $dsn )

外部の名前ディレクトリに接続します。 今のところ、サポートしているのはストリームだけです。

パラメータ

dsn

開きたい DSN。

返り値

成功したかどうかを boolean で返します。
