SplFileInfo::getInode

(PHP 5 >= 5.1.2)

SplFileInfo::getInode — ファイルの inode を取得する

説明

public int SplFileInfo::getInode ( void )

ファイルシステムオブジェクトの inode 番号を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

ファイルシステムオブジェクトの inode 番号を返します。

エラー / 例外

エラー時に RuntimeException をスローします。
