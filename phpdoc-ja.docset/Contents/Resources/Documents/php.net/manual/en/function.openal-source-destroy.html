openal_source_destroy

(PECL openal >= 0.1.0)

openal_source_destroy — ソースリソースを削除する

説明

bool openal_source_destroy ( resource $source )

パラメータ

source

Open AL (ソース) リソース (事前に openal_source_create() で作成したもの)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
