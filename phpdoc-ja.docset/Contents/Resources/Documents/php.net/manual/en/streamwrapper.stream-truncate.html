streamWrapper::stream_truncate

(PHP 5 >= 5.4.0)

streamWrapper::stream_truncate — ストリームを丸める

説明

public bool streamWrapper::stream_truncate ( int $new_size )

ftruncate() などによる丸めの際にコールされます。

パラメータ

new_size

新しいサイズ。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
