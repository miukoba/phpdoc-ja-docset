eio_realpath

(PECL eio >= 0.0.1dev)

eio_realpath — 正規化した絶対パスを取得する

説明

resource eio_realpath ( string $path , int $pri , callable $callback [, string $data = NULL ] )

eio_realpath() は、正規化した絶対パスを callback 関数の result 引数で返します。

パラメータ

path

短い形式のパス。

pri

callback

data

返り値

例

例1 eio_realpath() の例

<?php
var_dump(getcwd());

function my_realpath_allback($data, $result) {
    var_dump($result);
}

eio_realpath("../", EIO_PRI_DEFAULT, "my_realpath_allback");
eio_event_loop();
?>

上の例の出力は、 たとえば以下のようになります。

string(12) "/home/ruslan"
string(5) "/home"
