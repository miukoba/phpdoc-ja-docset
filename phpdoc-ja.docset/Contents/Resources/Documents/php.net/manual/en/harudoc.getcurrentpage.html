HaruDoc::getCurrentPage

(PECL haru >= 0.0.1)

HaruDoc::getCurrentPage — ドキュメントの現在のページを返す

説明

object HaruDoc::getCurrentPage ( void )

ドキュメントの現在のページを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に HaruPage のインスタンス、 現在のページが設定されていない場合に FALSE を返します。

エラー / 例外

エラー時に HaruException をスローします。
