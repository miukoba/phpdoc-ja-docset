posix_getsid

(PHP 4, PHP 5)

posix_getsid — プロセスの現在の sid を得る

説明

int posix_getsid ( int $pid )

プロセス pid のセッション ID を返します。 プロセスのセッション ID とは、セッションリーダーのプロセスグループ ID のことです。

パラメータ

pid

プロセス ID。0 を指定すると、現在のプロセスとみなされます。

返り値

ID を表す整数値を返します。

例

例1 posix_getsid() の使用例

<?php
$pid = posix_getpid();
echo posix_getsid($pid); //8805
?>
