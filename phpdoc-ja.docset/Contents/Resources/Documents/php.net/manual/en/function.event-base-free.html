event_base_free

(PECL libevent >= 0.0.1)

event_base_free — イベントベースを破棄する

説明

void event_base_free ( resource $event_base )

指定した event_base を破棄し、 関連づけられたリソースをすべて解放します。 イベントがアタッチされているイベントベースは破棄できないことに注意しましょう。

パラメータ

event_base

有効なイベントベースリソース。
