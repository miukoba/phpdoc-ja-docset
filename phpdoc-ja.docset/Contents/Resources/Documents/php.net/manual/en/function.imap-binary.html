imap_binary

(PHP 4, PHP 5)

imap_binary — 8 ビット文字列を base64 文字列に変換する

説明

string imap_binary ( string $string )

（» RFC2045, Section 6.8 に基づき）8 ビット文字列を base64 文字列に変換します。

パラメータ

string

8 ビット文字列。

返り値

base64 エンコードされた文字列を返します。
