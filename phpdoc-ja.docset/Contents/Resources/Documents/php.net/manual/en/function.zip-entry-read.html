zip_entry_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PECL zip >= 1.0.0)

zip_entry_read — オープンされたディレクトリエントリから読み込む

説明

string zip_entry_read ( resource $zip_entry [, int $length = 1024 ] )

オープンされたディレクトリエントリから読み込みます。

パラメータ

zip_entry

zip_read()により返されたディレクトリエントリ

length

返すバイト数。

注意:

これは、読み込むデータの非圧縮時の長さとなります。

返り値

読み込んだデータを返します。ファイルの終端に達した場合は空文字列、エラー時には FALSE を返します。
