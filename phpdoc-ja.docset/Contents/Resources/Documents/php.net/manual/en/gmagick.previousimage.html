Gmagick::previousimage

(PECL gmagick >= Unknown)

Gmagick::previousimage — オブジェクト内の前の画像に移動する

説明

public bool Gmagick::previousimage ( void )

画像リスト内の前の画像を Gmagick オブジェクトに関連付けます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
