gmp_neg

(PHP 4 >= 4.0.4, PHP 5)

gmp_neg — 符号を反転する

説明

GMP gmp_neg ( GMP $a )

ある数の符号を反転したものを返します。

パラメータ

a

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

返り値

-a を GMP 数で返します。

例

例1 gmp_neg() の例

<?php
$neg1 = gmp_neg("1");
echo gmp_strval($neg1) . "\n";
$neg2 = gmp_neg("-1");
echo gmp_strval($neg2) . "\n";
?>

上の例の出力は以下となります。

-1
1
