SplDoublyLinkedList::offsetExists

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::offsetExists — 指定した $index が存在するかどうかを返す

説明

public bool SplDoublyLinkedList::offsetExists ( mixed $index )

パラメータ

index

調べたいインデックス。

返り値

指定した index が存在する場合に TRUE、 それ以外の場合に FALSE を返します。
