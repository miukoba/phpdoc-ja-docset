HaruPage::getCMYKStroke

(PECL haru >= 0.0.1)

HaruPage::getCMYKStroke — 現在の描画色を取得する

説明

array HaruPage::getCMYKStroke ( void )

現在の描画色を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の描画色を、4 つの要素 ("c", "m", "y" および "k") からなる配列で返します。

エラー / 例外

エラー時に HaruException をスローします。
