SolrDocument::getFieldCount

(PECL solr >= 0.9.2)

SolrDocument::getFieldCount — この文書内のフィールド数を返す

説明

public int SolrDocument::getFieldCount ( void )

この文書内のフィールド数を返します。 複数値のフィールドは１回だけ数えられます。

パラメータ

この関数にはパラメータはありません。

返り値

成功時には整数、失敗時には FALSE を返します。
