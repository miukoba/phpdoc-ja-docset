MongoLog::getLevel

(PECL mongo >= 1.2.3)

MongoLog::getLevel — 現在記録されているログレベルを取得する

説明

public static int MongoLog::getLevel ( void )

この関数を使うと、現在有効になっているログのレベルを調べることができます。 返される整数値をビット演算子で MongoLog レベル定数 と比較すると、ログレベルを判断できます。

<?php

if (MongoLog::getLevel() & MongoLog::FINE) {
    echo "大量のログ出力\n";
}

if (MongoLog::getLevel() ^ MongoLog::NONE) {
    echo "少なくとも少しだけはログ出力\n";
}

if (MongoLog::getLevel() == MongoLog::ALL) {
    echo "最高レベルのログ出力\n";
}

?>

パラメータ

この関数にはパラメータはありません。

返り値

現在記録されているログレベルを返します。
