Gmagick::getquantumdepth

(PECL gmagick >= Unknown)

Gmagick::getquantumdepth — Gmagick quantum depth を取得する

説明

public array Gmagick::getquantumdepth ( void )

Gmagick quantum depth を文字列で返します。

パラメータ

この関数にはパラメータはありません。

返り値

Gmagick quantum depth を文字列で返します。

エラー / 例外

エラー時に GmagickException をスローします。
