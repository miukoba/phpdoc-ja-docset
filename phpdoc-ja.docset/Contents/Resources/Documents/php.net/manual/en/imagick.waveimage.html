Imagick::waveImage

(PECL imagick 2.0.0)

Imagick::waveImage — ウェーブフィルタを画像に適用する

説明

bool Imagick::waveImage ( float $amplitude , float $length )

ウェーブフィルタを画像に適用します。 このメソッドは、ImageMagick バージョン 6.2.9 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

amplitude

波の振幅。

length

波長。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
