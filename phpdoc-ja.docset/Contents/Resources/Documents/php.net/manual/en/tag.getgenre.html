KTaglib_Tag::getGenre

(0.0.1)

KTaglib_Tag::getGenre — ID3 タグからジャンルを返す

説明

public string KTaglib_Tag::getGenre ( void )

ID3 タグのジャンルを返します。 このメソッドは、ID3v1 および ID3v2 タグで実装されています。

返り値

ジャンル文字列を返します。
