eio_set_max_poll_time

(PECL eio >= 0.0.1dev)

eio_set_max_poll_time — 最大のポール時間を設定する

説明

void eio_set_max_poll_time ( float $nseconds )

ポールの所要時間が nseconds 秒を超えると、処理を止めます。

パラメータ

nseconds

秒数。

返り値

値を返しません。
