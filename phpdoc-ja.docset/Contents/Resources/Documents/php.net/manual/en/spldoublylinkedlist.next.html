SplDoublyLinkedList::next

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::next — 次のエントリに移動する

説明

public void SplDoublyLinkedList::next ( void )

イテレータを次のノードに移動します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

$dll = new SplDoublyLinkedList();

$dll->push(2);
$dll->push(3);
$dll->unshift(5);
$dll->rewind();

while($dll->valid()){
    echo $dll->current().'<br />';
    $dll->next();//switch to next list item
}
