OAuth::generateSignature — シグネチャを生成する

説明

public string OAuth::generateSignature ( string $http_method , string $url [, mixed $extra_parameters ] )

最後の HTTP メソッド、URL そしてパラメータ文字列あるいは配列にもとづいてシグネチャを生成します。

パラメータ

http_method

リクエストの HTTP メソッド。

url

リクエストの URL。

extra_parameters

パラメータ文字列あるいは配列。

返り値

生成されたシグネチャを文字列で返します。失敗した場合に FALSE を返します
