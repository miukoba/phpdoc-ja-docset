XMLWriter::flush

xmlwriter_flush

(PHP 5 >= 5.1.2, PECL xmlwriter >= 1.0.0)

XMLWriter::flush -- xmlwriter_flush — 現在のバッファをフラッシュする

説明

オブジェクト指向型

mixed XMLWriter::flush ([ bool $empty = true ] )

手続き型

mixed xmlwriter_flush ( resource $xmlwriter [, bool $empty = true ] )

現在のバッファをフラッシュします。

パラメータ

xmlwriter

手続き型のコールでのみ使用します。 変更される XMLWriter resource です。 このリソースは、xmlwriter_open_uri() あるいは xmlwriter_open_memory() のコールによって取得したものです。

empty

バッファを空にするかどうか。デフォルトは TRUE です。

返り値

ライターをメモリにオープンした場合は、この関数は出来上がった XML バッファを返します。 そうではなく URI を使用している場合は、この関数はバッファを書き込み、 書き込んだバイト数を返します。
