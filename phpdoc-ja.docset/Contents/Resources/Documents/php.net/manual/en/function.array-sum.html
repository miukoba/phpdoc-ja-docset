array_sum

(PHP 4 >= 4.0.4, PHP 5)

array_sum — 配列の中の値の合計を計算する

説明

number array_sum ( array $array )

array_sum() は、配列の中の値の合計を整数または float として返します。

パラメータ

array

入力の配列。

返り値

値の合計を整数または float として返します。

例

例1 array_sum() の例

<?php
$a = array(2, 4, 6, 8);
echo "sum(a) = " . array_sum($a) . "\n";

$b = array("a" => 1.2, "b" => 2.3, "c" => 3.4);
echo "sum(b) = " . array_sum($b) . "\n";
?>

上の例の出力は以下となります。

sum(a) = 20
sum(b) = 6.9
