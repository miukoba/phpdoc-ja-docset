fdf_errno

(PHP 4 >= 4.3.0, PHP 5)

fdf_errno — 直近の fdf 操作に関するエラーコードを返す

説明

int fdf_errno ( void )

直近の FDF 関数コールによって設定されたエラーコードを取得します。

エラーの内容についてのテキストを取得するには fdf_error() を使用します。

返り値

エラーコードを表す整数値、あるいはエラーがない場合にゼロを返します。
