Iterator::next

(PHP 5 >= 5.0.0)

Iterator::next — 次の要素に進む

説明

abstract public void Iterator::next ( void )

現在位置を次の要素に移動します。

注意:

このメソッドは、各 foreach ループの 後 にコールされます。

パラメータ

この関数にはパラメータはありません。

返り値

返り値は無視されます。
