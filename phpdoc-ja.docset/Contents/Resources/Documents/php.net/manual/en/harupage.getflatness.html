HaruPage::getFlatness

(PECL haru >= 0.0.1)

HaruPage::getFlatness — ページの平坦度を取得する

説明

float HaruPage::getFlatness ( void )

ページの平坦度を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

ページの平坦度を返します。

エラー / 例外

エラー時に HaruException をスローします。
