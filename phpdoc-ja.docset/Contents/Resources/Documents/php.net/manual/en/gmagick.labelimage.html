Gmagick::labelimage

(PECL gmagick >= Unknown)

Gmagick::labelimage — ラベルを画像に追加する

説明

public mixed Gmagick::labelimage ( string $label )

ラベルを画像に追加します。

パラメータ

label

追加するラベル。

返り値

ラベルを追加した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
