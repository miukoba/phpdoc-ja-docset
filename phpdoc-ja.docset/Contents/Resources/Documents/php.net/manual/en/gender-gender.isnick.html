Gender\Gender::isNick

(PECL gender >= 0.9.0)

Gender\Gender::isNick — name0 が name1 のエイリアスかどうかを調べる

説明

public array Gender\Gender::isNick ( string $name0 , string $name1 [, integer $country ] )

name0 が name1 のエイリアスかどうかを調べます。

パラメータ

name0

調べる名前。

name1

調べる名前。

country

Gender クラスの定数で指定する国 ID。省略した場合は ANY_COUNTRY を使います。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
