Generator::__wakeup

(PHP 5 >= 5.5.0)

Generator::__wakeup — シリアライズのコールバック

説明

public void Generator::__wakeup ( void )

ジェネレータはシリアライズできないので、例外をスローします。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
