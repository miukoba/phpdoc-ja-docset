(PECL pthreads >= 2.0.0)

Thread::detach — Execution

説明

public void Thread::detach ( void )

Detaches the referenced Thread from the calling context, dangerously!

警告

This method can cause undefined, unsafe behaviour. It should not usually be used, it is present for completeness and advanced use cases.
