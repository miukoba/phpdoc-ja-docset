GmagickDraw::setfontweight

(PECL gmagick >= Unknown)

GmagickDraw::setfontweight — フォントのウェイトを設定する

説明

public GmagickDraw GmagickDraw::setfontweight ( int $weight )

テキストのアノテーションで使うフォントのウェイトを設定します。

パラメータ

weight

フォントのウェイト (100-900 の範囲が有効)。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
