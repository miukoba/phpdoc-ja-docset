ReflectionProperty::getDocComment

(PHP 5 >= 5.1.0)

ReflectionProperty::getDocComment — ドキュメントコメントを取得する

説明

public string ReflectionProperty::getDocComment ( void )

ドキュメントコメントを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

ドキュメントコメントを返します。
