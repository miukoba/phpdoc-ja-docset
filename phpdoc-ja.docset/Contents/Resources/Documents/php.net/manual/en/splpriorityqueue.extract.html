SplPriorityQueue::extract

(PHP 5 >= 5.3.0)

SplPriorityQueue::extract — ヒープの先頭からノードを取り出す

説明

public mixed SplPriorityQueue::extract ( void )

パラメータ

この関数にはパラメータはありません。

返り値

取り出したノードの値あるいは優先順位 (あるいはその両方) を、抽出フラグの設定に応じて返します。
