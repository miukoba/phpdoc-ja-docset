Worker::getStacked

(PECL pthreads >= 2.0.0)

Worker::getStacked — スタックの状態を調べる

説明

public integer Worker::getStacked ( void )

ワーカーが実行終了を待っているオブジェクトの数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

整数値を返します。

例

例1 ワーカーが実行終了を待っているオブジェクトの数を返す

<?php
class Work extends Threaded {
    /** ... **/

    public function run(){
        /** ... **/
    }
}
$my = new Worker();
/** ... **/
$work = new Work();
$my->stack($work);
/** ... **/
printf("My worker has %d jobs remaining\n", $my->getStacked());
/** ... **/
?>

上の例の出力は以下となります。

My worker has 5 jobs remaining
