MultipleIterator::getFlags

(PHP 5 >= 5.3.0)

MultipleIterator::getFlags — フラグの情報を取得する

説明

public void MultipleIterator::getFlags ( void )

フラグについての情報を取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

フラグについての情報を integer で返します。
