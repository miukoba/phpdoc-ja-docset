Gmagick::setimagefilename

(PECL gmagick >= Unknown)

Gmagick::setimagefilename — シーケンス内の特定の画像のファイル名を設定する

説明

public Gmagick Gmagick::setimagefilename ( string $filename )

シーケンス内の特定の画像のファイル名を設定します。

パラメータ

filename

画像ファイル名。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
