fann_get_errstr

(PECL fann >= 1.0.0)

fann_get_errstr — Returns the last errstr

説明

string fann_get_errstr ( resource $errdat )

Returns the last errstr.

パラメータ

errdat

ニューラルネットワークリソースあるいはニューラルネットワークトレーニングリソース。

返り値

The last error string, or FALSE on error.
