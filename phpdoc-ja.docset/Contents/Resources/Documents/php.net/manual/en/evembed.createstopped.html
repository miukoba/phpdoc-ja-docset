EvEmbed::createStopped

(PECL ev >= 0.2.0)

EvEmbed::createStopped — 停止した EvEmbed ウォッチャーのインスタンスを作る

説明

final public static void EvEmbed::createStopped ( object $other [, callable $callback [, mixed $data [, int $priority ]]] )

EvEmbed::__construct() と同じですが、ウォッチャーを自動的には開始しません。

パラメータ

other

EvEmbed::__construct() と同じ。

callback

ウォッチャーのコールバック を参照ください。

data

このウォッチャーに関連づけるカスタムデータ。

priority

ウォッチャーの優先度。

返り値

成功した場合に、停止した EvEmbed オブジェクトを返します。
