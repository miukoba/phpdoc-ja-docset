swf_actionsettarget

(PHP 4)

swf_actionsettarget — アクションのコンテキストを設定する

説明

void swf_actionsettarget ( string $target )

全てのアクションのコンテキストを設定します。 この関数を、現在実行中の他の flash ムービーを 制御するために使用することが可能です。

パラメータ

target

対象を表す文字列。

返り値

値を返しません。
