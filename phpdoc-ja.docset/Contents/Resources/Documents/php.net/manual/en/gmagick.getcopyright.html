Gmagick::getcopyright

(PECL gmagick >= Unknown)

Gmagick::getcopyright — GraphicsMagick API の著作権情報を文字列で返す

説明

public string Gmagick::getcopyright ( void )

GraphicsMagick API の著作権情報を文字列で返します。

返り値

GraphicsMagick および Magickwand C API の著作権情報を含む文字列を返します。

エラー / 例外

エラー時に GmagickException をスローします。
