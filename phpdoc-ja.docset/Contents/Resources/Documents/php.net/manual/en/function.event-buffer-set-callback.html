event_buffer_set_callback

(PECL libevent >= 0.0.4)

event_buffer_set_callback — バッファつきイベントのコールバックを設定あるいはリセットする

説明

bool event_buffer_set_callback ( resource $event , mixed $readcb , mixed $writecb , mixed $errorcb [, mixed $arg ] )

バッファつきイベントのコールバックを設定したり、既存のコールバックを変更したりします。

パラメータ

event

有効なバッファつきイベントリソース。

readcb

データを読み込むときに起動するコールバック。不要な場合は NULL を指定します。

writecb

記述子の書き込み準備ができたときに起動するコールバック。不要な場合は NULL を指定します。

errorcb

記述子でエラーが発生したときに起動するコールバック。不要な場合は NULL を指定します。

arg

各コールバックに渡す引数 (任意)。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
