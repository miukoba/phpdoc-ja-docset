Yaf_Application::getConfig

(Yaf >=1.0.0)

Yaf_Application::getConfig — 設定情報を取得する

説明

public Yaf_Config_Abstract Yaf_Application::getConfig ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Yaf_Config_Abstract のインスタンスを返します。

例

例1 Yaf_Application::getConfig() の例

<?php
$config = array(
    "application" => array(
        "directory" => realpath(dirname(__FILE__)) . "/application",
    ),
);

/** Yaf_Application */
$application = new Yaf_Application($config);
print_r($application->getConfig());
?>

上の例の出力は、 たとえば以下のようになります。

Yaf_Config_Simple Object
(
    [_config:protected] => Array
        (
            [application] => Array
                (
                    [directory] => /home/laruence/local/www/htdocs/application
                )

        )

    [_readonly:protected] => 1
)
