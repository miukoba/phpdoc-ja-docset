yp_cat

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5)

yp_cat — マップ全体を含む配列を返す

説明

array yp_cat ( string $domain , string $map )

すべてのマップエントリを返します。

パラメータ

domain

NIS ドメイン名。

map

NIS マップ。

返り値

すべてのマップエントリを配列で返します。 マップのキーが配列のインデックスに、 マップのエントリが配列のデータとなります。
