event_base_loop

(PECL libevent >= 0.0.1)

event_base_loop — イベントを処理する

説明

int event_base_loop ( resource $event_base [, int $flags = 0 ] )

指定したイベントベースのイベントループを開始します。

パラメータ

event_base

有効なイベントベースリソース。

flags

オプションのパラメータ。 EVLOOP_ONCE と EVLOOP_NONBLOCK の任意の組み合わせ。

返り値

event_base_loop() は、成功した場合に 0、エラー時に -1、 イベントが登録されていなければ 1 を返します。
