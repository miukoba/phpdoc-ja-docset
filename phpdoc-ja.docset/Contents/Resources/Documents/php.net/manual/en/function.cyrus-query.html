cyrus_query

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_query — Cyrus IMAP サーバーへクエリを送信する

説明

array cyrus_query ( resource $connection , string $query )

Cyrus IMAP サーバーへクエリを送信します。

パラメータ

connection

接続ハンドル。

query

クエリ文字列。

返り値

text、msgno、 そして keyword のキーを持つ連想配列を返します。
