HaruPage::getMiterLimit

(PECL haru >= 0.0.1)

HaruPage::getMiterLimit — マイターリミットの値を取得する

説明

float HaruPage::getMiterLimit ( void )

マイターリミットの値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

マイターリミットの値を返します。

エラー / 例外

エラー時に HaruException をスローします。
