AppendIterator::getArrayIterator

(PHP 5 >= 5.1.0)

AppendIterator::getArrayIterator — ArrayIterator を取得する

説明

public void AppendIterator::getArrayIterator ( void )

このメソッドは ArrayIterator を取得します。ここに、 AppendIterator::append() で追加したイテレータを格納します。

パラメータ

この関数にはパラメータはありません。

返り値

追加したイテレータを含む ArrayIterator を返します。
