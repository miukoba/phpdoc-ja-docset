EvChild::createStopped

(PECL ev >= 0.2.0)

EvChild::createStopped — 停止した EvChild ウォッチャーのインスタンスを作る

説明

final public static object EvChild::createStopped ( int $pid , bool $trace , callable $callback [, mixed $data [, int $priority ]] )

EvChild::__construct() と同じですが、ウォッチャーを自動的には開始しません。

パラメータ

pid

EvChild::__construct() と同じ。

trace

EvChild::__construct() と同じ。

callback

ウォッチャーのコールバック を参照ください。

data

このウォッチャーに関連づけるカスタムデータ。

priority

ウォッチャーの優先度。

返り値
