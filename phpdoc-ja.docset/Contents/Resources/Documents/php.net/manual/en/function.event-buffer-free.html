event_buffer_free

(PECL libevent >= 0.0.1)

event_buffer_free — バッファつきイベントを破棄する

説明

void event_buffer_free ( resource $bevent )

指定したバッファつきイベントを破棄し、関連づけられたすべてのリソースを解放します。

パラメータ

bevent

有効なバッファつきイベントリソース。
