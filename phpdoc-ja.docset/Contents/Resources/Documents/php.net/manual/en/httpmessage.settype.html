HttpMessage::setType

(PECL pecl_http >= 0.10.0)

HttpMessage::setType — メッセージの型を設定する

説明

public void HttpMessage::setType ( int $type )

メッセージの型を設定します。 HTTP_MSG_NONE、HTTP_MSG_REQUEST あるいは HTTP_MSG_RESPONSE のいずれかとなります。

パラメータ

type

HttpMessage::TYPE_*
