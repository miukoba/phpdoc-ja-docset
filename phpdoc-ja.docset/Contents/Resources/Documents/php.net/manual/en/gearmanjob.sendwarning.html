GearmanJob::sendWarning

(PECL gearman >= 0.6.0)

GearmanJob::sendWarning — Send a warning

説明

public bool GearmanJob::sendWarning ( string $warning )

Sends a warning for this job while it is running.

パラメータ

warning

A warning message.

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
