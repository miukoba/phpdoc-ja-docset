ArrayIterator::offsetGet

(PHP 5 >= 5.0.0)

ArrayIterator::offsetGet — オフセットの値を取得する

説明

public mixed ArrayIterator::offsetGet ( string $index )

指定したオフセットの値を取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

index

値を取得したいオフセット。

返り値

オフセット index の値を返します。
