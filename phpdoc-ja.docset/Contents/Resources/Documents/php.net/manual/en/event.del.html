Event::del

(PECL libevent >= 0.0.1)

Event::del — Makes event non-pending

説明

public bool Event::del ( void )

Removes an event from the set of monitored events, i.e. makes it non-pending.

パラメータ

この関数にはパラメータはありません。

返り値

Returns TRUE on success. Otherwise FALSE
