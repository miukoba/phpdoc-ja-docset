HttpRequest::getPostFiles

(PECL pecl_http >= 0.10.0)

HttpRequest::getPostFiles — POST ファイルを取得する

説明

public array HttpRequest::getPostFiles ( void )

事前に追加された POST ファイルをすべて取得します。

返り値

現在設定されている POST ファイルを含む配列を返します。
