GmagickDraw::getfillopacity

(PECL gmagick >= Unknown)

GmagickDraw::getfillopacity — 描画に使う不透明度を返す

説明

public float GmagickDraw::getfillopacity ( void )

描画に使う不透明度を返します。

パラメータ

この関数にはパラメータはありません。

返り値

塗りつぶしの色やテクスチャに使う不透明度を返します。完全に不透明な状態が 1.0 になります。
