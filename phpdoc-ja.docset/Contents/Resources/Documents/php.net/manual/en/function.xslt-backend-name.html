xslt_backend_name

(PHP 4 >= 4.3.0)

xslt_backend_name — バックエンドの名前を返す

説明

string xslt_backend_name ( void )

xslt_backend_name() は、バックエンドの名前を取得します。

返り値

Sablotron を返します。

例

例1 xslt_backend_name() の例

<?php

echo xslt_backend_name(); // Sablotron

?>
