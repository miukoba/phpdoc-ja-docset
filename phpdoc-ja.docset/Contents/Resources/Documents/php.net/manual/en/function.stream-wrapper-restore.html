stream_wrapper_restore

(PHP 5 >= 5.1.0)

stream_wrapper_restore — 事前に登録を解除された組み込みラッパーを復元する

説明

bool stream_wrapper_restore ( string $protocol )

stream_wrapper_unregister() で事前に登録を解除した 組み込みラッパーを復元します。

パラメータ

protocol

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
