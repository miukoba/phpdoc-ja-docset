counter_bump_value

counter_bump_value — カウンタリソースの現在の値を変更する

説明

voidcounter_bump_value ( resource $counter , integer $offset )

counter_bump_value() は、 カウンタリソースの現在の値を更新します。

パラメータ

counter
操作するカウンタリソース。
offset
カウンタの値の変更量。負の値を指定することもできます。
