apc_clear_cache

(PECL apc >= 2.0.0)

apc_clear_cache — APC キャッシュをクリアする

説明

bool apc_clear_cache ([ string $cache_type = "" ] )

ユーザー/システム キャッシュをクリアします。

パラメータ

cache_type

cache_type が "user" の場合はユーザーキャッシュがクリアされます。 それ以外の場合はシステムキャッシュ (キャッシュされたファイル) がクリアされます。

返り値

常に TRUE を返します。

Looking at the APC source, apc_clear_cache() only recognizes "user" as a cache_type.  If cache_type is omitted, or any other string (including "opcode") is provided, the effect is to clear the system cache.
