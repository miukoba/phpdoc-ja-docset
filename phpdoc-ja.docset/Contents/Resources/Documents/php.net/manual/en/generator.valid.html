Generator::valid

(PHP 5 >= 5.5.0)

Generator::valid — イテレータが終了したかどうかを調べる

説明

public bool Generator::valid ( void )

パラメータ

この関数にはパラメータはありません。

返り値

イテレータが終了した場合に FALSE を返します。まだ終了していない場合は TRUE を返します。
