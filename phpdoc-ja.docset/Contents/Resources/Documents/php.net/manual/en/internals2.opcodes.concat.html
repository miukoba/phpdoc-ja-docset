Concat . is a very useful tool in php while creating long strings or creating a set of data for example;

<?php
// GIVE THE STRING A NAME
$mystring="";

// NOW USE CONCAT . TO BUILD THE STRING

$mystring.="hello,";
$mystring.="world";

echo $mystring;
?>

output will be
hello,world
