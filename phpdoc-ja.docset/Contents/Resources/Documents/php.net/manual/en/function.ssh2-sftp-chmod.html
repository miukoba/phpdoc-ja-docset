ssh2_sftp_chmod

(PECL ssh2 >= 0.12)

ssh2_sftp_chmod — ファイルのモードを変更する

説明

bool ssh2_sftp_chmod ( resource $sftp , string $filename , int $mode )

指定したファイルのモードの mode への変更を試みます。

パラメータ

sftp

SSH2 SFTP リソース。ssh2_sftp() で開いたもの。

filename

ファイルへのパス。

mode

ファイルのパーミッション。このパラメータの詳細は chmod() を参照ください。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

例

例1 リモートサーバー上でのディレクトリの作成

<?php
$connection = ssh2_connect('shell.example.com', 22);
ssh2_auth_password($connection, 'username', 'password');
$sftp = ssh2_sftp($connection);

ssh2_sftp_chmod($sftp, '/somedir/somefile', 0755);
?>
