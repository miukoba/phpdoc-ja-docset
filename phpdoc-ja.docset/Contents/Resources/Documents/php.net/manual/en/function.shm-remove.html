shm_remove

(PHP 4, PHP 5)

shm_remove — Unix システムから共有メモリを削除する

説明

bool shm_remove ( resource $shm_identifier )

shm_remove() は、共有メモリ shm_identifier を削除します。 全てのデータは破棄されます。

パラメータ

shm_identifier

shm_attach() が返す 共有メモリ ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
