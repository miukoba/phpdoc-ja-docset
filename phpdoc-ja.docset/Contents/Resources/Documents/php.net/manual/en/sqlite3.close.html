SQLite3::close

(PHP 5 >= 5.3.0)

SQLite3::close — データベースとの接続を閉じる

説明

public bool SQLite3::close ( void )

データベースとの接続を閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE、失敗した場合に FALSE を返します。

例

例1 SQLite3::close() の例

<?php
$db = new SQLite3('mysqlitedb.db');
$db->close();
?>
