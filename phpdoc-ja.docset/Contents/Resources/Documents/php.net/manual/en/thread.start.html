Thread::start

(PECL pthreads >= 2.0.0)

Thread::start — 実行する

説明

public boolean Thread::start ([ integer $options ] )

新しいスレッドを開始し、実装されている run メソッドを実行します。

パラメータ

options

オプションのマスク。継承に関する定数で、デフォルトは PTHREADS_INHERIT_ALL。

返り値

成功したかどうかを表す boolean 値を返します。

例

例1 スレッドの開始

<?php
class My extends Thread {
    public function run() {
        /** ... **/
    }
}
$my = new My();
var_dump($my->start());
?>

上の例の出力は以下となります。

bool(true)
