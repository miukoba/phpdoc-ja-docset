Mongo::dropDB — データベースを削除する [非推奨]

説明

public array Mongo::dropDB ( mixed $db )
警告

非推奨

かわりに MongoDB::drop() を使ってください。

パラメータ

db

削除したいデータベース。MongoDB オブジェクト、あるいはデータベース名を指定します。

返り値

データベースからの応答を返します。
