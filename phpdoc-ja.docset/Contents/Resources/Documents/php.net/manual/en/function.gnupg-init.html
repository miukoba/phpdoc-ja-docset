gnupg_init

(PECL gnupg >= 0.4)

gnupg_init — 接続を初期化する

説明

resource gnupg_init ( void )

パラメータ

この関数にはパラメータはありません。

返り値

GnuPG 接続リソースを返します。これを他の GnuPG 関数で使用します。

例

例1 手続き型の gnupg_init() の例

<?php
$res = gnupg_init();
?>

例2 オブジェクト指向の gnupg の初期化の例

<?php
$gpg = new gnupg();
?>
