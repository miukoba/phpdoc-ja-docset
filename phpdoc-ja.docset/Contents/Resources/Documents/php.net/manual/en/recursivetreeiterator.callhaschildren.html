RecursiveTreeIterator::callHasChildren

(PHP 5 >= 5.3.0)

RecursiveTreeIterator::callHasChildren — 子を持っているかどうかを調べる

説明

public bool RecursiveTreeIterator::callHasChildren ( void )

各要素が子を持っているかどうかを調べるときにコールします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

子がある場合に TRUE、それ以外の場合に FALSE を返します。
