gmp_sub

(PHP 4 >= 4.0.4, PHP 5)

gmp_sub — 数値の減算

説明

GMP gmp_sub ( GMP $a , GMP $b )

a から b を引いた結果を返します。

パラメータ

a

引かれる数。

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

b

a から引く数。

PHP 5.5 以前での GMP 数リソース、PHP 5.6 以降での GMP オブジェクト、あるいは数値に変換可能な数値形式の文字列。

返り値

PHP 5.5 以前は GMP 数リソースを、そして PHP 5.6 以降は GMP オブジェクトを返します。

例

例1 gmp_sub() の例

<?php
$sub = gmp_sub("281474976710656", "4294967296"); // 2^48 - 2^32
echo gmp_strval($sub) . "\n";
?>

上の例の出力は以下となります。

281470681743360
