gc_disable

(PHP 5 >= 5.3.0)

gc_disable — 循環参照コレクタを無効にする

説明

void gc_disable ( void )

zend.enable_gc を 0 に設定して、循環参照コレクタを無効にします。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
