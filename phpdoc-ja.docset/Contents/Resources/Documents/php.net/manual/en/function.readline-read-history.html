readline_read_history

(PHP 4, PHP 5)

readline_read_history — ヒストリを読み込む

説明

bool readline_read_history ([ string $filename ] )

この関数は、コマンドヒストリをファイルから読み込みます。

パラメータ

filename

コマンドヒストリを含むファイルへのパス。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
