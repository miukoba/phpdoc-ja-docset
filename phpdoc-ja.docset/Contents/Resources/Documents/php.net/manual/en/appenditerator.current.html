AppendIterator::current

(PHP 5 >= 5.1.0)

AppendIterator::current — 現在の値を取得する

説明

public mixed AppendIterator::current ( void )

現在の値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

有効な状態の場合は現在の値、それ以外の場合は NULL を返します。
