RecursiveIterator::hasChildren

(PHP 5 >= 5.1.0)

RecursiveIterator::hasChildren — 現在のエントリのイテレータを作れるかどうかを返す

説明

public bool RecursiveIterator::hasChildren ( void )

現在のエントリのイテレータを作れるかどうかを返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のエントリが反復処理可能な場合に TRUE、それ以外の場合に FALSE を返します。
