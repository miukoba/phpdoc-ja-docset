    GearmanWorker::setId â€” Give the worker an identifier so it can be tracked when asking gearmand for the list of available workers.

    Description

    public bool GearmanWorker::setId ( string $id )

    Assigns the worker an identifier.

    Parameters

    id

    A string identifier.

    Return Values

    Returns TRUE on success or FALSE on failure.

    Examples

    Example #1 GearmanWorker::setId() example

    Set an identifier for a simple worker.

    <?php
    $worker    = new GearmanWorker();
    $worker->setId('test');
    ?>

    The above example will output something similar to:

    Run the following command:
gearadmin --workers
