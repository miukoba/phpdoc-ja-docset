Imagick::animateImages — 画像を動かす

説明

bool Imagick::animateImages ( string $x_server )

このメソッドは、ローカルあるいはリモートの X サーバーで画像を動かします。 このメソッドは Windows では使えません。 このメソッドは、ImageMagick バージョン 6.3.6 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

x_server

X サーバーのアドレス。

返り値

成功した場合に TRUE を返します。

参考

  • Imagick::displayImage() - 画像を表示する
