ArrayObject::asort

(PHP 5 >= 5.2.0)

ArrayObject::asort — 値でエントリをソートする

説明

public void ArrayObject::asort ( void )

エントリをソートします。キーとそれに対応する値の関連は保持されたままとなります。 これは主に、連想配列など実際の要素の並びが重要となる場合のソートで使います。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

例

例1 ArrayObject::asort() の例

<?php
$fruits = array("d" => "lemon", "a" => "orange", "b" => "banana", "c" => "apple");
$fruitArrayObject = new ArrayObject($fruits);
$fruitArrayObject->asort();

foreach ($fruitArrayObject as $key => $val) {
    echo "$key = $val\n";
}
?>

上の例の出力は以下となります。

c = apple
b = banana
d = lemon
a = orange

フルーツがアルファベット順で並べ替えられ、 各エントリに対応するキーもそれにあわせて並べ替えられます。
