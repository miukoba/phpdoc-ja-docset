PDF_set_horiz_scaling

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_horiz_scaling — テキストの横方向倍率を設定する [古い関数]

説明

この関数は PDFlib バージョン 3 で廃止されました。かわりに PDF_set_value() でパラメータ horizscaling を使用してください。
