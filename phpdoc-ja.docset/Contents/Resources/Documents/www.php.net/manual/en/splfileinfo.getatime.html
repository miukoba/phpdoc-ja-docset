SplFileInfo::getATime

(PHP 5 >= 5.1.2)

SplFileInfo::getATime — ファイルの最終アクセス時刻を取得する

説明

public int SplFileInfo::getATime ( void )

ファイルの最終アクセス時刻を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

ファイルに最後にアクセスがあった時刻を返します。

エラー / 例外

エラー時に RunTimeException をスローします。
