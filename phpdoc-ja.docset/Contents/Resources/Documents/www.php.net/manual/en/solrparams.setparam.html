SolrParams::setParam

(PECL solr >= 0.9.2)

SolrParams::setParam — パラメータを指定された値に設定

説明

public SolrParams SolrParams::setParam ( string $name , string $value )

クエリーパラメータを指定された値に設定します。 これは、一回だけ指定可能なパラメータで使われます。 同じパラメータ名で後で呼び出すと、既存の値を上書きします。

パラメータ

name

パラメータ名

value

パラメータ値

返り値

成功時に SolrParam オブジェクト、値の場合に FALSE を返します。

例

例1 SolrParams::setParam() の例

<?php

$param = new SolrParams();

$param->setParam('q', 'solr')->setParam('rows', 2);

?>

上の例の出力は、 たとえば以下のようになります。
