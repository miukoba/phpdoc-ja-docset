aggregate_methods

(PHP 4 >= 4.2.0)

aggregate_methods — クラスのメソッドを、動的にオブジェクトに集約する

説明

void aggregate_methods ( object $object , string $class_name )

クラスで定義されているすべてのメソッドを既存のオブジェクトに集約します。 ただし、クラスのコンストラクタは例外です。 また名前がアンダースコア文字 (_) で始まるメソッドも、 プライベートとみなされて例外となります。

パラメータ

object

class_name

返り値

値を返しません。

You can use this to dynamicily add methods to a class

class foo {
  function addMethod(  $code ) {
    $cname = uniqid("class");
    eval( "class ${cname} { ${code} }" );       
    aggregate_methods( $this , $cname );
  }

  function bar() {
    $this->addMethod( "function baz() { echo 'sha'; }" );
  }
}

$bang = new foo();
$bang->bar();
$bang->baz();
