Gmagick::getimageprofile

(PECL gmagick >= Unknown)

Gmagick::getimageprofile — 指定した名前の画像プロファイルを返す

説明

public string Gmagick::getimageprofile ( string $name )

指定した名前の画像プロファイルを返します。

パラメータ

この関数にはパラメータはありません。

返り値

画像プロファイルを含む文字列を返します。

エラー / 例外

エラー時に GmagickException をスローします。
