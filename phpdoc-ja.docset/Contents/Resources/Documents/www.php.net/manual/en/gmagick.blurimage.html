Gmagick::blurimage

(PECL gmagick >= Unknown)

Gmagick::blurimage — blur (にじみ) フィルタを画像に追加する

説明

public Gmagick Gmagick::blurimage ( float $radius , float $sigma [, int $channel ] )

blur (にじみ) フィルタを画像に追加します。

パラメータ

radius

にじみの半径。

sigma

標準偏差。

返り値

にじませた Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
