Gender\Gender::country

(PECL gender >= 0.8.0)

Gender\Gender::country — 国を表すテキストを取得する

説明

public array Gender\Gender::country ( integer $country )

Gender クラスの定数から、国を表す文字列を取得します。

パラメータ

country

Gender\Gender クラスの定数で表される国 ID。

返り値

成功した場合は、短い国名と完全な国名の配列を返します。 失敗した場合に FALSE を返します。

例

例1 Gender\Gender::country() の使用例

$gender = new Gender\Gender;
var_dump($gender->country(Gender\Gender::BRITAIN));

上の例の出力は以下となります。

array(2) {
  'country_short' =>
  string(2) "UK"
  'country' =>
  string(13) "Great Britain"
}
