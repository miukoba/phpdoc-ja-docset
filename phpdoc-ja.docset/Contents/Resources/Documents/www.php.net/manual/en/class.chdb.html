chdb クラス

(PECL chdb >= 0.1.0)

導入

読み込まれた chdb ファイルを表現します。

クラス概要

chdb {
/* メソッド */
public __construct ( string $pathname )
public string get ( string $key )
}

目次

  • chdb::__construct — chdb クラスのインスタンスを生成します。
  • chdb::get — キーに対応した値を取得する
