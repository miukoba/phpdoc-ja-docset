imageflip

(PHP 5 >= 5.5.0)

imageflip — 指定したモードで画像を反転させる

説明

bool imageflip ( resource $image , int $mode )

image を、指定した mode で反転させます。

パラメータ

image

imagecreatetruecolor() のような画像作成関数が返す画像リソース。

mode

反転のモード。定数 IMG_FLIP_* のいずれかを指定します。

定数 意味
IMG_FLIP_HORIZONTAL 水平方向に、上下を反転させます。
IMG_FLIP_VERTICAL 垂直方向に、左右を反転させます。
IMG_FLIP_BOTH 水平方向、垂直方向の両方に反転させます。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

例

例1 垂直方向の反転

この例では、定数 IMG_FLIP_VERTICAL を使います。

<?php
// 画像ファイル
$filename = 'phplogo.png';

// コンテントタイプ
header('Content-type: image/png');

// 読み込み
$im = imagecreatefrompng($filename);

// 垂直反転
imageflip($im, IMG_FLIP_VERTICAL);

// 出力
imagejpeg($im);
imagedestroy($im);
?>

注意

注意: この関数は、PHP がバンドル版の GD ライブラリで コンパイルされている場合のみ使用可能です。
