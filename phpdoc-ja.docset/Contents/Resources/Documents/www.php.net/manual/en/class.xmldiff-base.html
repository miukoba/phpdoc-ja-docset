The XMLDiff\Base class

(PECL xmldiff >= 0.8.0)

Base abstract class for all the comparsion classes in the extension.

XMLDiff\Base {

abstract public mixed diff ( mixed $from , mixed $to )
abstract public mixed merge ( mixed $src , mixed $diff )
