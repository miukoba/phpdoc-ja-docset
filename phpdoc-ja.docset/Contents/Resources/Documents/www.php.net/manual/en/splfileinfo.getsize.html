SplFileInfo::getSize

(PHP 5 >= 5.1.2)

SplFileInfo::getSize — ファイルサイズを取得する

説明

public int SplFileInfo::getSize ( void )

参照するファイルのファイルサイズをバイト数で返します。

パラメータ

この関数にはパラメータはありません。

返り値

ファイルサイズをバイト数で返します。

エラー / 例外

ファイルが存在しなかった場合やエラーが発生した場合に RuntimeException をスローします。
