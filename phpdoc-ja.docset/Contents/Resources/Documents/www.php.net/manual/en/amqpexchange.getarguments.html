AMQPExchange::getArguments

(PECL amqp >= Unknown)

AMQPExchange::getArguments — 指定したキュー上で設定されたすべての引数を取得する

説明

public array AMQPExchange::getArguments ( void )

指定した exchange 上で現在設定されているすべての引数を、キー/バリュー のペアの配列で取得します。

パラメータ

この関数にはパラメータはありません。

返り値

設定されているすべてのキー/バリューのペアを配列で返します。
