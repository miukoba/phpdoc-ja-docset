hwapi_attribute_new — hw_api_attribute クラスのインスタンスを作成する

説明

HW_API_Attribute hwapi_attribute_new ([ string $name [, string $value ]] )

指定した名前および値を使用して、hw_api_attribute の新しいインスタンスを作成します。

パラメータ

name

属性の名前。

value

属性の値。

返り値

hw_api_attribute のインスタンスを返します。
