AMQPConnection::isConnected

(PECL amqp >= Unknown)

AMQPConnection::isConnected — AMQPConnection オブジェクトがブローカと接続しているかどうかを調べる

説明

public bool AMQPConnection::isConnected ( void )

このメソッドは、AMQP ブローカへの接続がまだ有効な状態にあるかどうかを調べます。 直近のコマンドが返したステータスで判断します。

パラメータ

この関数にはパラメータはありません。

返り値

接続している場合に true、していない場合に false を返します。

例

例1 AMQPConnection::isConnected() の例

<?php

/* 新しい接続を作成します */
$conn = new AMQPConnection();
$conn->connect();

/* 接続が動作しているかどうかを調べます */
if ($conn->isConnected()) {
    echo "Connected to the broker \o/";
}
else {
    echo "Cannot connect to the broker";
}

?>
