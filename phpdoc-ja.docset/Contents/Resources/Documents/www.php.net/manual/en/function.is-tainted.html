is_tainted

(PECL taint >=0.1.0)

is_tainted — 文字列が汚染されているかどうかを調べる

説明

bool is_tainted ( string $string )

文字列が汚染されているかどうかを調べます。

パラメータ

string

返り値

文字列が汚染されている場合に TRUE、それ以外の場合に FALSE を返します。
