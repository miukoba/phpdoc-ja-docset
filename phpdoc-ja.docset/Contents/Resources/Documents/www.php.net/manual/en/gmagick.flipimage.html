Gmagick::flipimage

(PECL gmagick >= Unknown)

Gmagick::flipimage — 垂直方向に反転した画像を作成する

説明

public Gmagick Gmagick::flipimage ( void )

中心の x 軸を基準してピクセルを反転させ、垂直方向のミラー画像を作成します。

返り値

垂直反転した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
