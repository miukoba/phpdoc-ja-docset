apd_set_pprof_trace

(PECL apd >= 0.2)

apd_set_pprof_trace — セッションのデバッグを開始する

説明

string apd_set_pprof_trace ([ string $dump_directory = ini_get("apd.dumpdir") [, string $fragment = "pprof" ]] )

ダンプディレクトリの pprof_{process_id} へのデバッグを開始します。

パラメータ

dump_directory

プロファイルのダンプファイルを書き込むディレクトリ。 指定しない場合は、php.ini の設定 apd.dumpdir を使用します。

fragment

返り値

対象となるファイルへのパスを返します。

例

例1 apd_set_pprof_trace() の例

<?php
apd_set_pprof_trace();
?>
