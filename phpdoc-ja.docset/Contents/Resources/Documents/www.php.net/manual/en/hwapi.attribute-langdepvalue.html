hw_api_attribute::langdepvalue

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

説明

string hw_api_attribute::langdepvalue ( string $language )

属性の、指定した言語における値を返します。

パラメータ

language

属性の値を表す文字列を返します。
