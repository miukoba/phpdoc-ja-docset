Gmagick::getimagewidth

(PECL gmagick >= Unknown)

Gmagick::getimagewidth — 画像の幅を返す

説明

public int Gmagick::getimagewidth ( void )

画像の幅を返します。

パラメータ

この関数にはパラメータはありません。

返り値

画像の幅を返します。

エラー / 例外

エラー時に GmagickException をスローします。
