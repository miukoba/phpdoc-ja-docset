Imagick::getReleaseDate

(PECL imagick 2.0.0)

Imagick::getReleaseDate — ImageMagick のリリース日を返す

説明

string Imagick::getReleaseDate ( void )

ImageMagick のリリース日を文字列で返します。

返り値

ImageMagick のリリース日を文字列で返します。

エラー / 例外

エラー時に ImagickException をスローします。
