EvPrepare::createStopped

(PECL ev >= 0.2.0)

EvPrepare::createStopped — 停止した EvPrepare ウォッチャーのインスタンスを作る

説明

final public static EvPrepare EvPrepare::createStopped ( callable $callback [, mixed $data = NULL [, int $priority = 0 ]] )

停止した EvPrepare ウォッチャーのインスタンスを作ります。 EvPrepare::__construct() とは違い、 このメソッドはウォッチャーを自動的に開始させません。

パラメータ

callback

ウォッチャーのコールバック を参照ください。

data

このウォッチャーに関連づけるカスタムデータ。

priority

ウォッチャーの優先度。

返り値

成功した場合に EvPrepare オブジェクトを返します。
