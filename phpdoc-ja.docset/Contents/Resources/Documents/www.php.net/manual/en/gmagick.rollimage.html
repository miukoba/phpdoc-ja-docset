Gmagick::rollimage

(PECL gmagick >= Unknown)

Gmagick::rollimage — 画像を補正する

説明

public void Gmagick::rollimage ( int $x , int $y )

x および y を指定して画像を補正します。

パラメータ

x

x オフセット。

y

y オフセット。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
