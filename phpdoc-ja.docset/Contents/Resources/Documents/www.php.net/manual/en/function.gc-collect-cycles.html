gc_collect_cycles

(PHP 5 >= 5.3.0)

gc_collect_cycles — すべての既存ガベージサイクルを強制的に収集する

説明

int gc_collect_cycles ( void )

すべての既存ガベージサイクルを強制的に収集します。

パラメータ

この関数にはパラメータはありません。

返り値

収集したサイクルの数を返します。
