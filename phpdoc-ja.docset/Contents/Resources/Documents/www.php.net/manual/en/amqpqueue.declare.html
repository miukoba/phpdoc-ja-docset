AMQPQueue::declare

(PECL amqp >= Unknown)

AMQPQueue::declare — 新しいキューを宣言する

説明

public int AMQPQueue::declare ( void )

新しいキューをブローカ上で宣言します。

パラメータ

エラー / 例外

チャネルが開いていない場合に AMQPChannelException をスローします。

ブローカへの接続が途切れた場合に AMQPConnectionException をスローします。

返り値

メッセージ数を返します。
