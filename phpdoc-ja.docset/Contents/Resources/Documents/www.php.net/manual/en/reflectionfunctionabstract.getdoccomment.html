ReflectionFunctionAbstract::getDocComment

(PHP 5 >= 5.1.0)

ReflectionFunctionAbstract::getDocComment — ドキュメントコメントを取得する

説明

public string ReflectionFunctionAbstract::getDocComment ( void )

関数のドキュメントコメントを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

ドキュメントコメントが存在する場合はそれを返します。それ以外の場合は FALSE を返します。
