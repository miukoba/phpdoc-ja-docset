ReflectionProperty::getName

(PHP 5)

ReflectionProperty::getName — プロパティ名を取得する

説明

public string ReflectionProperty::getName ( void )

プロパティ名を取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

調べているプロパティの名前を返します。
