QuickHashIntSet::saveToString

(PECL quickhash >= Unknown)

QuickHashIntSet::saveToString — シリアライズしたセットを返す

説明

public string QuickHashIntSet::saveToString ( void )

このメソッドは、シリアライズしたセットを返します。 loadFromString() で読めるフォーマットになります。

返り値

このメソッドは、セットをシリアライズした文字列を返します。 各要素は 4 バイトの値となり、システムで使っているのと同じエンディアンになります。

例

例1 QuickHashIntSet::saveToString() の例

<?php
$set = new QuickHashIntSet( 1024 );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );

var_dump( $set->saveToString() );
?>
