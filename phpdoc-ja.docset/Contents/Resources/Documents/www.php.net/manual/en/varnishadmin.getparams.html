VarnishAdmin::getParams

(PECL varnish >= 0.4)

VarnishAdmin::getParams — 現在の varnish インスタンスの設定パラメータを取得する

説明

public array VarnishAdmin::getParams ( void )

パラメータ

この関数にはパラメータはありません。

返り値

varnish 設定パラメータの配列を返します。
