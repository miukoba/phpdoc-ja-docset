posix_getpid

(PHP 4, PHP 5)

posix_getpid — 現在のプロセス ID を返す

説明

int posix_getpid ( void )

現在のプロセスのプロセス ID を返します。

返り値

ID を表す整数値を返します。

例

例1 posix_getpid() の使用例

<?php
echo posix_getpid(); //8805
?>
