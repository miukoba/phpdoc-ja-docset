ob_etaghandler

(PECL pecl_http >= 0.10.0)

ob_etaghandler — ETag 出力ハンドラ

説明

string ob_etaghandler ( string $data , int $mode )

ob_start() とともに使用します。

この出力バッファハンドラは、 INI 設定 http.etag.mode で指定したハッシュアルゴリズムを使用して、ETag を生成します。

この出力ハンドラは、 http_cache_etag() によって使用されます。
