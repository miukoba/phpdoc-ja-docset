MongoLog::setLevel

(PECL mongo >= 1.2.3)

MongoLog::setLevel — ログレベルを設定する

説明

public static void MongoLog::setLevel ( int $level )

この関数で、どの程度の冗長性でログを記録するかとどんな活動を記録するかを設定します。 MongoLog のセクションで説明した定数をビット演算子でつないでレベルを指定します。

<?php

// まず、モジュールのログ記録を有効にします
MongoLog::setModule(MongoLog::POOL);


// すべてのレベルのログを記録します
MongoLog::setLevel(MongoLog::ALL);

// 警告と情報を記録します
MongoLog::setLevel(MongoLog::WARNING|MongoLog::INFO);

// 良好な活動以外はすべて記録します
MongoLog::setLevel(MongoLog::ALL & (~MongoLog::FINE));

?>

MongoLog::setModule() を使って、 ドライバのどの部分をログに記録するのかを選ばなければならないことに注意しましょう。

パラメータ

level

記録したいログのレベル。
