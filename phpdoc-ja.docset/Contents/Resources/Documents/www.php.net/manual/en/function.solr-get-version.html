solr_get_version

(PECL solr >= 0.9.1)

solr_get_version — Apache Solr エクステンションの現行バージョンを返す

説明

string solr_get_version ( void )

この関数は、エクステンションの現行バージョンを文字列として返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功時に文字列、失敗時に FALSE を返します。

エラー / 例外

この関数はエラーや例外をスローしません。

例

例1 solr_get_version() の例

例の目的や、例で何が行なわれるか示すテキストは全てここになければいけません。

<?php

$solr_version = solr_get_version();

print $solr_version;

?>

上の例の出力は、 たとえば以下のようになります。
