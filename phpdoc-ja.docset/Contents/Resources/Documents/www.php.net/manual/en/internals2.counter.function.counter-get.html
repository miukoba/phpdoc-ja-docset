counter_get

counter_get — 基本カウンタの現在の値を取得する

説明

integer counter_get ( void )

counter_get() は、 基本インターフェイスのカウンタの現在値を返します。

返り値

counter_get() は整数値を返します。

参考

  • counter_bump() - 基本カウンタの現在の値を更新する
  • counter_reset() - 基本カウンタの現在の値をリセットする
