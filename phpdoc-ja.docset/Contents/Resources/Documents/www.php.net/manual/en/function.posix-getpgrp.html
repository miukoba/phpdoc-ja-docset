posix_getpgrp

(PHP 4, PHP 5)

posix_getpgrp — 現在のプロセスのグループ ID を返す

説明

int posix_getpgrp ( void )

現在のプロセスのグループ ID を返します。

返り値

ID を表す整数値を返します。

参考

  • プロセスグループの詳細な情報に関しては、POSIX.1 および POSIX システムのマニュアルページ getpgrp(2) を参照ください。
