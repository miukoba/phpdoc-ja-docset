Gmagick::getimagechanneldepth

(PECL gmagick >= Unknown)

Gmagick::getimagechanneldepth — 特定の画像チャネルの深度を返す

説明

public int Gmagick::getimagechanneldepth ( int $channel_type )

特定の画像チャネルの深度を返します。

パラメータ

この関数にはパラメータはありません。

返り値

画像チャネルの深度を返します。

エラー / 例外

エラー時に GmagickException をスローします。
