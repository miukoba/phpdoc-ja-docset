event_buffer_enable

(PECL libevent >= 0.0.1)

event_buffer_enable — バッファつきイベントを有効にする

説明

bool event_buffer_enable ( resource $bevent , int $events )

指定したバッファつきイベントを有効にします。

パラメータ

bevent

有効なバッファつきイベントリソース。

events

EV_READ と EV_WRITE の任意の組み合わせ。

返り値

event_buffer_enable() は、成功した場合に TRUE、エラー時に FALSE を返します。
