Gmagick::getimageredprimary

(PECL gmagick >= Unknown)

Gmagick::getimageredprimary — 赤が一番強い点を返す

説明

public array Gmagick::getimageredprimary ( void )

赤が一番強い点を返します。"x" および "y" というキーを持つ配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

赤が一番強い点を "x" および "y" というキーを持つ配列で返します。

エラー / 例外

エラー時に GmagickException をスローします。
