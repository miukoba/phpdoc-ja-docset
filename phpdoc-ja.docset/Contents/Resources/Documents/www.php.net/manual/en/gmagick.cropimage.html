Gmagick::cropimage

(PECL gmagick >= Unknown)

Gmagick::cropimage — 画像の一部を抽出する

説明

public Gmagick Gmagick::cropimage ( int $width , int $height , int $x , int $y )

画像の一部を抽出します。

パラメータ

width

抽出する幅。

height

抽出する高さ。

x

抽出する領域の左上の X 座標。

y

抽出する領域の左上の Y 座標。

返り値

抽出した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。

A slight note since this is not immediately obvious:
Just as the C CropImage() function, you can specify all zero parameters to get an "autocrop" effect, meaning it will remove all single-color "edges" of an image.
