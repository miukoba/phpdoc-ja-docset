Gmagick::borderimage

(PECL gmagick >= Unknown)

Gmagick::borderimage — 画像の周りを枠線で囲む

説明

public Gmagick Gmagick::borderimage ( GmagickPixel $color , int $width , int $height )

画像の周りを枠線で囲みます。枠線の色は GmagickPixel オブジェクトあるいは色文字列で指定します。

パラメータ

color

枠線の色を含む GmagickPixel オブジェクトあるいは文字列。

width

枠線の幅。

height

枠線の高さ。

返り値

枠線を定義した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
