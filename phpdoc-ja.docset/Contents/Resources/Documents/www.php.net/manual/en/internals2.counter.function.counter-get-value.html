counter_get_value

counter_get_value — カウンタリソースの現在の値を取得する

説明

integer counter_get_value ( resource $counter )

counter_get_value() は、 カウンタリソースの現在の値を返します。

パラメータ

counter
操作するカウンタリソース。

返り値

counter_get_value() は整数値を返します。

参考

  • counter_bump_value() - カウンタリソースの現在の値を変更する
  • counter_reset_value() - カウンタリソースの現在の値をリセットする
