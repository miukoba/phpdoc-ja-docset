Making the upgrade from PHP 5.3 to 5.5, I noticed the set of functions I was using to convert a Unix time stamp to a friendly time stamp no longer worked. It's now necessary to set the timezone BEFORE attempting to create a new DateTime object. Below is a function which will convert a Unix time stamp to a human readable format working as of PHP 5.5

function formatTimeStamp($unixTimeStamp)
{
    date_default_timezone_set('America/Chicago'); // a full list of timezones is available at http://php.net/manual/en/timezones.php         
    $time = new DateTime("@" . $unixTimeStamp);
    return $time->format("Y-m-d H:i:s"); //convert to friendly date stamp
}
