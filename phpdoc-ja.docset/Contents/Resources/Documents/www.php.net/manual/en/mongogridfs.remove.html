MongoGridFS::remove

(PECL mongo >=0.9.0)

MongoGridFS::remove — ファイルをコレクションから削除する

説明

public bool MongoGridFS::remove ([ array $criteria = array() [, array $options = array() ]] )

パラメータ

query

探したいファイル名あるいは条件。

options

削除時のオプション。以下のオプションが使えます。

  • "w"

    WriteConcerns を参照ください。MongoClient でのデフォルト値は 1 です。

返り値

削除がデータベースに正しく送信されたかどうかを返します。
