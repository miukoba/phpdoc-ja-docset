libxml_get_last_error

(PHP 5 >= 5.1.0)

libxml_get_last_error — libxmlから直近のエラーを取得する

説明

LibXMLError libxml_get_last_error ( void )

libxmlから直近のエラーを取得します。

返り値

エラーがバッファにある場合にLibXMLErrorオブジェクト、 それ以外の場合に FALSE を返します。
