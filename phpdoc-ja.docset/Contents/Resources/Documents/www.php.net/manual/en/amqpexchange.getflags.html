AMQPExchange::getFlags

(PECL amqp >= Unknown)

AMQPExchange::getFlags — フラグのビットマスクを取得する

説明

public int AMQPExchange::getFlags ( void )

指定した exchange 上で現在設定されているすべてのフラグを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

この exchange オブジェクト上で現在設定されているすべてのフラグのビットマスクを返します。
