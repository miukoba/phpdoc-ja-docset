Worker::getCreatorId

(PECL pthreads >= 0.36)

Worker::getCreatorId — 識別

説明

final public long Worker::getCreatorId ( void )

このスレッドを作ったスレッドの ID を返します。

パラメータ

この関数にはパラメータはありません。

返り値

数値の ID を返します。

例

例1 このワーカーを作ったスレッドあるいはプロセスの ID を返す

<?php
class My extends Thread {
    public function run() {
        printf("%s created by Thread #%lu\n", __CLASS__, $this->getCreatorId());
    }
}
$my = new My();
$my->start();
?>

上の例の出力は以下となります。

My created by Thread #123456778899
