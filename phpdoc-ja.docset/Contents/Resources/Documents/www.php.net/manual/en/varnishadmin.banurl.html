VarnishAdmin::banUrl

(PECL varnish >= 0.3)

VarnishAdmin::banUrl — VCL の式を使って URL を禁止する

説明

public int VarnishAdmin::banUrl ( string $vcl_regex )

パラメータ

vcl_regex

PCRE 互換形式で指定した、URL の正規表現。 varnish の ban.url コマンドに基づくもの。

返り値

varnish のコマンドステータスを返します。
