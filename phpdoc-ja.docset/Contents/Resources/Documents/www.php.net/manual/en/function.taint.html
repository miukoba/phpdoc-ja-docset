taint

(PECL taint >=0.1.0)

taint — 文字列を汚染させる

説明

bool taint ( string &$string [, string $... ] )

文字列を汚染させます。この関数は、テスト用としてのみ使うものです。

パラメータ

string

...

返り値

変換に成功したときに TURE を返します。 taint 拡張モジュールが有効になっていない場合は常に TRUE を返します。
