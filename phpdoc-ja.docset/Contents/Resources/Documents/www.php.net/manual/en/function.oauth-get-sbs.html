oauth_get_sbs

(PECL OAuth >=0.99.7)

oauth_get_sbs — SBSを生成する

説明

string oauth_get_sbs ( string $http_method , string $uri [, array $request_parameters ] )

pecl/oauth に関するSBSを生成します

パラメータ

http_method

HTTPメソッド

uri

エンコードするURI

request_parameters

リクエストパラメータの配列

返り値

SBSを返します
