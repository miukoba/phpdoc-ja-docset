DirectoryIterator::seek

(PHP 5 >= 5.3.0)

DirectoryIterator::seek — DirectoryIterator のアイテムを移動する

説明

public void DirectoryIterator::seek ( int $position )

DirectoryIterator の指定した位置に移動します。

パラメータ

position

異動先を表す、ゼロから始まる位置。

返り値

値を返しません。

例

例1 DirectoryIterator::seek() の例

スクリプトを含むディレクトリの四番目のアイテムに移動します。 最初の二つは、通常は . と .. です。

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
$iterator->seek(3);
if ($iterator->valid()) {
    echo $iterator->getFilename();
} else {
    echo 'No file at position 3';
}
?>
