PDF_get_buffer

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_get_buffer — PDF 出力バッファを取得する

説明

string PDF_get_buffer ( resource $p )

生成された PDF データを含むバッファの内容を取得します。
