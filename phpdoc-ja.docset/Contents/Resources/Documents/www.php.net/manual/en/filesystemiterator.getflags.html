FilesystemIterator::getFlags

(PHP 5 >= 5.3.0)

FilesystemIterator::getFlags — 処理フラグを取得する

説明

public int FilesystemIterator::getFlags ( void )

FilesystemIterator::__construct() あるいは FilesystemIterator::setFlags() で設定した処理フラグを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

設定されているフラグの値を返します。

  • FilesystemIterator::__construct() - 新しい filesystem イテレータを作成する
  • FilesystemIterator::setFlags() - 処理フラグを設定する
