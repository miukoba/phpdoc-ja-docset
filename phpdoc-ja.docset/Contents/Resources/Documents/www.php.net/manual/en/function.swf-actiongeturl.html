swf_actiongeturl

(PHP 4)

swf_actiongeturl — Shockwave Flash ムービーから URL を得る

説明

void swf_actiongeturl ( string $url , string $target )

指定したターゲット target からパラメータ url で指定したURLを得ます。

パラメータ

url

URL を表す文字列。

target

対象を表す文字列。

返り値

値を返しません。
