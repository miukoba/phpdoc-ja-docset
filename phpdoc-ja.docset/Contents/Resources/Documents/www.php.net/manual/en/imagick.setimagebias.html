Imagick::setImageBias

(PECL imagick 2.0.0)

Imagick::setImageBias — 画像を折りたたむ任意のメソッドについて画像のバイアスを設定する

説明

bool Imagick::setImageBias ( float $bias )

画像を折りたたむ任意のメソッド (Imagick::ConvolveImage() など) について画像のバイアスを設定します。

パラメータ

bias

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
