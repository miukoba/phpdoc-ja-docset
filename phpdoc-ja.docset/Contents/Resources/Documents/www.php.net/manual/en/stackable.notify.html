Stackable::notify

(PECL pthreads >= 0.36)

Stackable::notify — 同期処理

説明

final public boolean Stackable::notify ( void )

通知を待っている Stackable に通知を送ります。

パラメータ

この関数にはパラメータはありません。

返り値

状態を表す boolean 値を返します。
