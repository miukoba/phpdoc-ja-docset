EvEmbed::sweep

(PECL ev >= 0.2.0)

EvEmbed::sweep — 埋め込んだループに対して単一の非ブロックスイープを行う

説明

public void EvEmbed::sweep ( void )

埋め込んだループに対して単一の非ブロックスイープを行います。 次のコードとほぼ同じ動きですが、埋め込んだループ用に最適な方法をとります。

<?php
$other->start(Ev::RUN_NOWAIT);
?>

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
