hw_Document_Content

(PHP 4)

hw_Document_Content — hw_document の内容を返す

説明

string hw_document_content ( int $hw_document )

ドキュメントの内容を取得します。

パラメータ

hw_document

ドキュメント ID。

返り値

ドキュメントの内容を返します。ドキュメントが HTML ドキュメントの場合、 内容は、全て BODY タグの後になります。HEAD および BODY タグからの情報は、 オブジェクトレコードに保存されます。
