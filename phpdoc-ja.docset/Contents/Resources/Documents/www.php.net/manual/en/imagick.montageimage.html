Imagick::montageImage

(PECL imagick 2.0.0)

Imagick::montageImage — 合成画像を作成する

説明

Imagick Imagick::montageImage ( ImagickDraw $draw , string $tile_geometry , string $thumbnail_geometry , int $mode , string $frame )

複数の画像を合成して画像を作成します。 合成画像上で、画像の名前でタイル状に並べられます。 またオプションで各タイルの直下に画像を置くこともできます。

パラメータ

draw

フォント名、サイズ、色をこのオブジェクトから取得します。

tile_geometry

行単位、ページ単位のタイル数 (例 6x4+0+0)。

thumbnail_geometry

各サムネイルの画像サイズと枠線のサイズ (例 120x120+4+3>)。

mode

サムネイルのフレームモード。 モンタージュモード定数 を参照ください。

frame

画像を枠線で囲みます (例 15x15+3+3)。 枠の色は、サムネイルのマット色となります。

返り値

成功した場合に TRUE を返します。
