kadm5_flush

(PECL kadm5 >= 0.2.3)

kadm5_flush — Kerberos データベースに対するすべての変更を取り消す

説明

bool kadm5_flush ( resource $handle )

Kerberos データベースに対するすべての変更を取り消し、 Kerberos 管理サーバーとの接続はオープンしたままにします。

パラメータ

handle

KADM5 ハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
