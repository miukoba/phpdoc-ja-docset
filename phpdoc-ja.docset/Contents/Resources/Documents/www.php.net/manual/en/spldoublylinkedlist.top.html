SplDoublyLinkedList::top

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::top — 双方向リンクリストの最後のノードを取得する

説明

public mixed SplDoublyLinkedList::top ( void )

パラメータ

この関数にはパラメータはありません。

返り値

最後のノードの値を返します。

エラー / 例外

データが空のときは RuntimeException をスローします。
