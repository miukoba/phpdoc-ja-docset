Imagick::setImageUnits

(PECL imagick 2.0.0)

Imagick::setImageUnits — 画像の解像度の単位を設定する

説明

bool Imagick::setImageUnits ( int $units )

画像の解像度の単位を設定します。

パラメータ

units

返り値

成功した場合に TRUE を返します。
