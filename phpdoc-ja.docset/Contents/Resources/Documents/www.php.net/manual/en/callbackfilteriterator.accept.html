CallbackFilterIterator::accept

(PHP 5 >= 5.4.0)

CallbackFilterIterator::accept — 現在の値、現在のキー、そして内部イテレータを引数としてでコールバックを呼び出す

説明

public string CallbackFilterIterator::accept ( void )

このメソッドは、現在の値とキーそして内部イテレータを使ってコールバックをコールします。

コールバックは、現在のアイテムを受け付ける場合に TRUE、 拒否する場合に FALSE を返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のアイテムを受け付ける場合に TRUE、それ以外の場合に FALSE を返します。
