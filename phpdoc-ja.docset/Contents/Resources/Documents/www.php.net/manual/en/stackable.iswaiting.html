Stackable::isWaiting

(PECL pthreads >= 0.36)

Stackable::isWaiting — 状態を検出する

説明

final public boolean Stackable::isWaiting ( void )

参照先の Stackable が通知を待っているかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

状態を表す boolean 値を返します。

例

例1 Stackable の状態の検出

<?php
class Work extends Stackable {
    public function run() {
        $this->synchronized(function($object){
            $object->wait();
        }, $this);
    }
}

class My extends Worker {
    public function run() {
        /** ... **/
    }
}

$my = new My();
$work = new Work();
$my->stack($work);
$my->start();
usleep(10000);

$work->synchronized(function($object){
    var_dump($object->isWaiting());
    $object->notify();
}, $work);
?>

上の例の出力は以下となります。

bool(true)
