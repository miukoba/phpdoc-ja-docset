SplPriorityQueue::insert

(PHP 5 >= 5.3.0)

SplPriorityQueue::insert — キューに要素を挿入する

説明

public void SplPriorityQueue::insert ( mixed $value , mixed $priority )

value を優先順位 priority でキューに挿入します。

パラメータ

value

挿入する値。

priority

関連づける優先順位。

返り値

値を返しません。
