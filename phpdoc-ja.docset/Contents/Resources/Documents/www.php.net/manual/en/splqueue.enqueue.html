SplQueue::enqueue

(PHP 5 >= 5.3.0)

SplQueue::enqueue — 要素をキューに追加する

説明

void SplQueue::enqueue ( mixed $value )

value をキューの末尾に追加します。

注意:

SplQueue::enqueue() は SplDoublyLinkedList::push() のエイリアスです。

パラメータ

value

キューに追加する値。

返り値

値を返しません。
