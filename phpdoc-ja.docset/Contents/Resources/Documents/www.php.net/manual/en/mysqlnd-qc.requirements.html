要件

PHP 5.3.3 またはそれ以降の PHP。

PECL/mysqlnd_qc は mysqlnd プラグインで、mysqlnd ライブラリに組み込んで使います。 このプラグインを PHP の MySQL 拡張モジュール (mysqli, mysql, PDO_MYSQL) で使うには、その拡張モジュールが mysqlnd ライブラリを使えるようにしなければなりません。

APC ストレージ・ハンドラを PECL/mysqlnd_qc 1.0 で使うには、 APC 3.1.3p1-beta またはそれ以降が必要です。 PECL/mysqlnd_qc 1.2 は、APC 3.1.13-beta でテストしています。 APC ストレージ・ハンドラを共有ライブラリとしてビルドすることはできません。 PECL/mysqlnd_qc のストレージ・ハンドラとして APC を使う場合は、PHP の設定ファイルの extension で APC や PECL/mysqlnd_qc を読み込むことはできません。 APC ストレージ・ハンドラを使うには、APC と PECL/mysqlnd_qc を静的に組み込んだ状態で PHP をコンパイルする必要があります。

MEMCACHE ストレージ・ハンドラを使うには、 libmemcache 0.38 またはそれ以降を使います。 PECL/mysqlnd_qc 1.2 は、libmemcache 1.4.0 でテストしています。

sqlite ストレージ・ハンドラを使うには、 PHP にバンドルされた sqlite3 モジュールを使います。
