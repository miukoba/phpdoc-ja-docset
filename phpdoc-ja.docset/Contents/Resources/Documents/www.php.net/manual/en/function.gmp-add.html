gmp_add

(PHP 4 >= 4.0.4, PHP 5)

gmp_add — 数値を加算する

説明

resource gmp_add ( resource $a , resource $b )

2 つの数を加算します。

パラメータ

a

足される数。

GMP 数リソース、 あるいは数値に変換可能な数値形式の文字列。

b

足す数。

GMP 数リソース、 あるいは数値に変換可能な数値形式の文字列。

返り値

加算の結果を GMP 数で返します。

例

例1 gmp_add() の例

<?php
$sum = gmp_add("123456789012345", "76543210987655");
echo gmp_strval($sum) . "\n";
?>

上の例の出力は以下となります。

200000000000000
