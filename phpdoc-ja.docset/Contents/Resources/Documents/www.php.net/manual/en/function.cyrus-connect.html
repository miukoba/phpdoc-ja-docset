cyrus_connect

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_connect — Cyrus IMAP サーバーに接続する

説明

resource cyrus_connect ([ string $host [, string $port [, int $flags ]]] )

Cyrus IMAP サーバーに接続します。

パラメータ

host

Cyrus IMAP ホスト名。

port

ポート番号。

flags

返り値

成功した場合に接続ハンドラ、失敗した場合に FALSE を返します。
