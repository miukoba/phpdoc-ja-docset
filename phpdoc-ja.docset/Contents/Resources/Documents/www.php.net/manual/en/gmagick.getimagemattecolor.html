Gmagick::getimagemattecolor

(PECL gmagick >= Unknown)

Gmagick::getimagemattecolor — 画像のマット色を返す

説明

public void Gmagick::getimagemattecolor ( void )

成功した場合に GmagickPixel オブジェクトを返します。エラー時に GmagickException をスローします。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に GmagickPixel オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
