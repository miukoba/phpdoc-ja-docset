LuaClosure クラス

(PECL lua >=0.9.0)

導入

LuaClosure は、Lua 関数呼び出しから返されることがありえる LUA_TFUNCTION に対するラッパー・クラスです。

クラス概要

LuaClosure {
/* メソッド */
public void __invoke ( mixed $arg [, mixed $... ] )
}
