ovrimos_free_result

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_free_result — 指定した result_id を開放する

説明

bool ovrimos_free_result ( int $result_id )

指定した結果 ID を開放します。

パラメータ

result_id

ovrimos_execute() あるいは ovrimos_exec() が返す結果 ID。

返り値

TRUE を返します。
