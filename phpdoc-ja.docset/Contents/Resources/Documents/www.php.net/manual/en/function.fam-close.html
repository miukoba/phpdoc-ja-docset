fam_close

(PHP 5 <= 5.0.5)

fam_close — FAM 接続を閉じる

説明

void fam_close ( resource $fam )

FAM サービスへの接続をクローズします。

パラメータ

fam

fam_open() が返す、 FAM サービスへの接続を表すリソース。

返り値

値を返しません。
