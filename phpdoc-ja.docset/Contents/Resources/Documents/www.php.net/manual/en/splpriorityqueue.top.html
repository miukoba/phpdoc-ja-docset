SplPriorityQueue::top

(PHP 5 >= 5.3.0)

SplPriorityQueue::top — キューの先頭のノードを取り出す

説明

public mixed SplPriorityQueue::top ( void )

パラメータ

この関数にはパラメータはありません。

返り値

先頭のノードの値あるいは優先順位 (あるいはその両方) を、抽出フラグの設定に応じて返します。
