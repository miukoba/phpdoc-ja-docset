ReflectionMethod::isPublic

(PHP 5)

ReflectionMethod::isPublic — public メソッドであるかどうかを調べる

説明

public bool ReflectionMethod::isPublic ( void )

public メソッドであるかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

public メソッドである場合に TRUE、それ以外の場合に FALSE を返します。
