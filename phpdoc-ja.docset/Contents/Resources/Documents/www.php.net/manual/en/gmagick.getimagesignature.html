Gmagick::getimagesignature

(PECL gmagick >= Unknown)

Gmagick::getimagesignature — SHA-256 メッセージダイジェストを生成する

説明

public string Gmagick::getimagesignature ( void )

画像ピクセルストリームの SHA-256 メッセージダイジェストを生成します。

パラメータ

この関数にはパラメータはありません。

返り値

そのファイルの SHA-256 ハッシュを文字列で返します。

エラー / 例外

エラー時に GmagickException をスローします。
