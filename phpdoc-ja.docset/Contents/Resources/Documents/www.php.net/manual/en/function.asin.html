asin

(PHP 4, PHP 5)

asin — 逆正弦（アークサイン）

説明

float asin ( float $arg )

arg のアークサインをラジアンで返します。 asin() は sin() の逆関数で、 asin() がとりうる範囲内のすべての a について a==sin(asin(a)) が成立します。

パラメータ

arg

処理する角度。

返り値

arg のアークサインをラジアンで返します。

参考

  • sin() - 正弦（サイン）
  • asinh() - 逆双曲線正弦（アークハイパボリックサイン）
  • acos() - 逆余弦（アークコサイン）
  • atan() - 逆正接（アークタンジェント）
