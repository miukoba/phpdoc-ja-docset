Imagick::getImageFormat

(PECL imagick 2.0.0)

Imagick::getImageFormat — シーケンス内の特定の画像のフォーマットを返す

説明

string Imagick::getImageFormat ( void )

シーケンス内の特定の画像のフォーマットを返します。

返り値

成功した場合は画像のフォーマットを文字列で返します。

エラー / 例外

エラー時に ImagickException をスローします。
