CairoToyFontFace クラス

(PECL cairo >= 0.1.0)

導入

CairoToyFontFace クラスを CairoContext::selectFontFace() の代わりに使うと、コンテキストに依存しないトイフォントを作れます。

クラス概要

CairoToyFontFace extends CairoFontFace {
/* 継承したメソッド */
}
