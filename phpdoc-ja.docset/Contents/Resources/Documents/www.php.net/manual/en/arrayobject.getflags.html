ArrayObject::getFlags

(PHP 5 >= 5.1.0)

ArrayObject::getFlags — 振る舞いのフラグを取得する

説明

public int ArrayObject::getFlags ( void )

ArrayObject の振る舞いのフラグを取得します。 利用可能なフラグの一覧は、 ArrayObject::setFlags メソッドを参照ください。

パラメータ

この関数にはパラメータはありません。

返り値

ArrayObject の振る舞いのフラグを返します。

例

例1 ArrayObject::getFlags() の例

<?php
// フルーツの配列
$fruits = array("lemons" => 1, "oranges" => 4, "bananas" => 5, "apples" => 10);

$fruitsArrayObject = new ArrayObject($fruits);

// 現在のフラグを取得します
$flags = $fruitsArrayObject->getFlags();
var_dump($flags);

// 新しいフラグを設定します
$fruitsArrayObject->setFlags(ArrayObject::ARRAY_AS_PROPS);

// 新しいフラグを取得します
$flags = $fruitsArrayObject->getFlags();
var_dump($flags);
?>

上の例の出力は以下となります。

int(0)
int(2)
