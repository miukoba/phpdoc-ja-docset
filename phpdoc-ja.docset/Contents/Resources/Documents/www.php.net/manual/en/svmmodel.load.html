SVMModel::load

(PECL svm >= 0.1.00.1.0)

SVMModel::load — 保管された SVM モデルを読み込み

説明

public bool SVMModel::load ( string $filename )

識別および回帰の準備ができたモデルファイルを読み込みます。

パラメータ

filename

モデルのファイル名

返り値

エラー時に SVMException をスローします。 成功時に true を返します。

参考

  • SVMModel::save() - モデルをファイルに保管
