EvFork::createStopped

(PECL ev >= 0.2.0)

EvFork::createStopped — 停止した EvFork ウォッチャーのインスタンスを作る

説明

final public static object EvFork::createStopped ( string $callback [, string $data [, string $priority ]] )

EvFork::__construct() と同じですが、ウォッチャーを自動的には開始しません。

パラメータ

callback

ウォッチャーのコールバック を参照ください。

data

このウォッチャーに関連づけるカスタムデータ。

priority

ウォッチャーの優先度。

返り値

成功した場合に、停止した EvFork オブジェクトを返します。
