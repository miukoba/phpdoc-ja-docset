sybase_min_server_severity

(PHP 4, PHP 5)

sybase_min_server_severity — サーバーの severity の最小値を設定する

説明

void sybase_min_server_severity ( int $severity )

sybase_min_server_severity() はサーバーの severity レベルの最小値を設定します。

パラメータ

severity

返り値

値を返しません。

注意

注意: この関数は、Sybase の CT ライブラリインターフェイスでのみ使用可能で、DB ライブラリでは使用できません。
