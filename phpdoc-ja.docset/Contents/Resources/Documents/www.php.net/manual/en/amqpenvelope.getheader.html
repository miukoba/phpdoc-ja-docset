AMQPEnvelope::getHeader

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

AMQPEnvelope::getHeader — メッセージのヘッダを取得する

説明

public string AMQPEnvelope::getHeader ( string $header_key )

指定したキーのヘッダの値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

指定したヘッダの内容を返します。設定されていない場合は FALSE を返します。
