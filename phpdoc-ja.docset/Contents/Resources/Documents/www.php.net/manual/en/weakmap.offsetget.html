WeakMap::offsetGet

(PECL weakref >= 0.2.0)

WeakMap::offsetGet — オブジェクトを指定して、それが指す値を返す

説明

public mixed WeakMap::offsetGet ( object $object )

オブジェクトを指定して、それが指す値を返します。

パラメータ

object

マップ内のキーとして含まれるオブジェクト。

返り値

引数で渡したオブジェクトに関連づけられた値を返します。 値がない場合は NULL を返します。
