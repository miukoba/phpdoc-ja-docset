CURLFile::getPostFilename

(PHP 5 >= 5.5.0)

CURLFile::getPostFilename — POST 用のファイル名を取得する

説明

public string CURLFile::getPostFilename ( void )

パラメータ

この関数にはパラメータはありません。

返り値

POST 用のファイル名を返します。
