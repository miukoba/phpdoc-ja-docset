Imagick::setResourceLimit

(PECL imagick 2.0.0)

Imagick::setResourceLimit — 特定のリソースの制限をメガバイト単位で設定する

説明

bool Imagick::setResourceLimit ( int $type , int $limit )

特定のリソースの制限をメガバイト単位で設定します。

パラメータ

type

limit

返り値

成功した場合に TRUE を返します。
