SolrInputDocument::getBoost

(PECL solr >= 0.9.2)

SolrInputDocument::getBoost — 文書に対する現行の boost 値を取得

説明

public float SolrInputDocument::getBoost ( void )

文書に対する現行の boost 値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合は boost 値、失敗した場合は FALSE を返します。
