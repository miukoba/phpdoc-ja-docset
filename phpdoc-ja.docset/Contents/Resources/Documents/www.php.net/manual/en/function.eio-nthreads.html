eio_nthreads

(PECL eio >= 0.0.1dev)

eio_nthreads — 現在利用中のスレッド数を返す

説明

int eio_nthreads ( void )

パラメータ

この関数にはパラメータはありません。

返り値

eio_nthreads() は、現在利用中のスレッド数を返します。
