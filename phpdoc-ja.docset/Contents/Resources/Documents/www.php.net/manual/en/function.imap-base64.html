imap_base64

(PHP 4, PHP 5)

imap_base64 — BASE64 でエンコードされたテキストをデコードする

説明

string imap_base64 ( string $text )

BASE64 でエンコードされたテキスト text をデコードします。

パラメータ

text

エンコードされたテキスト。

返り値

デコードしたメッセージを文字列で返します。
