fann_duplicate_train_data

(PECL fann >= 1.0.0)

fann_duplicate_train_data — Returns an exact copy of a fann train data

説明

resource fann_duplicate_train_data ( resource $data )

Returns an exact copy of a fann train data resource.

パラメータ

data

ニューラルネットワークトレーニングリソース。

返り値

成功した場合にニューラルネットワークトレーニングリソース、エラー時に FALSE を返します。
