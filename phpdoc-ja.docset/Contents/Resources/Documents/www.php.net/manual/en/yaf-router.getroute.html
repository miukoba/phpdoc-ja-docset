Yaf_Router::getRoute

(Yaf >=1.0.0)

Yaf_Router::getRoute — 名前を指定してルートを取得する

説明

public Yaf_Route_Interface Yaf_Router::getRoute ( string $name )

名前を指定してルートを取得します。 Yaf_Router::getCurrentRoute() も参照ください。

パラメータ

この関数にはパラメータはありません。
