http_response_code

(PHP 5 >= 5.4.0)

http_response_code — HTTP レスポンスコードを取得または設定

説明

int http_response_code ([ int $response_code ] )

パラメータを渡さない場合は、http_response_code は現在のレスポンスコードを取得します。 パラメータを渡す場合は、レスポンスコードを設定します。

パラメータ

response_code

オプションの response_code は、レスポンスコードを設定します。

<?php
http_response_code(404);
?>

返り値

現在のレスポンスコード。デフォルトでは、返り値は int(200) です。

例

例1 http_response_code() の使用例

<?php

// 現在のデフォルトのレスポンスコードを取得
var_dump(http_response_code()); // int(200)

// レスポンスコードを設定
http_response_code(404);

// 新しいレスポンスコードを取得
var_dump(http_response_code()); // int(404)
?>
