AMQPConnection::connect

(PECL amqp >= Unknown)

AMQPConnection::connect — AMQP ブローカとの接続を確立する

説明

public bool AMQPConnection::connect ( void )

このメソッドは、AMQP ブローカとの接続を確立します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

例

例1 AMQPConnection::connect() の例

<?php

/* 新しい接続を作成します */
$cnn = new AMQPConnection();

// ログインの詳細を設定します
$cnn->setLogin('mylogin');
$cnn->setPassword('mypass');

if ($cnn->connect()) {
    echo "ブローカへの接続を確立しました";
}
else {
    echo "ブローカに接続できませんでした";
}


?>
