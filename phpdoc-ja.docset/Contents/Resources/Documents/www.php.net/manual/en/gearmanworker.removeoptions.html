GearmanWorker::removeOptions

(PECL gearman >= 0.6.0)

GearmanWorker::removeOptions — Remove worker options

説明

public bool GearmanWorker::removeOptions ( int $option )

Removes (unsets) one or more worker options.

パラメータ

option

The options to be removed (unset)

返り値

Always returns TRUE.
