Yaf_Application::environ

(Yaf >=1.0.0)

Yaf_Application::environ — 環境を取得する

説明

public void Yaf_Application::environ ( void )

環境を取得します。これは yaf.environ で定義されており、デフォルト値は "product" です。

パラメータ

この関数にはパラメータはありません。

返り値

例

例1 Yaf_Application::environ() の例

<?php
$config = array(
    "application" => array(
        "directory" => realpath(dirname(__FILE__)) . "/application",
    ),
);

/** Yaf_Application */
$application = new Yaf_Application($config);
print_r($application->environ());
?>

上の例の出力は、 たとえば以下のようになります。

product
