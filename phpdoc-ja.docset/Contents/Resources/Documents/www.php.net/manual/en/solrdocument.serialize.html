SolrDocument::serialize

(PECL solr >= 0.9.2)

SolrDocument::serialize — カスタムのシリアライズで使われます

説明

public string SolrDocument::serialize ( void )

カスタムのシリアライズで使われます.

パラメータ

この関数にはパラメータはありません。

返り値

シリアライズされた Solr 文書を表現する文字列を返します。
