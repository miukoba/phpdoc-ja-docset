event_buffer_fd_set

(PECL libevent >= 0.0.1)

event_buffer_fd_set — バッファつきイベントのファイル記述子を変更する

説明

void event_buffer_fd_set ( resource $bevent , resource $fd )

バッファつきイベントが操作するファイル記述子を変更します。

パラメータ

bevent

有効なバッファつきイベントリソース。

fd

有効な PHP ストリーム。ファイル記述子にキャスト可能でなければなりません。
