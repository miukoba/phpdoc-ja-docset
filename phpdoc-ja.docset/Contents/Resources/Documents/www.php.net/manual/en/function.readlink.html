readlink

(PHP 4, PHP 5)

readlink — シンボリックリンク先を返す

説明

string readlink ( string $path )

readlink() は同名の C 関数と同じ動作をします。

パラメータ

path

シンボリックリンクのパス。

変更履歴

バージョン 説明
5.3.0 この関数は、Windows プラットフォーム (Vista 以降、あるいは Server 2008 以降) でも動作するようになりました。

返り値

シンボリックリンク・パスの内容を返します。エラーの場合は FALSE を返します。

例

例1 readlink() の例

<?php

// 出力例 /boot/vmlinux-2.4.20-xfs
echo readlink('/vmlinuz');

?>
