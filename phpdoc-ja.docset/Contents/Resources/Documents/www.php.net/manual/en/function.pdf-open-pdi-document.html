PDF_open_pdi_document

(PECL pdflib >= 2.1.0)

PDF_open_pdi_document — pdi ドキュメントを準備する

説明

int PDF_open_pdi_document ( resource $p , string $filename , string $optlist )

ディスク上の PDF ドキュメントあるいは仮想 PDF ドキュメントをオープンし、 あとで使えるように準備します。
