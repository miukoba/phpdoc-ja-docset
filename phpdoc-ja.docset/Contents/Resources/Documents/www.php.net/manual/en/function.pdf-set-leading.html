PDF_set_leading

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_leading — テキストの行間を設定する [古い関数]

説明

この関数は PDFlib バージョン 3 で廃止されました。かわりに PDF_set_value() で パラメータ leading を使用してください。
