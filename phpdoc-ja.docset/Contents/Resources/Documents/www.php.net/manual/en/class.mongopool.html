MongoPool クラス

(PECL mongo >= 1.2.3)

導入

警告

現在のリリース (1.3.0 以降) のドライバは、プーリングを実装しなくなりました。 このクラスやメソッドは廃止予定であり、使ってはいけません。

クラス概要

MongoPool {
/* メソッド */
public static int getSize ( void )
public array info ( void )
public static bool setSize ( int $size )
}

変更履歴

バージョン 説明
1.3.0 この機能は削除され、単に非推奨の警告を出すだけで何もしないようになりました。 このクラスは過去との互換性のためだけに残されているものであり、近い将来削除されます。
1.2.11 E_DEPRECATED を発行するようになりました。

目次

  • MongoPool::getSize — 接続プールのサイズを取得する
  • MongoPool::info — すべての接続プールに関する情報を返す
  • MongoPool::setSize — 新しい接続プールのサイズを設定する
