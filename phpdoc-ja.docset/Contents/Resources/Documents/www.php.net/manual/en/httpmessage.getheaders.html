HttpMessage::getHeaders

(PECL pecl_http >= 0.10.0)

HttpMessage::getHeaders — メッセージのヘッダを取得する

説明

public array HttpMessage::getHeaders ( void )

メッセージのヘッダを取得します。

返り値

メッセージの HTTP ヘッダを含む連想配列を返します。
