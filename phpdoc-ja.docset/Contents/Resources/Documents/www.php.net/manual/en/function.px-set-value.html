px_set_value

(PECL paradox >= 1.1.0)

px_set_value — 値を設定する

説明

bool px_set_value ( resource $pxdoc , string $name , float $value )

さまざまな値を設定します。

パラメータ

pxdoc

px_new() が返す、 paradox データベースのリソース ID。

name

name は以下のいずれかです。

numprimkeys

プライマリキーの数。paradox データベースは、 常に最初の numprimkeys フィールドを プライマリキーとして扱います。

value

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
