ReflectionFunctionAbstract::isGenerator

(PHP 5 >= 5.5.0)

ReflectionFunctionAbstract::isGenerator — この関数がジェネレータかどうかを返す

説明

public bool ReflectionFunctionAbstract::isGenerator ( void )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

この関数がジェネレータである場合に TRUE、そうでない場合に FALSE を返します。 処理に失敗した場合に NULL を返します。
