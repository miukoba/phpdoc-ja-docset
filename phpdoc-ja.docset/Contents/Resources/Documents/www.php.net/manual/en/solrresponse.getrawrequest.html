SolrResponse::getRawRequest

(PECL solr >= 0.9.2)

SolrResponse::getRawRequest — Solr サーバーに送られた生のリクエストを返す

説明

public string SolrResponse::getRawRequest ( void )

Solr サーバーに送られた生のリクエストを返します。

パラメータ

この関数にはパラメータはありません。

返り値

Solr サーバーに送られた生のリクエストを返します。
