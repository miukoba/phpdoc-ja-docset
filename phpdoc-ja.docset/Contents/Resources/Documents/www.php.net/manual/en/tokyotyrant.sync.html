TokyoTyrant::sync

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrant::sync — データベースを同期させる

説明

public mixed TokyoTyrant::sync ( void )

データベースを物理デバイスに同期させます。

パラメータ

この関数にはパラメータはありません。

返り値

現在のオブジェクトを返します。失敗した場合は TokyoTyrantException をスローします。
