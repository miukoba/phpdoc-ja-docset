SolrDocument::offsetExists

(PECL solr >= 0.9.2)

SolrDocument::offsetExists — 特定のフィールドが存在するかどうかチェック

説明

public bool SolrDocument::offsetExists ( string $fieldName )

特定のフィールドが存在するかどうかチェックします。これは、オブジェクトが配列として扱われる際に使用されます。

パラメータ

fieldName

フィールド名

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
