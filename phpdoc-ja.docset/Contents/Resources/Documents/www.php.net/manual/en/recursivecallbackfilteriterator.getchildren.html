RecursiveCallbackFilterIterator::getChildren

(PHP 5 >= 5.4.0)

RecursiveCallbackFilterIterator::getChildren — RecursiveCallbackFilterIterator に含まれる内部イテレータの子を返す

説明

public RecursiveCallbackFilterIterator RecursiveCallbackFilterIterator::getChildren ( void )

内部イテレータのフィルタリングした子を取得します。

RecursiveCallbackFilterIterator::hasChildren() を使って、取得できる子があるかどうかを調べなければなりません。

パラメータ

この関数にはパラメータはありません。

返り値

子を含む RecursiveCallbackFilterIterator を返します。
