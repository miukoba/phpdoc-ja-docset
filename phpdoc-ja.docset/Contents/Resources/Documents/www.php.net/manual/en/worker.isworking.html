Worker::isWorking

(PECL pthreads >= 0.37)

Worker::isWorking — 状態を検出する

説明

final public boolean Worker::isWorking ( void )

ワーカーが Stackable を実行中かどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

状態を表す boolean 値を返します。

例

例1 ワーカーの状態を検出する

<?php
class My extends Worker {
    public function run() {
        /* ... */
    }
}
$my = new My();
$my->start();
/* ... */
if ($my->isWorking()) {
    /* ... ワーカーは、他のオブジェクトを実行するのに忙しい状態 */
}
?>
