PDF_get_errmsg

(PECL pdflib >= 2.0.0)

PDF_get_errmsg — エラーテキストを取得する

説明

string PDF_get_errmsg ( resource $pdfdoc )

直近で発生した例外のテキストや、関数コールが失敗した理由の テキストを取得します。
