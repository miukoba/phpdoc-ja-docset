imagecreatefromwebp

(PHP 5 >= 5.5.0)

imagecreatefromwebp — 新しい画像をファイルあるいは URL から作成する

説明

resource imagecreatefromwebp ( string $filename )

imagecreatefromwebp() は、指定したファイル名から得られる画像を表す ID を返します。

ヒント

fopen wrappers が有効の場合、この関数のファイル名として URL を使用することができます。ファイル名の指定方法に関する詳細は fopen() を参照ください。 サポートするプロトコル/ラッパー には、さまざまなラッパーの機能やその使用法、 提供される定義済み変数などの情報がまとめられています。

パラメータ

filename

WebP 画像へのパス。

返り値

成功した場合に画像リソース ID、エラー時に FALSE を返します。

例

例1 imagecreatefromwebp() を使った、WebP から jpeg への変換

<?php
// WebP ファイルを読み込みます
$im = imagecreatefromwebp('./example.webp');

// 100% のクオリティで jpeg ファイルに変換します
imagejpeg($im, './example.jpeg', 100);
imagedestroy($im);
?>
