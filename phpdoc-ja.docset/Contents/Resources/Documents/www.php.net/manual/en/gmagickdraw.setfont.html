GmagickDraw::setfont

(PECL gmagick >= Unknown)

GmagickDraw::setfont — テキストのアノテーションで使うフォントの完全な名前を設定する

説明

public GmagickDraw GmagickDraw::setfont ( string $font )

テキストのアノテーションで使うフォントの完全な名前を設定します。

パラメータ

font

フォント名。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
