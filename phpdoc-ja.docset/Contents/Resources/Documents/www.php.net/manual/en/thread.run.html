Thread::run

(PECL pthreads >= 0.34)

Thread::run — 実行する

説明

abstract public void Thread::run ( void )

Thread::start を呼んだときに、スレッドがこのメソッドを実行します。

パラメータ

この関数にはパラメータはありません。

返り値

このメソッドの返り値は無視されます。
