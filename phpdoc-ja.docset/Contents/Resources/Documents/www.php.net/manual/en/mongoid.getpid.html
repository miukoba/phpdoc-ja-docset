MongoId::getPID

(PECL mongo >= 1.0.11)

MongoId::getPID — プロセス ID を取得する

説明

public int MongoId::getPID ( void )

Mongo ID から pid を取り出します。

パラメータ

この関数にはパラメータはありません。

返り値

MongoId の PID を返します。
