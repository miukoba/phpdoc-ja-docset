Thread::lock

(PECL pthreads >= 0.40)

Thread::lock — 同期処理

説明

final public boolean Thread::lock ( void )

参照するオブジェクトストレージを、呼び出し元のコンテキストでロックします。

パラメータ

この関数にはパラメータはありません。

返り値

成功したかどうかを表す boolean 値を返します。

例

例1 スレッドのロック

<?php
class My extends Thread {
    public function run() {
        var_dump($this->lock());
        /** 誰も読み書きできません **/
        var_dump($this->unlock());
        /** 他のコンテキストからも読み書きできるようになりました */
    }
}
$my = new My();
$my->start();
?>

上の例の出力は以下となります。

bool(true)
bool(true)
