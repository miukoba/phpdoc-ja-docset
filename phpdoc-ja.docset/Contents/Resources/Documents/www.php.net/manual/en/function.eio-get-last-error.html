eio_get_last_error

(PECL eio >= 1.0.0)

eio_get_last_error — リクエストリソースに関連づけられた直近のエラーの説明を文字列で返す

説明

string eio_get_last_error ( resource $req )

eio_get_last_error() は、 req に関連づけられた直近のエラーの説明を文字列で返します。

パラメータ

req

リクエストリソース。

返り値

eio_get_last_error() は、req に関連づけられた直近のエラーの説明を文字列で返します。
