HttpRequest::setBody

(PECL pecl_http >= 1.5.0)

HttpRequest::setBody — 送信する本文を設定し、既に設定されている本文を上書きする

説明

bool HttpRequest::setBody ([ string $request_body_data ] )

送信する本文を設定し、既に設定されている本文を上書きします。

パラメータ

request_body_data

既存のリクエスト本文を上書きするリクエスト本文。

警告

Content-Typeをリクエスト本文内で指定しておかなければなりません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

注意

警告

このメソッドは、かつては HttpRequest::setRawPostData() という名前でした。
