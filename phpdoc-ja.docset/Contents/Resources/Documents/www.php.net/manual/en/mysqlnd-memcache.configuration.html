実行時設定

php.ini の設定により動作が変化します。

Mysqlnd_memcache 設定オプション
名前 デフォルト 変更可能 変更履歴
mysqlnd_memcache.enable 1 PHP_INI_SYSTEM 1.0.0 以降で利用可能

以下に設定ディレクティブに関する 簡単な説明を示します。

mysqlnd_memcache.enable integer

プラグインを有効あるいは無効にします。無効にした場合は、この拡張モジュールを mysqlnd に組み込んで mysqlnd C API の呼び出しのプロキシとすることがなくなります。

注意:

このオプションを使う主な場面は、開発者がこの拡張モジュールを静的に PHP へ組み込む場合です。 一般のユーザーはこの拡張モジュールを共有オブジェクトとしてビルドすることをおすすめします。 そして、不要になれば完全にアンロードします。
