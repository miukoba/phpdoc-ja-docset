Imagick::implodeImage

(PECL imagick 2.0.0)

Imagick::implodeImage — 新しい画像をコピーとして作成する

説明

bool Imagick::implodeImage ( float $radius )

既存の画像をコピーして新しい画像を作成します。 既存の画像のピクセルを、指定したパーセンテージで "内破" します。

パラメータ

radius

内破処理の半径。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
