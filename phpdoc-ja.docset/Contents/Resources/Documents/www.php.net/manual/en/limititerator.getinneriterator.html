LimitIterator::getInnerIterator

(PHP 5 >= 5.1.0)

LimitIterator::getInnerIterator — 内部イテレータを取得する

説明

public Iterator LimitIterator::getInnerIterator ( void )

内部イテレータを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

LimitIterator::__construct() に渡された内部イテレータを返します。
