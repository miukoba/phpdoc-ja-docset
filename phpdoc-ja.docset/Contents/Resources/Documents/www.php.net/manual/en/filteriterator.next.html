FilterIterator::next

(PHP 5 >= 5.1.0)

FilterIterator::next — イテレータを前に移動する

説明

public void FilterIterator::next ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

イテレータを前方に移動します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
