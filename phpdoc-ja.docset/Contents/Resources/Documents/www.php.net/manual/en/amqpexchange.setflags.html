AMQPExchange::setFlags

(PECL amqp >= Unknown)

AMQPExchange::setFlags — exchange のフラグを設定する

説明

public void AMQPExchange::setFlags ( int $flags )

exchange 上のフラグを設定します。

パラメータ

flags

フラグのビットマスク。現在サポートしているフラグは AMQP_DURABLE、 AMQP_PASSIVE です。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
