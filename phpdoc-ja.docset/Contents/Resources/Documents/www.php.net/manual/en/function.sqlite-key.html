sqlite_key

SQLiteResult::key

(PHP 5 >= 5.1.0 and < 5.4.0)

sqlite_key -- SQLiteResult::key — カレントレコードのインデックスを返す

説明

オブジェクト指向型 (メソッド):

int SQLiteResult::key ( void )

SQLiteResult::key() は、バッファされた結果セット result のカレントレコードのインデックスを返します。

他の SQLite 関数とは異なり、この関数には手続き型のバージョンはありません。 SQLiteResult オブジェクトのメソッドとしてのみ呼ぶことができます。

返り値

バッファされた結果セット result のカレントレコードのインデックスを返します。

参考

  • sqlite_next() - 次のレコード番号へシークする
  • sqlite_current() - 結果セットからカレントのレコードを配列として取得する
  • sqlite_rewind() - 先頭レコード番号へシークする
