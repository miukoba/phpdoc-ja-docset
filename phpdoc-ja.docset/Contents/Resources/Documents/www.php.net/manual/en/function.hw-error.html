hw_Error

(PHP 4)

hw_Error — エラー番号

説明

int hw_error ( int $connection )

直近のコマンドのエラー番号を返します。

パラメータ

connection

接続 ID。

返り値

直近のエラー番号を返します。 返り値が 0 の場合、エラーは発生していません。
