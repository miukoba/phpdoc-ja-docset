Gmagick::getimagedispose

(PECL gmagick >= Unknown)

Gmagick::getimagedispose — 画像の配置方法を取得する

説明

public int Gmagick::getimagedispose ( void )

画像の配置方法を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

画像の配置方法を返します。

エラー / 例外

エラー時に GmagickException をスローします。
