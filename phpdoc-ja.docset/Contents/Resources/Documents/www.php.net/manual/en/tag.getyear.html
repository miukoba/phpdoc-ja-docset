KTaglib_Tag::getYear

(0.0.1)

KTaglib_Tag::getYear — ID3 タグから年を返す

説明

public int KTaglib_Tag::getYear ( void )

ID3 タグの年を返します。 このメソッドは、ID3v1 および ID3v2 タグで実装されています。

返り値

年を整数値で返します。
