Mongo::selectDB — データベースを取得する

説明

public MongoDB Mongo::selectDB ( string $name )

パラメータ

name

データベース名。

返り値

新しいデータベースオブジェクトを返します。

エラー / 例外

データベース名が不正な場合に Exception をスローします。
