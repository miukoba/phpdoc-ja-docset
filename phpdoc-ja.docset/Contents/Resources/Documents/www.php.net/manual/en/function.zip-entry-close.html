zip_entry_close

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PECL zip >= 1.0.0)

zip_entry_close — ディレクトリエントリを閉じる

説明

bool zip_entry_close ( resource $zip_entry )

指定されたディレクトリエントリを閉じます。

パラメータ

zip_entry

zip_entry_open()によりオープンされたディレクトリの エントリ

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
