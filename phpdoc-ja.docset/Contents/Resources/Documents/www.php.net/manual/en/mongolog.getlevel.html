MongoLog::getLevel

(PECL mongo >= 1.2.3)

MongoLog::getLevel — ログレベルを取得する

説明

public static int MongoLog::getLevel ( void )

これを使ってログレベルを調べることができます。 MongoLog のセクションで説明した定数をビット演算子でつないでレベルを調べます。

<?php

if (MongoLog::getLevel() & MongoLog::FINE) {
    echo "大量のログ出力\n";
}

if (MongoLog::getLevel() ^ MongoLog::NONE) {
    echo "少なくとも少しだけはログ出力\n";
}

if (MongoLog::getLevel() == MongoLog::ALL) {
    echo "最高レベルのログ出力\n";
}

?>

パラメータ

この関数にはパラメータはありません。

返り値

現在のレベルを返します。
