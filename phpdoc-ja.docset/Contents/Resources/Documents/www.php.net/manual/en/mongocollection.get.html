MongoCollection::__get

(PECL mongo >=1.0.2)

MongoCollection::__get — コレクションを取得する

説明

public MongoCollection MongoCollection::__get ( string $name )

ドット区切りの名前でコレクションを取得する簡潔な構文です。 コレクション名に特殊な文字が含まれている場合は MongoDB::selectCollection() を使います。

<?php

$mongo = new MongoClient();

// 次の 2 行はどちらも同じ意味です。
$collection = $mongo->selectDB("foo")->selectCollection("bar.baz");
$collection = $mongo->foo->bar->baz;

?>

パラメータ

name

コレクション名の中の次の文字列。

返り値

コレクションを返します。
