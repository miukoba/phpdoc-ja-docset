mssql_num_rows

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_num_rows — 結果の行数を得る

説明

int mssql_num_rows ( resource $result )

mssql_num_rows() は結果の行数を返します。

パラメータ

result

処理対象となる結果リソース。これは mssql_query() のコールによって取得します。

返り値

行の数を整数値で返します。

例

例1 mssql_num_rows() の例

<?php
// MSSQL に接続し、データベースを選択します
$link = mssql_connect('KALLESPC\SQLEXPRESS', 'sa', 'phpfi');
mssql_select_db('php');

// テーブルからすべてのレコードを取得します
$query = mssql_query('SELECT * FROM [php].[dbo].[persons]');

echo 'Total records in database: ' . mssql_num_rows($query);

// 後始末
mssql_free_result($query);
?>
