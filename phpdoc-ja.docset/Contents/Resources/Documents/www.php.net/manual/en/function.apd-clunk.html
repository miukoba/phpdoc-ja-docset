apd_clunk

(PECL apd 0.2-0.4)

apd_clunk — 警告とコールスタックをスローする

説明

void apd_clunk ( string $warning [, string $delimiter = "<BR />" ] )

perl の Carp::cluck と同じように動作します。警告とコールバックをスローします。 デフォルトの行区切り文字は "<BR />\n" です。

パラメータ

warning

スローする警告。

delimiter

区切り文字。デフォルトは <BR />。

返り値

値を返しません。

例

例1 apd_clunk() の例

<?php
apd_clunk("Some Warning", "<br/>");
?>
