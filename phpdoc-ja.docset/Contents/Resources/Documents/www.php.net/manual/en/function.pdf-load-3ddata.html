PDF_load_3ddata

(PECL pdflib >= 2.1.0)

PDF_load_3ddata — 3D モデルを読み込む

説明

int PDF_load_3ddata ( resource $pdfdoc , string $filename , string $optlist )

3D モデルを、ディスク上あるいは仮想ファイルから読み込みます。

この関数は PDF 1.6 を必要とします。
