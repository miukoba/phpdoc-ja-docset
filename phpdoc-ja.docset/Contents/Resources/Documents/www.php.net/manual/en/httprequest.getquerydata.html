HttpRequest::getQueryData

(PECL pecl_http >= 0.10.0)

HttpRequest::getQueryData — クエリデータを取得する

説明

public string HttpRequest::getQueryData ( void )

現在のクエリデータを、urlencode された形式の文字列で取得します。

返り値

urlencode されたクエリを含む文字列を返します。
