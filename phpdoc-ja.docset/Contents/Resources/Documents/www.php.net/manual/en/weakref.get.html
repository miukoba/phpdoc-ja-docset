Weakref::get

(PECL weakref >= 0.1.0)

Weakref::get — 弱い参照が指す先にオブジェクトを返す

説明

public object Weakref::get ( void )

弱い参照が指す先にオブジェクトを返します。

パラメータ

この関数にはパラメータはありません。

返り値

参照が有効な場合にその先のオブジェクトを返します。それ以外の場合に NULL を返します。

参考

  • Weakref::valid() - 参照先のオブジェクトがまだ存在するかどうかを調べる
