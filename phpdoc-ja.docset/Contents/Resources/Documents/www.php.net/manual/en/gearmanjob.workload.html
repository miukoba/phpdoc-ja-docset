GearmanJob::workload

(PECL gearman >= 0.5.0)

GearmanJob::workload — Get workload

説明

public string GearmanJob::workload ( void )

Returns the workload for the job. This is serialized data that is to be processed by the worker.

パラメータ

この関数にはパラメータはありません。

返り値

Serialized data.
