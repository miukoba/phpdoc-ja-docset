Gmagick::setimageindex

(PECL gmagick >= Unknown)

Gmagick::setimageindex — 画像リスト内でのイテレータの位置を、index パラメータで指定した場所に設定する

説明

public Gmagick Gmagick::setimageindex ( int $index )

画像リスト内でのイテレータの位置を、index パラメータで指定した場所に設定します。

パラメータ

index

シーン番号。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
