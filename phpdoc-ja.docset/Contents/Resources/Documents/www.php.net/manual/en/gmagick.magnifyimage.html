Gmagick::magnifyimage

(PECL gmagick >= Unknown)

Gmagick::magnifyimage — 画像を二倍に比例拡大する

説明

public mixed Gmagick::magnifyimage ( void )

画像を元のサイズの二倍に比例拡大します。

パラメータ

この関数にはパラメータはありません。

返り値

拡大した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
