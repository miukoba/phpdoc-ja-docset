OCI-Lob::rewind

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::rewind — 内部ポインタをラージオブジェクトの先頭に移動する

説明

bool OCI-Lob::rewind ( void )

内部ポインタをラージオブジェクトの先頭にセットします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
