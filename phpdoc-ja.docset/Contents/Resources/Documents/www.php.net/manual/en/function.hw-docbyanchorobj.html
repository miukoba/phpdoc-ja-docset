hw_DocByAnchorObj

(PHP 4)

hw_DocByAnchorObj — アンカーが指すオブジェクトレコード

説明

string hw_docbyanchorobj ( int $connection , int $anchorID )

anchorID が指す ドキュメントのオブジェクトレコードを返します。

パラメータ

connection

接続 ID。

anchorID

アンカー ID。

返り値

オブジェクトレコードを返します。
