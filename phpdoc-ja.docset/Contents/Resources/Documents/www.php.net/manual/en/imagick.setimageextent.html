Imagick::setImageExtent

(PECL imagick 2.0.0)

Imagick::setImageExtent — 画像のサイズを設定する

説明

bool Imagick::setImageExtent ( int $columns , int $rows )

画像のサイズ (つまりカラム数と行数) を設定します。

パラメータ

columns

rows

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
