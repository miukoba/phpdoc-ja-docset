Ev::stop

(PECL ev >= 0.2.0)

Ev::stop — デフォルトのイベントループを止める

説明

final public static void Ev::stop ([ int $how ] )

デフォルトのイベントループを止めます。

パラメータ

how

Ev::BREAK_* 定数 のいずれか。

返り値

値を返しません。

参考

  • Ev::run() - イベントのチェックとデフォルトループのコールバックの呼び出しを開始する
