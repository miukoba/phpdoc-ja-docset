http_build_cookie

(PECL pecl_http >= 1.2.0)

http_build_cookie — クッキー文字列を作成する

説明

string http_build_cookie ( array $cookie )

http_parse_cookie() が返すような形式の配列/オブジェクトから、クッキー文字列を作成します。

パラメータ

cookie

http_parse_cookie() が返すような形式の、クッキーのリスト。

返り値

クッキーを文字列で返します。

参考

  • http_parse_cookie() - HTTP クッキーをパースする

jonathan dot aquino at gmail dot com 21-Mar-2009 07:46
Note that you must cast the result of http_parse_cookie to an array before passing it to http_build_cookie (see http://bugs.php.net/bug.php?id=47740 ).

Example:

<?php
$cookie = http_parse_cookie('RMID=2dab5fc9296749c2f28ec0b7; expires=Saturday, 20-Mar-2010 01:34:06 GMT; path=/; domain=.example.com');
var_dump(http_build_cookie((array)$cookie));
?>
