hw_GetChildDocCollObj

(PHP 4)

hw_GetChildDocCollObj — コレクションの子ドキュメントのオブジェクトレコード

説明

array hw_getchilddoccollobj ( int $connection , int $objectID )

コレクションの子ドキュメントに関するオブジェクトレコードの配列を返します。

パラメータ

connection

接続 ID。

objectID

オブジェクト ID。

返り値

オブジェクトレコードの配列を返します。
