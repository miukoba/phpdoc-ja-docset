HaruDoc::useJPEncodings

(PECL haru >= 0.0.1)

HaruDoc::useJPEncodings — 日本語エンコーディングを有効にする

説明

bool HaruDoc::useJPEncodings ( void )

日本語エンコーディングを有効にします。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
