yp_master

(PHP 4, PHP 5 <= 5.0.5)

yp_master — 指定したマップのマスタ NIS サーバーのマシン名を返す

説明

string yp_master ( string $domain , string $map )

yp_master() は、指定したマップ map のマスタ NIS サーバーのマシン名を返します。

パラメータ

domain

NIS ドメイン名。

map

NIS マップ。

返り値

例

例1 NIS マスタの例

<?php
$number = yp_master($domain, $mapname);
echo "このマップのマスタは、" . $master;
?>
