Imagick::getImageHeight

(PECL imagick 2.0.0)

Imagick::getImageHeight — 画像の高さを返す

説明

int Imagick::getImageHeight ( void )

画像の高さを返します。

返り値

画像の高さをピクセル数で返します。

エラー / 例外

エラー時に ImagickException をスローします。
