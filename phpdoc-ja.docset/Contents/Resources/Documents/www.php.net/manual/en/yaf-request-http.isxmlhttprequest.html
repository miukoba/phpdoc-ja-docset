Yaf_Request_Http::isXmlHttpRequest

(Yaf >=1.0.0)

Yaf_Request_Http::isXmlHttpRequest — Ajax リクエストかどうかを調べる

説明

public bool Yaf_Request_Http::isXmlHttpRequest ( void )

そのリクエストが Ajax リクエストであるかどうかを調べます。

注意:

このメソッドはリクエストヘッダーの HTTP_X_REQUESTED_WITH を見て判断しますが、 Javascript ライブラリの中には、Ajax リクエストの際にこのヘッダーをセットしないものがあります。

パラメータ

この関数にはパラメータはありません。
