PDF_initgraphics

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_initgraphics — 描画状態をリセットする

説明

bool PDF_initgraphics ( resource $p )

色および描画状態パラメータを全てデフォルト値にリセットします。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
