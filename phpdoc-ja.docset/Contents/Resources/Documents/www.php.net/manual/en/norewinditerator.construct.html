NoRewindIterator::__construct

(PHP 5 >= 5.1.0)

NoRewindIterator::__construct — NoRewindIterator を作成する

説明

public NoRewindIterator::__construct ( Iterator $iterator )

NoRewindIterator を作成します。

パラメータ

iterator

使用するイテレータ。

返り値

渡されたイテレータにもとづく NoRewindIterator() を返します。

例

例1 NoRewindIterator::__construct() の例

二番目のループは何も出力しません。 このイテレータは一度しか使えず、巻き戻しされないからです。

<?php
$fruit = array('apple', 'banana', 'cranberry');

$arr = new ArrayObject($fruit);
$it  = new NoRewindIterator($arr->getIterator());

echo "Fruit A:\n";
foreach( $it as $item ) {
    echo $item . "\n";
}

echo "Fruit B:\n";
foreach( $it as $item ) {
    echo $item . "\n";
}
?>

上の例の出力は、 たとえば以下のようになります。

Fruit A:
apple
banana
cranberry
Fruit B:
