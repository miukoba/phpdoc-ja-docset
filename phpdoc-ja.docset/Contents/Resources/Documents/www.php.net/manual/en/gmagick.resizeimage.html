Gmagick::resizeimage

(PECL gmagick >= Unknown)

Gmagick::resizeimage — 画像のサイズを変更する

説明

public void Gmagick::resizeimage ( int $width , int $height , int $filter , float $blur [, bool $fit = false ] )

指定した大きさとフィルタで、画像のサイズを変更します。

パラメータ

width

画像の幅。

height

画像の高さ。

filter

利用するフィルタ。

blur

blur 要素。> 1 はぼやけた状態、< 1 はシャープな状態を表します。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
