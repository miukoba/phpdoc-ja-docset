Memcached::getVersion

(PECL memcached >= 0.1.5)

Memcached::getVersion — サーバープールのバージョン情報を取得する

説明

public array Memcached::getVersion ( void )

Memcached::getVersion() は、全ての利用可能な memcached サーバーのバージョン情報を含んだ配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

memcached サーバーのバージョン情報の配列。サーバーひとつにつき配列の要素一つが割り当てられます。

例

例1 Memcached::getVersion() の例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

print_r($m->getVersion());
?>

上の例の出力は、 たとえば以下のようになります。

Array
(
    [localhost:11211] => 1.2.6
)
