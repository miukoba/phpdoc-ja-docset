SolrInputDocument::__clone

(PECL solr >= 0.9.2)

SolrInputDocument::__clone — SolrDocument のコピーを作成

説明

public void SolrInputDocument::__clone ( void )

直接呼出しされてはいけません。SolrInputDocument のディープコピー作成に使用されます。

パラメータ

この関数にはパラメータはありません。

返り値

SolrInputDocument インスタンスを新規作成。
