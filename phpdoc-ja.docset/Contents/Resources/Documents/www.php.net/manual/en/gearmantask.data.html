GearmanTask::data

(PECL gearman >= 0.5.0)

GearmanTask::data — Get data returned for a task

説明

public string GearmanTask::data ( void )

Returns data being returned for a task by a worker.

パラメータ

この関数にはパラメータはありません。

返り値

The serialized data, or FALSE if no data is present.
