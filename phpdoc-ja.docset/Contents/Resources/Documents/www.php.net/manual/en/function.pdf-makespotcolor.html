PDF_makespotcolor

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_makespotcolor — スポット色を作成する

説明

int PDF_makespotcolor ( resource $p , string $spotname )

組み込みのスポット色名を検索するか、 現在の塗りつぶし色から名前付きのスポット色を作成します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
