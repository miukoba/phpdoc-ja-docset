Exception::getFile

(PHP 5 >= 5.1.0)

Exception::getFile — 例外が発生したファイルを取得する

説明

final public string Exception::getFile ( void )

例外が作られたファイルの名前を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

例外が作られたファイルの名前を返します。

例

例1 Exception::getFile() の例

<?php
try {
    throw new Exception;
} catch(Exception $e) {
    echo $e->getFile();
}
?>

上の例の出力は、 たとえば以下のようになります。

/home/bjori/tmp/ex.php
