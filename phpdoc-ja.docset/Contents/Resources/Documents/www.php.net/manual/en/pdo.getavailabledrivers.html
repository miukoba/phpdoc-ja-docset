PDO::getAvailableDrivers

(PHP 5 >= 5.1.3, PECL pdo >= 1.0.3)

PDO::getAvailableDrivers — 利用可能な PDO ドライバの配列を返す

説明

public static array PDO::getAvailableDrivers ( void )
array pdo_drivers ( void )

この関数は、 PDO::__construct() の DSN パラメータで利用可能な全ての有効な PDO ドライバを返します。

返り値

PDO::getAvailableDrivers() は PDO ドライバ名の配列を返します。 もしドライバが何も利用できない場合、空の配列を返します。

例

例1 PDO::getAvailableDrivers() の例

<?php
print_r(PDO::getAvailableDrivers());
?>

上の例の出力は、 たとえば以下のようになります。

Array
(
    [0] => mysql
    [1] => sqlite
)
