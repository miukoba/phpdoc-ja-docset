fann_set_cascade_num_candidate_groups

(PECL fann >= 1.0.0)

fann_set_cascade_num_candidate_groups — Sets the number of candidate groups

説明

bool fann_set_cascade_num_candidate_groups ( resource $ann , int $cascade_num_candidate_groups )

Sets the number of candidate groups.

パラメータ

ann

ニューラルネットワークリソース。

cascade_num_candidate_groups

The number of candidate groups.

返り値

成功した場合に TRUE、それ以外の場合に FALSE を返します。
