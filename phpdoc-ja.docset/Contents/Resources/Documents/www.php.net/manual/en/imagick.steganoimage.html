Imagick::steganoImage

(PECL imagick 2.0.0)

Imagick::steganoImage — デジタル透かしを画像に埋め込む

説明

Imagick Imagick::steganoImage ( Imagick $watermark_wand , int $offset )

デジタル透かしを画像に埋め込みます。 あとでこの透かしが見つかれば、その画像の出所を証明することができます。 offset は、透かしの埋め込みを開始する位置を指定します。

パラメータ

watermark_wand

offset

返り値

成功した場合に TRUE を返します。
