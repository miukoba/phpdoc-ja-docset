AMQPQueue::unbind

(PECL amqp >= Unknown)

AMQPQueue::unbind — キューをルーティングキーから削除する

説明

public bool AMQPQueue::unbind ( string $exchange_name , string $routing_key )

指定したキューから、exchange 上のルーティングキーのバインドを削除します。

パラメータ

exchange_name

キューがバインドされている exchange の名前。

routing_key

キューが使っているルーティングキー。

エラー / 例外

チャネルが開いていない場合に AMQPChannelException をスローします。

ブローカへの接続が途切れた場合に AMQPConnectionException をスローします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
