SplQueue::__construct

(PHP 5 >= 5.3.0)

SplQueue::__construct — 双方向リンクリストを使用して実装した新しい空のキューを作成する

説明

SplQueue::__construct() ( void )

これは、新しい空のキューを作成します。

注意:

このメソッドは、イテレータのモードを自動的に SplDoublyLinkedList::IT_MODE_FIFO とします。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

例

例1 SplQueue::__construct() の例

<?php
$q = new SplQueue();

$q[] = 1;
$q[] = 2;
$q[] = 3;

foreach ($q as $elem)  {
 echo $elem."\n";
}
?>

上の例の出力は以下となります。

1
2
3

例2 SplQueue による効率的なタスクの処理

<?php
$q = new SplQueue();
$q->setIteratorMode(SplQueue::IT_MODE_DELETE);

// ... タスクをキューに放り込みます ...

// それを処理します
foreach ($q as $task) {
    // ... $task の処理 ...

    // 新たなタスクをキューに追加します
    $q[] = $newTask;
    // ...
}
?>
