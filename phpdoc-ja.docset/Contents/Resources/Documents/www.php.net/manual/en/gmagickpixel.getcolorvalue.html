GmagickPixel::getcolorvalue

(PECL gmagick >= Unknown)

GmagickPixel::getcolorvalue — 指定した色チャネルの正規化した値を取得する

説明

public float GmagickPixel::getcolorvalue ( int $color )

指定した色チャネルの値を取得します。0 から 1 までの間の浮動小数点数値になります。

パラメータ

color

調べたいチャネル。Gmagick チャネル定数で指定します。

返り値

チャネルの値を、正規化した浮動小数点数値で返します。エラー時には GmagickPixelException をスローします。
