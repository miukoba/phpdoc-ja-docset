AMQPQueue::setArguments

(PECL amqp >= Unknown)

AMQPQueue::setArguments — キュー上のすべての引数を設定する

説明

public void AMQPQueue::setArguments ( array $arguments )

指定したキュー上のすべての引数を設定します。その他すべての引数の設定は一掃されます。

パラメータ

arguments

引数のキー/バリューペアの配列。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
