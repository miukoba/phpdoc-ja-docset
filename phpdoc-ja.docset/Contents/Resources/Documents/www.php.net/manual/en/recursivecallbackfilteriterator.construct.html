RecursiveCallbackFilterIterator::__construct

(PHP 5 >= 5.4.0)

RecursiveCallbackFilterIterator::__construct — RecursiveIterator から RecursiveCallbackFilterIterator を作成する

説明

public RecursiveCallbackFilterIterator::__construct() ( RecursiveIterator $iterator , string $callback )

callback でフィルタリングした RecursiveIterator を作って、 どのアイテムを受け付けてどれを拒否するかを決めます。

パラメータ

iterator

フィルタリングする再帰イテレータ。

callback

コールバック。現在のアイテムを受け付ける場合に TRUE、 拒否する場合に FALSE を返します。 例 を参照ください。

任意の callable 値を指定できます。

返り値

値を返しません。
