GmagickDraw::getfontweight

(PECL gmagick >= Unknown)

GmagickDraw::getfontweight — フォントのウェイトを返す

説明

public int GmagickDraw::getfontweight ( void )

テキストのアノテーションで使うフォントのウェイトを返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に整数値を返します。ウェイトが設定されていない場合は 0 を返します。
