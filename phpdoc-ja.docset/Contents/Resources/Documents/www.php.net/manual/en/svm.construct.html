SVM::__construct

(PECL svm >= 0.1.0)

SVM::__construct — SVM オブジェクトを新規構築

説明

public SVM::__construct ( void )

訓練データを受け取る準備ができている SVM オブジェクトを新規構成します。

パラメータ

この関数にはパラメータはありません。

返り値

libsvm ライブラリが読み込まれなかった場合、SVMException をスローします。
