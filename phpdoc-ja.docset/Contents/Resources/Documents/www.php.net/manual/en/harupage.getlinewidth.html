HaruPage::getLineWidth

(PECL haru >= 0.0.1)

HaruPage::getLineWidth — 現在の線幅を取得する

説明

float HaruPage::getLineWidth ( void )

現在の線幅を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の線幅を返します。

エラー / 例外

エラー時に HaruException をスローします。
