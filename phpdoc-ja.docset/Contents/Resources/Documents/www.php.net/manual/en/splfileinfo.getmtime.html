SplFileInfo::getMTime

(PHP 5 >= 5.1.2)

SplFileInfo::getMTime — 最終変更時刻を取得する

説明

public int SplFileInfo::getMTime ( void )

ファイルの中身が変更された時刻を返します。返される時刻は Unix タイムスタンプ形式となります。

パラメータ

この関数にはパラメータはありません。

返り値

ファイルの最終変更時刻を Unix タイムスタンプで返します。
