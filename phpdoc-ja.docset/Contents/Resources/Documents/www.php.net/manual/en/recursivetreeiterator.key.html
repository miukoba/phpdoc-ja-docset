RecursiveTreeIterator::key

(PHP 5 >= 5.3.0)

RecursiveTreeIterator::key — 現在の要素のキーを取得する

説明

public string RecursiveTreeIterator::key ( void )

接頭辞と接尾辞をつけた現在のキーを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

接頭辞と接尾辞をつけた現在のキーを返します。
