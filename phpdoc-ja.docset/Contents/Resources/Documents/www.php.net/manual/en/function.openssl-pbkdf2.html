openssl_pbkdf2

(PHP 5 >= 5.5.0)

openssl_pbkdf2 — PKCS5 v2 の PBKDF2 文字列を生成する。デフォルトは SHA-1

説明

string openssl_pbkdf2 ( string $password , string $salt , int $key_length , int $iterations [, string $digest_algorithm ] )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

password

salt

key_length

iterations

digest_algorithm

返り値

文字列を返します。失敗した場合に FALSE を返します。
