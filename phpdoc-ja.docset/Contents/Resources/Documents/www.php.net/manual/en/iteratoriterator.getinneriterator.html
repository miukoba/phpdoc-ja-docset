IteratorIterator::getInnerIterator

(PHP 5 >= 5.1.0)

IteratorIterator::getInnerIterator — 内部イテレータを取得する

説明

public Traversable IteratorIterator::getInnerIterator ( void )

内部イテレータを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

IteratorIterator::__construct() に渡した内部イテレータを返します。
