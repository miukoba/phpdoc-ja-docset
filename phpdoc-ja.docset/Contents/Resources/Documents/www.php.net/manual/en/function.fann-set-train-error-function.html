fann_set_train_error_function

(PECL fann >= 1.0.0)

fann_set_train_error_function — Sets the error function used during training

説明

bool fann_set_train_error_function ( resource $ann , int $error_function )

Sets the error function used during training.

The error functions are described further in error functions constants.

パラメータ

ann

ニューラルネットワークリソース。

error_function

The error function constant

返り値

成功した場合に TRUE、それ以外の場合に FALSE を返します。
