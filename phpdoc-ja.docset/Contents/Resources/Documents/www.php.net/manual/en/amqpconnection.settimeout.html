AMQPConnection::setTimeout

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

AMQPConnection::setTimeout — タイムアウトを設定する

説明

public bool AMQPConnection::setTimeout ( float $timeout )

タイムアウトの値を秒単位で指定します。この時間が過ぎると、 この AMQPConnection オブジェクトのインスタンスはブローカへのリクエストをタイムアウトとみなします。 この値を設定すると、INI 設定 amqp.timeout で指定したデフォルト値を上書きします。 たとえば、タイムアウトを 500 ミリ秒にするには 0.5 を指定します。

パラメータ

timeout

AMQP ブローカへの接続に使うタイムアウト。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

エラー / 例外

timeout に負の数を指定した場合に AMQPConnectionException をスローします。

例

例1 AMQPConnection::setTimeout() の例

<?php

/* 新しい接続を作成します */
$cnn = new AMQPConnection();

// タイムアウトを設定します
$cnn->setTimeout(5672);

?>
