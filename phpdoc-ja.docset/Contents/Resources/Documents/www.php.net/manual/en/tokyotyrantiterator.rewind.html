TokyoTyrantIterator::rewind

(PECL tokyo_tyrant >= 0.2.0)

TokyoTyrantIterator::rewind — イテレータを巻き戻す

説明

public void TokyoTyrantIterator::rewind ( void )

イテレータを巻き戻して新たに反復処理を始められるようにします。 foreach の最初に自動的にコールされます。

パラメータ

この関数にはパラメータはありません。

返り値

イテレータの初期化に失敗した場合に TokyoTyrantException をスローします。

  • TokyoTyrantIterator::next() - 次のキーに移動する
