DateTimeImmutable::__construct

date_create_immutable

(PHP 5 >= 5.5.0)

DateTimeImmutable::__construct -- date_create_immutable — 新しい DateTimeImmutable オブジェクトを返す

説明

オブジェクト指向型

public DateTimeImmutable::__construct() ([ string $time = "now" [, DateTimeZone $timezone = NULL ]] )

手続き型

DateTimeImmutable date_create_immutable ([ string $time = "now" [, DateTimeZone $timezone = NULL ]] )

DateTime::__construct() と似ていますが、 DateTimeImmutable で動作します。
