QuickHashIntSet::saveToFile

(PECL quickhash >= Unknown)

QuickHashIntSet::saveToFile — メモリ上のセットをディスクに保存する

説明

public void QuickHashIntSet::saveToFile ( string $filename )

このメソッドは、既存のセットをディスクに保存します。 loadFromFile() で読めるフォーマットになります。

パラメータ

filename

セットの保存先のファイル名。

返り値

値を返しません。

例

例1 QuickHashIntSet::saveToFile() の例

<?php
$set = new QuickHashIntSet( 1024 );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );
var_dump( $set->exists( 4 ) );
var_dump( $set->add( 4 ) );

$set->saveToFile( '/tmp/test.set' );
?>
