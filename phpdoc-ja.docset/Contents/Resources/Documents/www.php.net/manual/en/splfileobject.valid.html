SplFileObject::valid

(PHP 5 >= 5.1.0)

SplFileObject::valid — ファイル終端でないかチェックする

説明

public bool SplFileObject::valid ( void )

ファイル終端に到達したかチェックします。

パラメータ

この関数にはパラメータはありません。

返り値

ファイル終端に到達していなければ TRUE、そうでなければ FALSE を返します。

例

例1 SplFileObject::valid() の例

<?php
// Loop over a file, line by line
$file = new SplFileObject("file.txt");
while ($file->valid()) {
    echo $file->fgets();
}
?>
