boolval

(PHP 5 >= 5.5.0)

boolval — 変数の boolean としての値を取得する

説明

boolean boolval ( mixed $var )

var の boolean としての値を返します。

パラメータ

var

boolean に変換したいスカラー値。

返り値

var の boolean としての値を返します。

例

例1 boolval() の例

<?php
echo '0:        '.(boolval(0) ? 'true' : 'false')."\n";
echo '42:       '.(boolval(42) ? 'true' : 'false')."\n";
echo '0.0:      '.(boolval(0.0) ? 'true' : 'false')."\n";
echo '4.2:      '.(boolval(4.2) ? 'true' : 'false')."\n";
echo '"":       '.(boolval("") ? 'true' : 'false')."\n";
echo '"string": '.(boolval("string") ? 'true' : 'false')."\n";
echo '"0":      '.(boolval("0") ? 'true' : 'false')."\n";
echo '"1":      '.(boolval("1") ? 'true' : 'false')."\n";
echo '[1, 2]:   '.(boolval([1, 2]) ? 'true' : 'false')."\n";
echo '[]:       '.(boolval([]) ? 'true' : 'false')."\n";
echo 'stdClass: '.(boolval(new stdClass) ? 'true' : 'false')."\n";
?>

上の例の出力は以下となります。

0:        false
42:       true
0.0:      false
4.2:      true
"":       false
"string": true
"0":      false
"1":      true
[1, 2]:   true
[]:       false
stdClass: true

When someone is working on large projects, this feature can be useful boolval. In simple scripts, it is best to cast to a boolean.

<?php
echo boolval(2); #1

Will see the same thing:

<?php
echo (bool)2; #1
?>
