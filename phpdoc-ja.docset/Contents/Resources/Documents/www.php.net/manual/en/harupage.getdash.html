HaruPage::getDash

(PECL haru >= 0.0.1)

HaruPage::getDash — 現在の破線のパターンを取得する

説明

array HaruPage::getDash ( void )

現在の破線のパターンを取得します。破線のパターンについての詳細な情報は HaruPage::setDash() を参照ください。

パラメータ

この関数にはパラメータはありません。

返り値

現在の破線のパターンを、二つの要素 "pattern" と "phase" からなる配列で返します。破線のパターンが設定されていない場合は FALSE を返します。

エラー / 例外

エラー時に HaruException をスローします。
