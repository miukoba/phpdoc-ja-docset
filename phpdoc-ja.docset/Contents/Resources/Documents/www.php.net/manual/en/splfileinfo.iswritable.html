SplFileInfo::isWritable

(PHP 5 >= 5.1.2)

SplFileInfo::isWritable — エントリが書き込み可能であるかどうかを調べる

説明

public bool SplFileInfo::isWritable ( void )

現在のエントリが書き込み可能かどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

書き込み可能な場合に TRUE、そうでない場合に FALSE を返します。
