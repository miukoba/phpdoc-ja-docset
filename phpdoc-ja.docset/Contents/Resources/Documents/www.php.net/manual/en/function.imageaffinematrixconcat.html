imageaffinematrixconcat

(PHP 5 >= 5.5.0)

imageaffinematrixconcat — 二つの行列を連結する (複数の操作を一度に行う)

説明

array imageaffinematrixconcat ( array $m1 , array $m2 )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

m1

0 から 5 までのキーを持つ配列。

m2

0 から 5 までのキーを持つ配列。

返り値

0 から 5 までのキーに float 値を持つ配列を返します。失敗した場合に FALSE を返します。
