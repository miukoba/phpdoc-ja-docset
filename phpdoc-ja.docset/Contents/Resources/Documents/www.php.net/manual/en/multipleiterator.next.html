MultipleIterator::next

(PHP 5 >= 5.3.0)

MultipleIterator::next — アタッチしているすべてのイテレータインスタンスを次に移動させる

説明

public void MultipleIterator::next ( void )

アタッチしているすべてのイテレータインスタンスを次に移動させます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
