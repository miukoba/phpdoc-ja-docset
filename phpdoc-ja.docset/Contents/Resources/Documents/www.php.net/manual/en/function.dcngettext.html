dcngettext

(PHP 4 >= 4.2.0, PHP 5)

dcngettext — dcgettext の複数形版

説明

string dcngettext ( string $domain , string $msgid1 , string $msgid2 , int $n , int $category )

この関数により単一の複数形メッセージの参照について カレントのドメインを上書きすることができます。

パラメータ

domain

ドメイン。

msgid1

msgid2

n

category

返り値

成功した場合に文字列を返します。
