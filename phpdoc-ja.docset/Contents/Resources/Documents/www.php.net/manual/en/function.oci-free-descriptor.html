oci_free_descriptor

(PHP 5, PECL OCI8 >= 1.1.0)

oci_free_descriptor — ディスクリプタを解放する

説明

bool oci_free_descriptor ( resource $descriptor )

oci_new_descriptor() で確保したディスクリプタを解放します。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

注意

注意:

この関数は、一般に OCI-LOB::free メソッドとして使います。
