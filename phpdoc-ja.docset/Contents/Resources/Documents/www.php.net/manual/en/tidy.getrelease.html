tidy::getRelease

tidy_get_release

(PHP 5, PECL tidy >= 0.5.2)

tidy::getRelease -- tidy_get_release — Tidy ライブラリのリリース日 (バージョン) を取得する

説明

オブジェクト指向型

string tidy::getRelease ( void )

手続き型

string tidy_get_release ( void )

Tidy ライブラリのリリース日を取得します。

返り値

Tidy ライブラリのリリース日を文字列で返します。
