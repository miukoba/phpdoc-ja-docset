counter_create

counter_create — 単一の数値を保持するカウンタを作成する

説明

resource counter_create ( string $name [, integer $initial_value [, integer $flags ]] )

単一の数値を保持するカウンタを作成します。

パラメータ

name
カウンタの名前。
initial_value
カウンタの初期値。デフォルトはゼロ (0)。
flags
カウンタのフラグ。 COUNTER_FLAG_* 定数群から選択します。

返り値

カウンタリソースを返します。
