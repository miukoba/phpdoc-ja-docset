MongoLog::getModule

(PECL mongo >= 1.2.3)

MongoLog::getModule — 現在ログを記録しているモジュールを取得する

説明

public static int MongoLog::getModule ( void )

この関数を使って、ドライバのどの機能をログに記録しているのかを調べます。 MongoLog のセクションで説明した定数をビット演算子でつないで、 特定のモジュールのログが記録されているかどうかを調べます。

<?php

if (MongoLog::getModule() & MongoLog::RS) {
    echo "レプリカセットを記録しています\n";
}

if (MongoLog::getModule() ^ MongoLog::NONE) {
    echo "何かを記録しています\n";
}

if ((MongoLog::getModule() & MongoLog::IO) == 0) {
    echo "入出力は記録していません\n";
}

?>

パラメータ

この関数にはパラメータはありません。

返り値

現在ログを記録しているモジュールを返します。
