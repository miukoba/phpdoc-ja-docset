SplPriorityQueue::key

(PHP 5 >= 5.3.0)

SplPriorityQueue::key — 現在のノードのインデックスを返す

説明

public mixed SplPriorityQueue::key ( void )

この関数は、現在のノードのインデックスを返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のノードのインデックスを返します。
