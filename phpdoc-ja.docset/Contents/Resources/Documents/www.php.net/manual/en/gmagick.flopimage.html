Gmagick::flopimage

(PECL gmagick >= Unknown)

Gmagick::flopimage — 水平方向に反転した画像を作成する

説明

public Gmagick Gmagick::flopimage ( void )

中心の y 軸を基準してピクセルを反転させ、水平方向のミラー画像を作成します。

返り値

水平反転した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
