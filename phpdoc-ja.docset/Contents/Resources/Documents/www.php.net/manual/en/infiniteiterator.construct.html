InfiniteIterator::__construct

(PHP 5 >= 5.1.0)

InfiniteIterator::__construct — InfiniteIterator を作成する

説明

public InfiniteIterator::__construct ( Iterator $iterator )

イテレータから InfiniteIterator を作成します。

パラメータ

iterator

無限に処理し続けるイテレータ。

返り値

値を返しません。

エラー / 例外

iterator にイテレータ以外を渡したときに E_RECOVERABLE_ERROR をスローします。

例

例1 InfiniteIterator::__construct() の例

<?php
$arrayit  = new ArrayIterator(array('cat','dog'));
$infinite = new InfiniteIterator($arrayit);
$limit    = new LimitIterator($infinite, 0, 7);
foreach($limit as $value)
{
    echo "$value\n";
}
?>

上の例の出力は以下となります。

cat
dog
cat
dog
cat
dog
cat
