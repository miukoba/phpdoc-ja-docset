Worker::run

(PECL pthreads >= 0.36)

Worker::run — 実行する

説明

abstract public void Worker::run ( void )

run メソッドはワーカーのメンバー (およびリソース) を準備しなければいけません。 Stackables がワーカーとそのメソッド/メンバー/リソースにアクセスできるようになります。

パラメータ

この関数にはパラメータはありません。

返り値

このメソッドの返り値は無視されます。
