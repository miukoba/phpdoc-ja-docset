Ev::feedSignalEvent

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Ev::feedSignalEvent — シグナルイベントの情報をデフォルトループに送る

説明

final public static void Ev::feedSignalEvent ( int $signum )

シグナルイベントをデフォルトループに送ります。 Ev は、signal で指定したシグナルを受け取ったかのように反応します。

パラメータ

signum

シグナル番号。詳細は signal(7) の man ページを参照ください。 pcntl 拡張モジュールが公開する定数も使えます。

返り値

値を返しません。
