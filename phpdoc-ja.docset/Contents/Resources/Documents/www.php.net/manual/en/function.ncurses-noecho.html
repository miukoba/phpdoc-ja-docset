ncurses_noecho

(PHP 4 >= 4.0.7, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0)

ncurses_noecho — キーボード入力エコーを無効にする

説明

bool ncurses_noecho ( void )
警告

この関数は、 実験的 なものです。この関数の動作・ 名前・その他ドキュメントに書かれている事項は、予告なく、将来的な PHP のリリースにおいて変更される可能性があります。 この関数は自己責任で使用してください。

ユーザーが入力した文字のエコーを無効にします。

返り値

何らかのエラーが発生した場合に TRUE、 それ以外の場合に FALSE を返します。

The ncurses_noecho() function basically disables user input from being echoed to your present cursor location.

Re-enable with the ncurses_echo() function. Echo is enabled by default in a new ncurses session.
