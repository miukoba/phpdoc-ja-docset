MongoClient::getConnections

(PECL mongo >=1.3.0)

MongoClient::getConnections — 開いているすべての接続の情報を返す

説明

public static array MongoClient::getConnections ( void )

開いているすべての接続と、それぞれのサーバーの情報を配列で返します。

パラメータ

この関数にはパラメータはありません。

返り値

開いている接続に関する配列を返します。

例

例1 MongoClient::getConnections() の例

<?php
$m = new MongoClient;
var_dump($m->getConnections());
?>
