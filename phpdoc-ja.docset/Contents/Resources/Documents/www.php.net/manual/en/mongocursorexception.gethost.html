MongoCursorException::getHost

(PECL mongo >= 1.0.0)

MongoCursorException::getHost — エラーが発生したサーバーのホスト名を取得する

説明

public string MongoCursorException::getHost ( void )

クエリの送信先サーバーのホスト名を返します。

パラメータ

この関数にはパラメータはありません。

返り値

ホスト名を返します。ホスト名が不明な場合は NULL を返します。
