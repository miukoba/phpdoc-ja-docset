swf_nextid

(PHP 4)

swf_nextid — 次の未使用のオブジェクト ID を返す

説明

int swf_nextid ( void )

関数 swf_nextid() は、次に利用可能なオブジェクト ID を返します。

返り値

ID を表す整数値を返します。
