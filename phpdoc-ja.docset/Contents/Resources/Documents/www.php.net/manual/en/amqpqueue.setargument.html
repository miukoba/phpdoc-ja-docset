AMQPQueue::setArgument

(PECL amqp >= Unknown)

AMQPQueue::setArgument — 指定したキーの値を設定する

説明

public void AMQPQueue::setArgument ( string $key , mixed $value )

キーに値を設定します。

パラメータ

key

設定するキー。

value

設定する値。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
