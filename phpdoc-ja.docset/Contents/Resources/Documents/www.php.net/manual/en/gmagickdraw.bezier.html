GmagickDraw::bezier

(PECL gmagick >= Unknown)

GmagickDraw::bezier — ベジエ曲線を描く

説明

public GmagickDraw GmagickDraw::bezier ( array $coordinate_array )

指定した点を通るベジエ曲線を画像上に描きます。

パラメータ

coordinate_array

座標の配列。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
