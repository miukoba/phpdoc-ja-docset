msql_num_fields

(PHP 4, PHP 5)

msql_num_fields — 結果におけるフィールドの数を得る

説明

int msql_num_fields ( resource $result )

msql_num_fields()は、結果セットにおける フィールドの数を返します。

パラメータ

result

評価された結果 リソース。 この結果は、 msql_query() のコールにより得られたものです。

返り値

結果セットにおけるフィールドの数を返します。

参考

  • msql_query() - mSQL クエリを送信する
  • msql_fetch_field() - フィールド情報を得る
  • msql_num_rows() - 結果における行の数を得る
