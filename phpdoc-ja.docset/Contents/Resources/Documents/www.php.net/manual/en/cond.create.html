Cond::create

(PECL pthreads >= 0.34)

Cond::create — 条件を作る

説明

final public static long Cond::create ( void )

新しい条件変数を作ります。

パラメータ

この関数にはパラメータはありません。

返り値

条件変数へのハンドルを返します。

例

例1 条件の作成と破棄

<?php
/** "new" は使えません。Cond は PHP のオブジェクトではありません **/
$cond = Cond::create();
/** Cond を任意のコンテキストで使えます **/
var_dump($cond);
/** 自分が作った Cond は常に破棄します **/
Cond::destroy($cond);
?>

上の例の出力は以下となります。

int(4540682)
