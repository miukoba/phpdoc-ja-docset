GearmanWorker::returnCode

(PECL gearman >= 0.5.0)

GearmanWorker::returnCode — Get last Gearman return code

説明

public int GearmanWorker::returnCode ( void )

Returns the last Gearman return code.

パラメータ

この関数にはパラメータはありません。

返り値
