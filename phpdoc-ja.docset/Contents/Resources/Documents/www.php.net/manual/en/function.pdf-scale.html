PDF_scale

(PHP 4, PECL pdflib >= 1.0.0)

PDF_scale — スケールを設定する

説明

bool PDF_scale ( resource $p , float $sx , float $sy )

座表系のスケールを設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
