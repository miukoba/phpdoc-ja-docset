filepro_fieldname

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldname — フィールド名を取得する

説明

string filepro_fieldname ( int $field_number )

field_number に対応するフィールド名を返します。

パラメータ

field_number

フィールド番号。

返り値

フィールド名を表す文字列、あるいはエラーの場合に FALSE を返します。
