定義済み定数

以下の定数が定義されています。 この関数の拡張モジュールが PHP 組み込みでコンパイルされているか、 実行時に動的にロードされている場合のみ使用可能です。

MySQL Memcache Plugin 関連

MYSQLND_MEMCACHE_DEFAULT_REGEXP (string)
デフォルトの正規表現 (PCRE スタイル) で、これを使って SELECT 文のマッチングをします。そして、可能なら MySQL Memcache Plugin のアクセスポイントにマップします。 mysqlnd_memcache_set() も使えますが、 デフォルトではこの正規表現でパターンマッチングを行います。

その他

このプラグインのバージョン番号を取得するには MYSQLND_MEMCACHE_VERSION あるいは MYSQLND_MEMCACHE_VERSION_ID を使います。 MYSQLND_MEMCACHE_VERSION は文字列形式で表したバージョンで、 MYSQLND_MEMCACHE_VERSION_ID が数値のバージョン番号です。この数値は 10000 のような整数になります。 この数値から、次のようにバージョンを計算します。

バージョン (の一部) 例
Major*10000 1*10000 = 10000
Minor*100 0*100 = 0
Patch 0 = 0
MYSQLND_MEMCACHE_VERSION_ID 10000

MYSQLND_MEMCACHE_VERSION (string)
プラグインのバージョンを表す文字列。"1.0.0-alpha" など。
MYSQLND_MEMCACHE_VERSION_ID (integer)
プラグインのバージョン番号。10000 など。
