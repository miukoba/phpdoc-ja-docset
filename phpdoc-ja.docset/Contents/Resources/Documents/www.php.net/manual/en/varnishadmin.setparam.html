VarnishAdmin::setParam

(PECL varnish >= 0.4)

VarnishAdmin::setParam — 現在の varnish インスタンスの構成パラメータを設定する

説明

public int VarnishAdmin::setParam ( string $name , string|integer $value )

パラメータ

name

Varnish のパラメータ名。

value

Varnish のパラメータの値。

返り値

varnish のコマンドステータスを返します。
