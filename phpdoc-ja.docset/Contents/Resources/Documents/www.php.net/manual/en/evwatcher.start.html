EvWatcher::start

(PECL ev >= 0.2.0)

EvWatcher::start — Starts the watcher

説明

public void EvWatcher::start ( void )

Marks the watcher as active. Note that only active watchers will receive events.

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
