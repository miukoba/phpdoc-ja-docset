HttpRequest::getHeaders

(PECL pecl_http >= 0.10.0)

HttpRequest::getHeaders — ヘッダを取得する

説明

public array HttpRequest::getHeaders ( void )

事前に設定されたリクエストヘッダを取得します。

返り値

現在設定されているすべてのヘッダを含む連想配列を返します。
